4.1.1 Object Assignment

对象可以为其他对象提供服务，其中“分配自”对象作用于或观察“分配至”对象的 要求。存在一个通用的分配周期，其中 参与者（人员）发布 控制项（如工单或计划），这可能导致由 产品（如建筑构件）组成的 组（如建筑系统），这些产品由 过程（如施工任务）作用于，而这些过程由 资源（如劳动力）执行，而这些资源又可能由参与者（人员）来满足。要求在“至”端建立，并在“自”端满足，其中成本、时间、范围或其他指标可以在“自到”方向上计算。
