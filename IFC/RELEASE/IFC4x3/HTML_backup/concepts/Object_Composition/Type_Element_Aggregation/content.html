4.1.4.6 Type Element Aggregation

元素类型可以定义聚合组件。

此类组件的放置可以显式使用相对于封闭的_IfcElementType_的_IfcLocalPlacement_来定义，或者如果_IfcElement.ObjectPlacement_为空，则隐式定义。

如果给定的_IfcElementType_被实例化为一个实例，那么该实例必须包含等效的聚合元素，其名称与元素类型内的名称相对应。
