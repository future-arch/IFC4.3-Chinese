6.1.2.12 IfcMemberTypeEnum（杆件类型枚举）

6.1.2.12.1 语义定义（Semantic definition）

此枚举定义了 IfcMember 或 IfcMemberType 对象可以实现的各类线性构件类型。

沿革 x2 中新增的枚举类型。
IFC2x2 变更 新增了 CHORD、PLATE、STUD 标识符。
IFC2x3 变更 新增了 MULLION 标识符。

6.1.2.12.2 类型值

6.1.2.12.3 形式化表示（Formal representation） 

TYPE IfcMemberTypeEnum = ENUMERATION OF
	(ARCH_SEGMENT
	,BRACE
	,CHORD
	,COLLAR
	,MEMBER
	,MULLION
	,PLATE
	,POST
	,PURLIN
	,RAFTER
	,STAY_CABLE
	,STIFFENING_RIB
	,STRINGER
	,STRUCTURALCABLE
	,STRUT
	,STUD
	,SUSPENDER
	,SUSPENSION_CABLE
	,TIEBAR
	,USERDEFINED
	,NOTDEFINED);
END_TYPE;
