8.15.3.9 IfcCompositeProfileDef

RV ⓘ
AbV ⓘ

8.15.3.9.1 语义定义（Semantic definition）

IfcCompositeProfileDef 通过组合其他轮廓来定义轮廓。组合由一组至少两个其他轮廓定义组成。任何轮廓定义（除了另一个复合轮廓）都可以用来构建复合轮廓。

沿革 New entity in IFC2x.

图 8.15.3.9.A 说明了复合轮廓定义。 IfcCompositeProfileDef 不定义自己的位置坐标系，它直接定义在底层坐标系中。底层坐标系由使用该轮廓定义的扫描曲面或扫描区域实体定义。它是以下任一项的位置的 xy 平面：

  • IfcSweptSurface.Position
  • IfcSweptAreaSolid.Position

或者在分段脊线的情况下，它是 IfcSectionedSpine.CrossSectionPositions 的每个列表成员的 xy 平面。 IfcCompositeProfileDef 是使用其他轮廓定义来定义的。这些其他轮廓定义直接插入到底层坐标系中。

  • 对于参数化轮廓定义，这些标准轮廓的 Position 属性用于将轮廓相互相对放置。
  • 对于任意轮廓定义，每个笛卡尔坐标直接在底层坐标系中给出。
注释 The black coordinate axes show the underlying coordinate system of the swept surface or swept area solid.

双轮廓特殊情况

如果双轮廓由轮廓组合建模，则基准轮廓应只指定一次。然后它直接包含在复合轮廓中，并通过 IfcMirroredProfileDef 间接包含。例如，一个由两个 L100x10 组成的双角钢，它们之间有 10mm 的空气间隙，即一个 | | 形状，可以建模为

```

single_L : IfcLShapeProfileDef := IfcLShapeProfileDef(AREA, 'L100X100X10', IfcAxis2Placement2D(IfcCartesianPoint(((.100+.010)/2., .0)), ?), .100, .100, .010, .012, ?, 0., ?, ?);

double_L : IfcCompositeProfileDef := IfcCompositeProfileDef(AREA, 'double angle', (single_L, IfcMirroredProfileDef(AREA, ?, single_L, ?)), 'twin profile');

```

8.15.3.9.7 形式化表示（Formal representation） 

ENTITY IfcCompositeProfileDef
 SUBTYPE OF (IfcProfileDef);
	Profiles : SET [2:?] OF IfcProfileDef;
	Label : OPTIONAL IfcLabel;
 WHERE
	InvariantProfileType : SIZEOF(QUERY(temp <* Profiles | temp.ProfileType <> Profiles[1].ProfileType)) = 0;
	NoRecursion : SIZEOF(QUERY(temp <* Profiles | 'IFC4X3_DEV_62887e3f.IFCCOMPOSITEPROFILEDEF' IN TYPEOF(temp))) = 0;
END_ENTITY;
