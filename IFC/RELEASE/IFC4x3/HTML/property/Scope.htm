语义定义（Semantic definition）

Industry Foundation Classes (IFC) 是一个开放的国际标准，用于建筑信息模型（BIM）数据，这些数据在建筑或设施管理行业部门的各种参与者使用的软件应用程序之间进行交换和共享。该标准包含涵盖建筑物和桥梁在其生命周期所需数据的定义。本版本及未来版本将把范围扩展到包括基础设施资产在其生命周期内的数据定义。

Industry Foundation Classes 规定了数据模式和交换文件格式结构。数据模式定义在：

  • EXPRESS 数据规范语言，定义于 ISO 10303-11
  • XML Schema 定义语言 (XSD)，定义于 XML Schema W3C Recommendation

其中，EXPRESS 模式定义是源，XML 模式定义根据 ISO 10303-28 中定义的映射规则从 EXPRESS 模式生成。根据概念模式进行数据交换和共享的交换文件格式为：

  • 交换结构纯文本编码，定义于 ISO 10303-21
  • 可扩展标记语言 (XML)，定义于 XML W3C Recommendation

如果替代交换文件格式符合数据模式，则可以使用它们。

本版本的 IFC 由数据模式（表示为 EXPRESS 模式和 XML 模式）以及参考数据（表示为属性和数量名称的定义，以及形式和信息性描述）组成。

数据模式和参考数据的子集称为模型视图定义（MVD）。特定的 MVD 定义用于支持建筑和设施管理行业部门的一个或多个公认的工作流程。每个工作流程识别软件应用程序的数据交换需求。符合要求的软件应用程序需要识别它们所符合的模型视图定义。
