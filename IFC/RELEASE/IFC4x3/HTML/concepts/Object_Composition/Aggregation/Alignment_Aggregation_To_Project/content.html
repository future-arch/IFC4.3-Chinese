4.1.4.1.1 Alignment Aggregation To Project

================================

每个 IfcAlignment 必须使用 IfcRelAggregates 关系与 IfcProject 相关联，无论是直接还是间接。间接情况是指子对齐线聚合到父对齐线。在这种情况下，只有父对齐线必须与 IfcProject 相关联。
