4.1.4.4.1.1 Alignment Layout - Horizontal, Vertical and Cant

IfcAlignment 与可能定义它的三个布局（即 IfcAlignmentHorizontal、IfcAlignmentVerticalSegment、IfcAlignmentCant）及其各自的线段之间的嵌套关系。

在定义业务逻辑的线段列表时（即 IfcAlignmentHorizontalSegment、IfcAlignmentVerticalSegment、IfcAlignmentCantSegment）：

  1. 应在 IfcAlignmentSegment.DesignParameters 的线段列表末尾添加一个零长度线段。
  2. 如果还存在几何定义，则每个零长度线段都应有一个长度为零的 IfcCurveSegment 对应项。
