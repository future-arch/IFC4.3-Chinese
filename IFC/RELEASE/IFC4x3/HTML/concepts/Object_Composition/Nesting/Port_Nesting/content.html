4.1.4.4.4 Port Nesting

端口指示根据指定的系统类型、流向和连接属性与其他对象的可能连接。端口通常通过电缆、管道或管道在设备之间连接。

端口可以定义其放置，指示端口相对于产品或产品类型的相对于产品的位置和朝向。端口还可以定义材料剖面集，指示流区域和连接外壳。
