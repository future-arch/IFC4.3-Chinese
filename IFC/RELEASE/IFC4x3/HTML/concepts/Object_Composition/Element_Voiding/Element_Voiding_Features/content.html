4.1.4.3.3 Element Voiding Features

构件可以定义挖空特征，这可以通过切割、钻孔或铣削各种材料制成的构件来实现，或者通过嵌入混凝土等材料的模板来实现。

构件的“主体”表示不考虑空腔，需要通过 CSG 操作来产生最终形状。

构件的“网格”表示考虑了空腔，因此无需额外的操作。
