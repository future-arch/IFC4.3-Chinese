4.1.4.3.2 Element Openings

构件可以定义开口（几何空腔），这些开口可以是部分凹陷或贯穿整个深度。开口可选择性地由另一个构件填充，例如门或窗。

构件的“主体”表示在导出时不考虑空腔，需要通过 CSG 操作在导入时产生最终形状。

构件的“参考”表示在导出时考虑了空腔，因此在导入时无需额外的操作。
