4.1.7.1.3.1 Axis 2D Geometry

The 'Axis' representation using 'Curve2D' representation type is predominately used to align material layer sets to geometric representations of standard walls.

The following attribute values for the IfcShapeRepresentation holding this geometric representation shall be used:

  • IfcShapeRepresentation.RepresentationIdentifier = 'Axis'
  • IfcShapeRepresentation.RepresentationType : 'Curve2D'
  • IfcShapeRepresentation.Items = IfcBoundedCurve
