4.1.3.1.1.2 Mechanical Fastener Attributes

注释 此属性模板与其他类似模板一样，源自与 mvdXML 相关联的旧有要求，以及 MVD 定义交换信息要求的早期时代。此类模板不再为规范增加价值，也不会传达超出模式中已定义信息的内容。此外，一些模板引用了已弃用的实体，可能导致不必要的混淆。 作为清理文档的更广泛工作的一部分，此模板和其他非必需模板将在下一个版本中删除。
