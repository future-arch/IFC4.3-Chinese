4.1.1.2 Assignment to Group

_分配到组_建立对象到组的分配，组是对象的任意集合。它是_组分配_的补充概念模板，定义了对象如何被组分配。

分组关系除了将对象分组外，不具有任何其他含义。它是非层次化的，意味着对象可以被分配到多个组，并且不干扰其他关系概念，例如_元素分解_。分组关系允许子组被分组到组中。
