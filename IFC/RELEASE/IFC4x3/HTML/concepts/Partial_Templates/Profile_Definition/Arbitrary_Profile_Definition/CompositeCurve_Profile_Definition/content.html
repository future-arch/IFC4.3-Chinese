4.2.8.1.2 CompositeCurve Profile Definition

复合曲线剖面定义通过由多个复合曲线段组成的复合曲线来定义横截面使用的闭合二维曲线。每个复合曲线段都有一个父曲线，该父曲线是折线或修剪曲线。修剪曲线的基曲线类型为直线、圆或椭圆。
