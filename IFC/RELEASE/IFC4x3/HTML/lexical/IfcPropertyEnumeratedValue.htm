8.16.3.8 IfcPropertyEnumeratedValue

RV ⓘ
AbV ⓘ

8.16.3.8.1 语义定义（Semantic definition）

具有枚举值的属性 IfcPropertyEnumeratedValue 定义了一个属性对象，该对象具有从枚举中选择的值。它定义了一个属性-值组合，其中给出了属性 Name、可选的 Description、可选的 EnumerationValues（带度量类型）以及可选的 Unit。

注释 Multiple choices from the property enumeration are supported.

单位由 Unit 特性处理，表 8.16.3.8.A 显示了枚举属性的示例：

  • 如果未提供 Unit 特性，则单位已由 IfcMeasureValue 或 IfcDerivedMeasureValue 的类型隐含。关联单位可在项目级别全局定义的 IfcUnitAssignment (IfcProject.UnitsInContext) 中找到。
  • 如果提供了 Unit 特性，则 Unit 特性分配的单位将覆盖全局分配的单位。

更具体地说：IfcPropertyEnumeratedValue 定义了一个属性，其值是从定义的枚举器列表中选择的。枚举器存储在动态枚举值中，包括来自 IfcValue 的类型信息（请参阅 IfcPropertyEnumeration）。这使得应用程序可以使用枚举值作为属性集 (IfcPropertySet) 中的属性，包括允许的值列表。

注释 An IfcPropertyEnumeratedValue may be exchanged with no values assigned yet. In this case the EnumerationValues are set to NIL.

IfcPropertyEnumeratedValue 引用 IfcPropertyEnumeration，请参阅表 8.16.3.8.B 的示例：

不强制要求使用 IfcPropertyEnumeration 的实例来保存 IfcPropertyEnumeratedValue 的适用值，但这是首选方式。一个 IfcPropertyEnumeration 实例可以被多个 IfcPropertyEnumeratedValue 实例引用。

沿革 IFC2.0 中的新实体。在 IFC2x 中增强了功能。

IFC2x 变更：实体已从 IfcEnumeratedProperty 重命名。

IFC4 变更：EnumerationValues 特性已设为可选，并保持文件交换的向上兼容性。

8.16.3.8.5 形式化表示（Formal representation） 

ENTITY IfcPropertyEnumeratedValue
 SUBTYPE OF (IfcSimpleProperty);
	EnumerationValues : OPTIONAL LIST [1:?] OF IfcValue;
	EnumerationReference : OPTIONAL IfcPropertyEnumeration;
 WHERE
	WR21 : NOT(EXISTS(EnumerationReference)) 
OR  NOT(EXISTS(EnumerationValues)) 
OR  (SIZEOF(QUERY(temp <* EnumerationValues |
    temp IN EnumerationReference.EnumerationValues))
    = SIZEOF(EnumerationValues));
END_ENTITY;
