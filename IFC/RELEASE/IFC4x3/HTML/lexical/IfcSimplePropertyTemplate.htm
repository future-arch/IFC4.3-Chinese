5.1.3.45 IfcSimplePropertyTemplate（简单属性模板）

注释 该实体不是标准化模式子集或实现级别的一部分。

5.1.3.45.1 语义定义（Semantic definition）

IfcSimplePropertyTemplate 定义了所有动态可扩展属性的模板，包括 IfcSimpleProperty 的子类型或 IfcPhysicalSimpleQuantity 的子类型。各个属性模板根据其 Name 属性进行解释，并可能具有预定义的模板类型、属性单位和属性度量类型。以下属性的正确解释：

  • PrimaryUnit
  • SecondaryUnit
  • PrimaryMeasureType
  • SecondaryMeasureType

由 TemplateType 决定。属性 Enumerators 和 Expression 仅适用于具有特定 TemplateType 的 IfcSimplePropertyTemplate 实例。TemplateType 还控制应使用 IfcSimpleProperty 或 IfcPhysicalSimpleQuantity 的哪个子类型来对应此模板的属性实例。

IfcSimplePropertyTemplate 是 IfcPropertySetTemplate 中包含的模板集合的一部分。该模板可通过反向属性 PartOfPsetTemplate 访问。IfcPropertySetTemplate 可以定义一个或多个 IfcPropertySet（或 IfcElementQuantity）实例。定义分配通过对象化关系 IfcRelDefinesByTemplate 建立，如图1所示。IfcSimplePropertyTemplate 与 IfcSimpleProperty 或 IfcPhysicalSimpleQuantity 的子类型之间没有直接链接。模板与各个属性（或数量）之间的定义关系通过 Name 属性建立。

注释 在 IfcPropertySetTemplate 和 IfcPropertySet（以及 IfcElementQuantity）处的约束保证了包含的属性模板和各个属性的 Name 属性是唯一的。
沿革 IFC4 中的新实体。

5.1.3.45.5 形式化表示（Formal representation） 

ENTITY IfcSimplePropertyTemplate
 SUBTYPE OF (IfcPropertyTemplate);
	TemplateType : OPTIONAL IfcSimplePropertyTemplateTypeEnum;
	PrimaryMeasureType : OPTIONAL IfcLabel;
	SecondaryMeasureType : OPTIONAL IfcLabel;
	Enumerators : OPTIONAL IfcPropertyEnumeration;
	PrimaryUnit : OPTIONAL IfcUnit;
	SecondaryUnit : OPTIONAL IfcUnit;
	Expression : OPTIONAL IfcLabel;
	AccessState : OPTIONAL IfcStateEnum;
END_ENTITY;
