8.8.3.48 IfcTriangulatedIrregularNetwork

注释 该实体不是标准化模式子集或实现级别的一部分。

8.8.3.48.1 语义定义（Semantic definition）

IfcTriangulatedIrregularNetwork 是一个三角形面集，用于表示水平面（域内所有 X 和 Y 坐标具有唯一的 Z 坐标），并为每个面提供额外的标志，指示面之间的断裂线或指定为孔或空腔。三角形应以从上方观察的逆时针顺序（遵循右手定则）定义顶点。

为了可视化，应用程序不应显示标志设置为负数的面（孔、空腔或可能的未来扩展）。

Void 标志应用于指示要排除的面，而不回退到任何其他几何体。此类指定可用于建筑物或结构下方的场地部分。

Hole 标志应用于指示要排除的面，但可以回退到其他几何体。此类指定可用于拟议场地中将保持不变的部分（符合可能已定义的现有场地）。

在组合多个用作输入的表面时，任何标记为 Void 的三角形都将保留为 void，而任何标记为 Hole 的三角形将被覆盖，如果另一个表面在相同的水平位置定义了可见几何体。
