8.4.3.2 IfcCostValue

RV ⓘ
AbV ⓘ

8.4.3.2.1 语义定义（Semantic definition）

IfcCostValue 是一个货币金额或影响货币金额的值。

每个 IfcCostValue 实例还可以有一个类别。可能存在的成本值类型有很多。虽然对不同成本类型名称的含义有广泛的理解，但对于命名成本类型没有通用标准，也没有广泛定义的分类。

以下定义了一些可能应用于 Category 属性的成本类别：

  • 年回报率
  • 奖金
  • 批量采购折扣
  • 合同
  • 咨询费
  • 交付费
  • 估算成本
  • 租赁费
  • 安装费
  • 利率
  • 人工费
  • 租金
  • 标价
  • 维护费
  • 材料费
  • 管理费
  • 邮寄和包装费
  • 利润
  • 采购费
  • 租金
  • 维修费
  • 更换费
  • 销售额
  • 小批量附加费
  • 备件费
  • 仓储费
  • 分包合同
  • 贸易折扣
  • 运输费
  • 损耗补贴
  • 全生命周期成本
注释 In the absence of any well-defined standard, it is recommended that local agreements should be made to define allowable and understandable cost value types within a project or region.
