5.1.3.26 IfcRelAssignsToProcess（关联关系-分配给过程）

注释 该实体不是标准化模式子集或实现级别的一部分。

5.1.3.26.1 语义定义（Semantic definition）

对象化关系 IfcRelAssignsToProcess 处理将一个或多个对象分配给过程或活动的关联关系。对象可以是过程操作的产品。过程和活动可以操作产品以外的事物，并且可以通过输入和输出以外的方式进行操作。

示例 在估算或调度过程中定义描述设计任务（生成文档）、采购任务（生成建筑材料）、计划任务（生成过程）等过程可能很常见。此外，过程操作事物的方式可能包括"安装"、"完成"、"运输"、"移除"等。这些方式被描述为操作类型。

继承属性 RelatedObjects 提供了过程所操作的对象或对象类型的引用。RelatingProcess 是操作该对象的过程或过程类型。操作类型在继承属性 Name 中捕获。

注释 关于 Name 属性有效和可识别值的约定是视图定义和实施者协议的一部分。
沿革 IFC1.5 中的新实体。在 IFC2x 中从 IfcRelProcessOperatesOn 重命名而来。
IFC4 变更 RelatingProcess 数据类型已扩展为同时包含 IfcTypeProcess

5.1.3.26.6 形式化表示（Formal representation） 

ENTITY IfcRelAssignsToProcess
 SUBTYPE OF (IfcRelAssigns);
	RelatingProcess : IfcProcessSelect;
	QuantityInProcess : OPTIONAL IfcMeasureWithUnit;
 WHERE
	NoSelfReference : SIZEOF(QUERY(Temp <* SELF\IfcRelAssigns.RelatedObjects | RelatingProcess :=: Temp)) = 0;
END_ENTITY;
