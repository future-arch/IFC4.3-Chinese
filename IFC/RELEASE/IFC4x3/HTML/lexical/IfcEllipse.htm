8.9.3.32 IfcEllipse（椭圆）

注释 该实体不是标准化模式子集或实现级别的一部分。

8.9.3.32.1 语义定义（Semantic definition）

IfcEllipse 是由一组到两个固定点距离之和为常数的点构成的曲线。

继承的属性 SELF\IfcConic.Position.Location 是 IfcEllipse 的中心，继承的属性 SELF\IfcConic.Position.P[1] 是 SemiAxis1 的方向。

IfcEllipse 在三维位置坐标系中的定义如图 8.9.3.32.A 所示。

它被放置在其所属表示元素的物体坐标系内。

注释 椭圆弧段通过使用 IfcTrimmedCurve 定义，其中 BasisCurve 为 IfcEllipse。
参考 根据 ISO/CD 10303-42:1992 的定义

椭圆是由半长轴和半短轴长度、曲线位置（中心或焦点连线中点）和方向定义的圆锥曲线。数据应按如下方式解释：

C = SELF\IfcConic.Position.Location
x = SELF\IfcConic.Position.P[1]
y = SELF\IfcConic.Position.P[2]
z = SELF\IfcConic.Position.P[3]
R1 = SemiAxis1
R2 = SemiAxis2

椭圆的参数化表示为：

$$ \lambda(u) = C + (R_1\cos(u))x + (R_2\sin(u))y $$

参数化范围为 0 ≤ u <≤ 2π（0 ≤ u ≤ 360 度）。在上述定义的定位坐标系中，椭圆满足方程 C = 0，其中

$$ C(x,y,z) = \frac{x^2}{R_1^2} + \frac{y^2}{R_2^2} - 1 $$

椭圆上任意点的正方向为曲线在该点的切线方向 T，其中

$$ T = (-C_y,C_x,0) $$
