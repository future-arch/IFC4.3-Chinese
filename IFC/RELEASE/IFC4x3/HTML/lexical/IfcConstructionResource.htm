7.3.3.7 IfcConstructionResource（施工资源）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

7.3.3.7.1 语义定义（Semantic definition）

IfcConstructionResource 是建筑项目中使用的不同资源的抽象泛化，主要包括劳动力、材料、设备和产品资源，以及分包资源和班组资源等聚合资源。

资源表示"某物的使用"，不一定对应单个项目（如人员或车辆），而是代表具有有限可用性的项目池，如普通劳动力或设备车队。资源可以表示通用资源池（没有任何任务分配）或特定任务的资源分配（具有 IfcTask 分配）。

沿革 IFC2x2 中的新实体。
IFC4 变更 修改为将 ResourceIdentifer 和 ResourceGroup（重命名为 LongDescription）提升至超类型 IfcResource，并添加了所述属性。

7.3.3.7.1.1 声明使用定义

根级资源（特别是 IfcCrewResource 或 IfcSubContractResource）通过 IfcRelDeclares 在项目内声明，其中 RelatingContext 引用单个 IfcProject，RelatedObjects 引用一个或多个 IfcConstructionResource 以及项目内的其他根级对象。

7.3.3.7.1.2 分配使用定义

资源可以通过 IfcRelAssignsToActor 分配给参与者，其中 RelatingActor 引用 IfcActor，RelatedObjects 引用一个或多个 IfcConstructionResource 或其他对象。这种关系表示负责分配资源的参与者，例如划分为特定任务的分配、委托给其他参与者和/或随时间进行调度。请注意，此分配不表示执行工作的人员或组织；这由 IfcRelAssignsToResource 表示。负责资源的参与者可能与执行工作的任何参与者相同或不同。

资源可以通过 IfcRelAssignsToControl 分配给控制项，其中 RelatingControl 引用 IfcControl，RelatedObjects 引用一个或多个 IfcConstructionResource 或其他对象。最常见的是分配 IfcWorkCalendar 表示资源的可用性，此类日历嵌套在基本日历或 IfcWorkPlan 中，而 IfcWorkPlan 又分配给 IfcProject。

资源可以通过 IfcRelAssignsToGroup 分配给组，其中 RelatingGroup 引用 IfcGroup，RelatedObjects 引用一个或多个 IfcConstructionResource 或其他对象。最常见的是分配 IfcAsset 表示要跟踪的资产，此类资产嵌套在 IfcInventory 中，而 IfcInventory 又分配给 IfcProject。

资源可以通过 IfcRelAssignsToProduct 分配给产品，其中 RelatingProduct 引用 IfcProduct，RelatedObjects 引用一个或多个 IfcConstructionResource 或其他对象。最常见的是分配 IfcElement 子类型表示要构建的产品，此类产品连接到空间结构，而空间结构又在 IfcProject 内聚合。

资源可以通过 IfcRelAssignsToProcess 分配给过程，其中 RelatingProcess 引用 IfcProcess，RelatedObjects 引用一个或多个 IfcConstructionResource 或其他对象。最常见的是分配 IfcTask 表示资源要执行的任务，此类任务嵌套在摘要任务中，而摘要任务又分配给 IfcProject。

资源可以使用 IfcRelAssignsToResource 分配其他对象，其中 RelatingResource 引用 IfcConstructionResource，RelatedObjects 引用一个或多个对象，如 IfcActor 或 IfcProduct 子类型。这种关系表示分配特定对象以满足资源使用。

图 7.3.3.7.1.2.A 说明了资源分配。

7.3.3.7.1.3 基线使用定义

资源可以使用关系 IfcRelDefinesByObject 定义任意数量的基线，其中 RelatingObject 是"当前"资源，RelatedObjects 由多个"基线"资源组成，每个基线资源表示资源在早期时间点存在的副本，如图 7.3.3.7.1.3.A 所示。每个基线 IfcConstructionResource 通过其嵌套的 IfcRelAssignsToControl 关系与 IfcWorkSchedule 标识，该 IfcWorkSchedule 具有 PredefinedType=BASELINE，IfcWorkSchedule.CreationDate 表示基线日期，IfcWorkSchedule.Name 表示基线名称。

7.3.3.7.5 概念用法（Concept usage）

7.3.3.7.6 形式化表示（Formal representation） 

ENTITY IfcConstructionResource
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcConstructionEquipmentResource
	,IfcConstructionMaterialResource
	,IfcConstructionProductResource
	,IfcCrewResource
	,IfcLaborResource
	,IfcSubContractResource))
 SUBTYPE OF (IfcResource);
	Usage : OPTIONAL IfcResourceTime;
	BaseCosts : OPTIONAL LIST [1:?] OF IfcAppliedValue;
	BaseQuantity : OPTIONAL IfcPhysicalQuantity;
END_ENTITY;
