8.15.3.13 IfcLShapeProfileDef

注释 该实体不是标准化模式子集或实现级别的一部分。

8.15.3.13.1 语义定义（Semantic definition）

IfcLShapeProfileDef 定义了一个 L 形截面轮廓，它提供了 L 形截面（也包括等边 L 形轮廓）的定义参数，用于扫掠面积实体。其参数和相对于位置坐标系的定向根据下图所示。较短的腿与正 Position.P[1] 轴方向相同，较长或相等的腿与正 Position.P[2] 轴方向相同。位置坐标系的中心位于轮廓的边界框中心。

沿革 New entity in IFC2x2.
IFC2x3 变更 All profile origins are now in the center of the bounding box.
IFC4 变更 Types of FilletRadius and EdgeRadius were relaxed to allow for zero values. Trailing attributes CentreOfGravityInX and CentreOfGravityInY deleted, use respective properties in IfcProfileProperties instead. WHERE rule which required Width <= Depth removed.

图1和图2说明了等边和不等边 L 形截面定义的参数。

参数化轮廓定义了自己的位置坐标系。底层坐标系由使用轮廓定义的扫掠面积实体定义。它是 IfcSweptAreaSolid.Position 的 xy 平面，通过使用位置偏移，参数化轮廓可以定位在中心（使用 x,y 偏移量 = 0.），或相对于轮廓的任何位置。

在图中所示的示例中，IfcProfileProperties 中的 'CentreOfGravityInX' 和 'CentreOfGravityInY' 属性（如果提供）均为负值。

注释 The black coordinate axes show the underlying coordinate system of the swept surface or swept area solid

通过使用 Position 属性，轮廓被插入到扫掠面积实体的底层坐标系中。在此示例中（重力基点），IfcAxis2Placement2D 的属性值为：

Location = IfcCartesianPoint(+|CentreOfGravityInX|, +|CentreOfGravityInY|)
RefDirection = NIL (默认为 1.,0.)

在图中所示的示例中，Position.Location 的 x 和 y 值，即 |CentreOfGravityInX| 和 |CentreOfGravityInY| 的度量均为正值。另一方面，IfcProfileProperties 中名为 'CentreOfGravityInX' 和 'CentreOfGravityInY' 的属性（如果提供）现在都必须设置为 0，因为结果轮廓定义的重心位于坐标原点。

8.15.3.13.7 形式化表示（Formal representation） 

ENTITY IfcLShapeProfileDef
 SUBTYPE OF (IfcParameterizedProfileDef);
	Depth : IfcPositiveLengthMeasure;
	Width : OPTIONAL IfcPositiveLengthMeasure;
	Thickness : IfcPositiveLengthMeasure;
	FilletRadius : OPTIONAL IfcNonNegativeLengthMeasure;
	EdgeRadius : OPTIONAL IfcNonNegativeLengthMeasure;
	LegSlope : OPTIONAL IfcPlaneAngleMeasure;
 WHERE
	ValidThickness : (Thickness < Depth) AND (NOT(EXISTS(Width)) OR (Thickness < Width));
END_ENTITY;
