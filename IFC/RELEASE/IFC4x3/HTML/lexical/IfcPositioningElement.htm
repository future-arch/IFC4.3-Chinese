5.4.3.42 IfcPositioningElement（定位构件）

注释 该实体是 IfcProduct 或 IfcTypeProduct 的子类型，因此是每个标准化模式子集和实现级别的一部分。

抽象 该定义可能不会被直接实例化。

5.4.3.42.1 语义定义（Semantic definition）

用于相对定位其他构件的定位和注释构件的新抽象实体定义。

示例 网格是一种定位构件，主要用于竖向结构中定位建筑构件；路线是一种线性定位构件，主要用于基础设施工程中定位地理和土木构件。
示例 路线是一种线性定位构件，用于采用线性参照方法定位其他项目。有关线性参照方法和表达式的一般信息，请参见 ISO 19148《线性参照》。

5.4.3.42.6 概念用法（Concept usage）

5.4.3.42.7 形式化表示（Formal representation） 

ENTITY IfcPositioningElement
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcGrid
	,IfcLinearPositioningElement
	,IfcReferent))
 SUBTYPE OF (IfcProduct);
 INVERSE
	ContainedInStructure : SET [0:1] OF IfcRelContainedInSpatialStructure FOR RelatedElements;
	Positions : SET [0:?] OF IfcRelPositions FOR RelatingPositioningElement;
 WHERE
	HasPlacement : EXISTS(SELF\IfcProduct.ObjectPlacement);
END_ENTITY;
