8.16.3.13 IfcPropertyTableValue

RV ⓘ
AbV ⓘ

8.16.3.13.1 语义定义（Semantic definition）

IfcPropertyTableValue 是一个具有值范围的属性，由一个属性对象定义，该对象具有两个（数值或描述性）值列表。这些值指定了一个两列的表格。定义值提供第一列，并建立定义值（第二列）的范围。可选的 Expression 特性可以给出用于推导范围值的方程，该方程仅供参考。

IfcPropertyTableValue 定义了一个定义/已定义属性值组合，其中给出了属性名称、具有度量类型（以及可选的定义和已定义值的单位）的表格。

注释 The IfcPropertyTableValue only captures properties that can be expressed by a table with two columns. Use IfcPropertyReferenceValue with the PropertyReference being an IfcTable to express all those properties that require a table with tree or more columns.

单位由 DefiningUnit 和 DefinedUnit 特性处理，表 8.16.3.13.A 显示了表格值属性的示例：

  • 如果未提供 DefiningUnit 或 DefinedUnit 特性，则单位已由 IfcMeasureValue 或 IfcDerivedMeasureValue 的类型隐含。关联单位可在项目级别全局定义的 IfcUnitAssignment (IfcProject.UnitsInContext) 中找到。
  • 如果提供了 DefiningUnit 或 DefinedUnit 特性，则 Unit 特性分配的单位将覆盖全局分配的单位。

IfcPropertyTableValue 允许为属性描述指定一个定义/已定义值对的表格。可选的 CurveInterpolation 特性允许确定两个给定值之间的间隔。

沿革 IFC2.0 中的新实体。

IFC4 变更：DefiningValues 和 DefinedValues 特性已设为可选，并保持文件交换的向上兼容性。添加了 CurveInterpolation 特性。

非正式命题

  1. DefinedValues 列表和 DefiningValues 列表是对应的列表。

8.16.3.13.5 形式化表示（Formal representation） 

ENTITY IfcPropertyTableValue
 SUBTYPE OF (IfcSimpleProperty);
	DefiningValues : OPTIONAL LIST [1:?] OF UNIQUE IfcValue;
	DefinedValues : OPTIONAL LIST [1:?] OF IfcValue;
	Expression : OPTIONAL IfcText;
	DefiningUnit : OPTIONAL IfcUnit;
	DefinedUnit : OPTIONAL IfcUnit;
	CurveInterpolation : OPTIONAL IfcCurveInterpolationEnum;
 WHERE
	WR21 : (NOT(EXISTS(DefiningValues)) AND NOT(EXISTS(DefinedValues)))
OR (SIZEOF(DefiningValues) = SIZEOF(DefinedValues));
	WR22 : NOT(EXISTS(DefiningValues)) OR 
(SIZEOF(QUERY(temp <* SELF.DefiningValues | TYPEOF(temp) <> TYPEOF(SELF.DefiningValues[1])
)) = 0);
	WR23 : NOT(EXISTS(DefinedValues)) OR 
(SIZEOF(QUERY(temp <* SELF.DefinedValues | TYPEOF(temp) <> TYPEOF(SELF.DefinedValues[1])
)) = 0);
END_ENTITY;
