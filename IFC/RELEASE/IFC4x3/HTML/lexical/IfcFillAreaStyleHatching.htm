8.12.3.15 IfcFillAreaStyleHatching

RV ⓘ
AbV ⓘ

8.12.3.15.1 语义定义（Semantic definition）

IfcFillAreaStyleHatching 用于定义简单的、基于矢量的填充图案，这些图案基于带样式的直线。曲线字体、颜色和粗细由 HatchLineAppearance 给出，角度由 HatchLineAngle 给出，到下一条填充线的距离由 StartOfNextHatchLine 给出，它可以是偏移距离或矢量。

注释 根据 ISO 10303-46 定义： 填充区域样式填充定义了一个带样式的曲线图案，用于填充注释填充区域或表面。
注释 如果填充图案涉及两条（可能交叉的）填充线，则应将 IfcFillAreaStyleHatching 的两个实例分配给 IfcFillAreaStyle。两者共享填充的相同（虚拟）原点，该原点由参考填充线使用（如果存在偏移，则使用 PointOfReferenceHatchLine）。

为了更好地控制填充外观，当使用非连续曲线字体的填充线时，PatternStart 允许沿着参考填充线偏移曲线字体图案的起点（如果未给出，则 PatternStart 距离虚拟原点为零）。如果参考填充线不通过原点（虚拟填充坐标系的原点），则可以使用 PatternStart 进行偏移。

注释 PatternStart 的坐标是相对于 IfcAnnotationFillArea 的对象坐标原点给出的，或者如果存在，则是相对于 IfcAnnotationFillArea 的 FillAreaTarget 属性。度量值以全局绘图长度单位给出，表示模型填充，并且可以通过 TargetScale 转换为绘图单位，以用于有比例依赖的 IfcGeometricRepresentationSubContext（如果提供）。

DEPRECATION 不推荐使用 PointOfReferenceHatchLine。

图 8.12.3.15.A 显示了通过在 HatchLineAppearance 中使用曲线字体“continuous”（连续）给出的简单填充。

填充线的距离由正长度度量值给出。角度（此处为 45 度，如果以度为单位）由 HatchLineAngle 提供。

在此示例中，PatternStart 设置为 NIL ($)。

图 8.12.3.15.B 显示了示例 1 的填充，使用了不同的曲线字体在 HatchLineAppearance 中。

填充线的距离由正长度度量值给出，因此字体图案的起点位于下一条填充线上的点，该点由垂直于参考填充线上原点的矢量给出。

在此示例中，PatternStart 设置为 NIL ($)。

图 8.12.3.15.C 使用了示例 2 的填充，并使用矢量来确定下一条填充线的图案起点。

图案起点是 IfcCurveStyleFont.PatternList 中第一个可见曲线字体图案段的开始。

在此示例中，PatternStart 设置为 NIL ($)。

图 8.12.3.15.D 使用了示例 3 的填充，其中图案起点从参考填充线上的原点偏移。也就是说，第一个可见曲线字体图案段现在不从参考填充线上的原点开始。

此示例显示了在一个 IfcFillAreaStyle 中使用 IfcFillAreaStyleHatching 的两个实例。

请注意，PatternStart 现在将参考填充线从原点以及曲线图案的起点进行了位移。这可用于在一个 IfcFillAreaStyle 中使用多个 IfcFillAreaStyleHatching 来放置相互偏移的填充线行。

注释 实体改编自 ISO10303-46 中定义的 fill_area_style_hatching。
沿革 IFC2x2 新增实体。
IFC2x3 变更 IfcFillAreaStyleHatching 已更改，将 PatternStart 和 PointOfReferenceHatchLine 属性设为 OPTIONAL。StartOfNextHatchLine 属性已更改为 SELECT，并增加了 IfcPositiveLengthMeasure 的选择。文件交换的向上兼容性得到保证。
IFC4 变更 StartOfNextHatchLine 的属性数据类型已更改为 IfcPositiveLengthMeasure 和 IfcVector 的选择。

8.12.3.15.5 形式化表示（Formal representation） 

ENTITY IfcFillAreaStyleHatching
 SUBTYPE OF (IfcGeometricRepresentationItem);
	HatchLineAppearance : IfcCurveStyle;
	StartOfNextHatchLine : IfcHatchLineDistanceSelect;
	PointOfReferenceHatchLine : OPTIONAL IfcCartesianPoint;
	PatternStart : OPTIONAL IfcCartesianPoint;
	HatchLineAngle : IfcPlaneAngleMeasure;
 WHERE
	PatternStart2D : NOT(EXISTS(PatternStart)) OR (PatternStart.Dim = 2)
;
	RefHatchLine2D : NOT(EXISTS(PointOfReferenceHatchLine)) OR (PointOfReferenceHatchLine.Dim = 2);
END_ENTITY;
