5.1.3.34 IfcRelDeclares（关联关系-声明）

RV ⓘ
AbV ⓘ

5.1.3.34.1 语义定义（Semantic definition）

对象化关系 IfcRelDeclares 处理对象（IfcObject 的子类型）或属性（IfcPropertyDefinition 的子类型）向项目或项目库（由 IfcProject 或 IfcProjectLibrary 表示）的声明。

该关系处理其他对象（如 IfcActor 或 IfcTypeObject）向项目或项目库的分配。属性 RelatedDefinitions 提供了对作为上下文内对象的第一级对象的引用。所有与第一级对象相关的其他对象也在该上下文中定义。

注释 每个对象（作为 IfcObject 的子类型）必须在单个 IfcProject 的上下文中声明，或在分配给单个 IfcProject 的 IfcProjectLibrary 中声明。此声明是传递性的。例如：作为第一级对象的 IfcWorkPlan 通过 IfcRelDeclares 在 IfcProject 的上下文中声明，所有相关的 IfcWorkSchedule 通过 IfcWorkPlan 以传递方式与上下文相关。
注释 使用 IfcRelDeclares 的此分配排除了已通过其他关系与 IfcProject 相关的每个对象（作为 IfcObject 的子类型）。例如，通过 IfcRelAggregates 关系与 IfcProject 相关的 IfcSpatialStructureElement；以及通过 IfcRelContainedInSpatialStructure 关系包含在这些 IfcSpatialStructureElement 中的 IfcProduct 子类型。

RelatingContext 是包含所有对象的项目或项目库。在 IfcProject 或 IfcProjectLibrary 中定义的单位分配和表示上下文适用于所有这些对象。

沿革 IFC4 中的新实体。

5.1.3.34.6 形式化表示（Formal representation） 

ENTITY IfcRelDeclares
 SUBTYPE OF (IfcRelationship);
	RelatingContext : IfcContext;
	RelatedDefinitions : SET [1:?] OF IfcDefinitionSelect;
 WHERE
	NoSelfReference : SIZEOF(QUERY(Temp <* RelatedDefinitions | RelatingContext :=: Temp)) = 0;
END_ENTITY;
