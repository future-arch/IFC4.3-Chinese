6.2.4.22 Pset_SoundAttenuation（消声）

6.2.4.22.1 语义定义（Semantic definition）

用于在建筑设备与流体输配系统背景下，代表设备采集某点声压的通用定义。为表示来自 IfcDistributionFlowElement 实例在特定位置的声学数值，应通过 IfcRelAssignsToProduct 关系将 IfcAnnotation 实例分配给该 IfcDistributionFlowElement。该 IfcAnnotation 应按 IfcAnnotation 中的说明，将对象类型指定为"Sound"，并采用由单个 IfcPoint 子类型构成的"标注点"几何表达。本属性集在每个频带上均会在对象上多次实例化。

历史记录：IFC Release 2x4中新增的属性集。

6.2.4.22.2 适用实体

PSET_OCCURRENCEDRIVEN 由此 IfcPropertySetTemplate 定义的属性集只能分配给 IfcObject 的子类型。
