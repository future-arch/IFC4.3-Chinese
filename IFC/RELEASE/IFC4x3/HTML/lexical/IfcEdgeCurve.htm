8.20.3.5 IfcEdgeCurve（边曲线）

注释 该实体不是标准化模式子集或实现级别的一部分。

8.20.3.5.1 语义定义（Semantic definition）

IfcEdgeCurve 定义了两个顶点之间的拓扑连接，并包含连接的几何表示。

注释 拓扑用于修剪边的几何形状。无需对边进行几何修剪以匹配拓扑。
示例 图 8.20.3.5.A 展示了一个示例，其中边的几何形状由无界曲线（此处为 IfcCircle）给出。边界由 EdgeStart 和 EdgeEnd 提供，IfcEdgeCurve 的拓扑方向通过 SameSense = FALSE 与 IfcCircle 的方向相反。
注释 根据 ISO/CD 10303-42:1992 的定义 边曲线是边的一种特殊子类型，其几何形状已完全定义。几何形状通过将边与一条可能无界的曲线关联来定义。由于拓扑方向和几何方向可能相反，使用指示符（相同方向）来标识边和曲线的方向是否一致或相反。布尔值指示曲线方向是否与边方向一致（TRUE）或相反（FALSE）。与边顶点关联的任何几何形状应与边的几何形状一致。多个边可以引用同一条曲线。
注释 实体改编自 ISO 10303-42 中定义的 edge_curve。
沿革 IFC2x 中的新实体。

非正式命题

  1. 边曲线的域正式定义为其边几何形状被顶点修剪后的域。该域不包括顶点。
  2. 边曲线具有非零有限范围。
  3. 边曲线是流形。
  4. 边曲线是弧连通的。
  5. 边起点不是边域的一部分。
  6. 边终点不是边域的一部分。
  7. 顶点几何形状应与边几何形状一致。
