8.16.3.9 IfcPropertyEnumeration

RV ⓘ
AbV ⓘ

8.16.3.9.1 语义定义（Semantic definition）

IfcPropertyEnumeration 是简单值或度量值的集合，用于定义从中选择“枚举值”的规定备选集。这使得可以将枚举值包含在属性集中。IfcPropertyEnumeration 为枚举提供名称以及一组唯一的（数值或描述性）值（可能已分配度量类型）。该实体定义了可与 IfcPropertyEnumeratedValue 类型属性一起（或单独）交换的潜在枚举器列表，这些属性从该枚举中选择其实际属性值。

单位由 Unit 特性处理，表 8.16.3.9.A 显示了无单位属性枚举的示例：

  • 如果未提供 Unit 特性，则单位已由 IfcMeasureValue 或 IfcDerivedMeasureValue 的类型隐含。关联单位可在项目级别全局定义的 IfcUnitAssignment (IfcProject.UnitsInContext) 中找到。
  • 如果提供了 Unit 特性，则 Unit 特性分配的单位将覆盖全局分配的单位。
沿革 IFC2.0 中的新实体，在 IFC2x 中增强了功能。

IFC2x 变更：实体已从 IfcEnumeration 重命名。

8.16.3.9.5 形式化表示（Formal representation） 

ENTITY IfcPropertyEnumeration
 SUBTYPE OF (IfcPropertyAbstraction);
	Name : IfcLabel;
	EnumerationValues : LIST [1:?] OF UNIQUE IfcValue;
	Unit : OPTIONAL IfcUnit;
 UNIQUE
	UR1 : Name;
 WHERE
	WR01 : SIZEOF(QUERY(temp <* SELF.EnumerationValues | 
  NOT(TYPEOF(SELF.EnumerationValues[1]) = TYPEOF(temp))
  )) = 0;
END_ENTITY;
