5.4.3.46 IfcRelAssociatesMaterial（关系关联材料）

RV ⓘ
AbV ⓘ

5.4.3.46.1 语义定义（Semantic definition）

IfcRelAssociatesMaterial 是一种对象化的关系，用于关联材料定义与适用该材料定义的构件或构件类型。

材料定义可以：

  • 作为层集或剖面集的具体应用分配给构件实例
  • 作为层集、剖面集、组分集或单一材料分配给构件实例或构件类型

材料可以按层布置并应用于分层构件。典型构件包括墙体和楼板。

  • IfcMaterialLayerSet，用于分层构件，指示分层方向和单个层厚度
  • IfcMaterialLayerSetUsage，即具有沿构件参考轴或表面定位信息的材料层集
注释 由于材料层集使用是基于实例的信息，适用于每个单独构件，因此不能分配给构件类型。

材料可应用于剖面。使用剖面材料的典型构件包括梁、柱、杆件

  • IfcMaterialProfileSet，即分配给一组剖面的材料集，默认情况下将单一材料分配给单一剖面
  • IfcMaterialProfileSetUsage，即具有相对于构件轴定位信息的材料剖面集，也称为基点
注释 由于材料剖面集使用是基于实例的信息，适用于每个单独构件，因此不能分配给构件类型。

材料可以按基于组件的构件的标识部分进行排列。典型构件包括门/窗（具有如衬里、框架和玻璃等组件）或分布构件。

  • IfcMaterialConstituentSet，用于基于组件的构件，通过关键字指示材料组分适用的组件
注释 有关这些关键字的提供，请参见 IfcElement 或 IfcElementType 各适用子类型的材料使用定义。

作为备用方案，或在仅需要单一材料信息的情况下，材料信息可以直接关联

  • 单一 IfcMaterial，适用于材料使用定义不禁止其直接关联的任何构件
  • IfcMaterialList，例如用于复合构件，不提供任何关于不同材料如何排列的信息

IFC4 弃用说明 从 IFC4 开始，IfcMaterialList 的使用已被弃用。请改用 IfcMaterialConstituentSet。

IfcRelAssociatesMaterial 关系是 IfcRelAssociates 关系的一种特殊类型。它可应用于 IfcElement 的子类型和 IfcElementType 的子类型。

  • IfcElement 通过从 IfcObject 继承的 HasAssociations 属性与其材料定义具有反向关系
  • IfcElementType 通过从 IfcObjectDefinition 继承的 HasAssociations 属性与其材料定义具有反向关系

如果构件实例（通过 IfcElement 实例）和构件类型（通过 IfcElementType 实例，通过 IfcRelDefinesByType 连接）都有关联材料，则与构件实例关联的材料将覆盖与构件类型关联的材料。

沿革 IFC2x 中的新实体。

非正式命题

  1. IfcMaterialLayerSetUsage 不应与 IfcElementType 的子类型关联，应仅与单个实例关联
  2. IfcMaterialProfileSetUsage 不应与 IfcElementType 的子类型关联，应仅与单个实例关联

5.4.3.46.6 形式化表示（Formal representation） 

ENTITY IfcRelAssociatesMaterial
 SUBTYPE OF (IfcRelAssociates);
	RelatingMaterial : IfcMaterialSelect;
 WHERE
	AllowedElements : SIZEOF(QUERY(temp <* SELF\IfcRelAssociates.RelatedObjects | (
  SIZEOF(TYPEOF(temp) * [   
    'IFC4X3_DEV_62887e3f.IFCELEMENT', 
    'IFC4X3_DEV_62887e3f.IFCELEMENTTYPE',
    'IFC4X3_DEV_62887e3f.IFCSTRUCTURALMEMBER',
    'IFC4X3_DEV_62887e3f.IFCPORT']) = 0) 
)) = 0;
	NoVoidElement : SIZEOF(QUERY(temp <* SELF\IfcRelAssociates.RelatedObjects | 
  ('IFC4X3_DEV_62887e3f.IFCFEATUREELEMENTSUBTRACTION' IN TYPEOF(temp)) OR 
  ('IFC4X3_DEV_62887e3f.IFCVIRTUALELEMENT' IN TYPEOF(temp)) 
)) = 0;
END_ENTITY;
