8.10.2.1 IfcCardinalPointReference（基准点参考）

8.10.2.1.1 语义定义（Semantic definition）

IfcCardinalPointReference 是一种对截面轮廓关键点的索引参考。该索引用于描述构件截面与同一构件参考轴之间的空间关系。

沿革 IFC4 中的新增类型。

索引 1...9 指向轮廓边界框上的点。索引 10...19 指向由几何形心（通常为重心）和剪切中心定义的点，及其与边界框坐标的组合。本规范特别规定了以下索引值：

  1. 左下
  2. 中下
  3. 右下
  4. 左中
  5. 中心
  6. 右中
  7. 左上
  8. 中上
  9. 右上
  10. 几何形心
  11. 与几何形心水平的下边点
  12. 与几何形心垂直的左边点
  13. 与几何形心垂直的右边点
  14. 与几何形心水平的上边点
  15. 剪切中心
  16. 与剪切中心水平的下边点
  17. 与剪切中心垂直的左边点
  18. 与剪切中心垂直的右边点
  19. 与剪切中心水平的上边点
  20. 管道或槽型轮廓内部最低点（平底中心）

其他索引值可能存在，但不在本规范范围内。

图 8.10.2.1.A 展示了基准点取值。图 8.10.2.1.B 展示了具有任意轮廓（IfcArbitraryClosedProfileDef）的拉伸形状示例，该轮廓在构件轴上按"右中"对齐。视线方向沿拉伸方向 Z，指向上图绘制平面内。因此，"左"方向为 IfcProfileDef 的正 X 方向，"上"方向为 IfcProfileDef 的正 Y 方向。

8.10.2.1.2 形式化命题（Formal propositions）

8.10.2.1.3 形式化表示（Formal representation） 

TYPE IfcCardinalPointReference = INTEGER;
 WHERE
	 GreaterThanZero : SELF > 0;
END_TYPE;
