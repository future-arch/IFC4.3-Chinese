8.10.2.1 IfcCardinalPointReference

8.10.2.1.1 语义定义（Semantic definition）

IfcCardinalPointReference 是截面型材重要点的索引引用。此索引用于描述构件截面与同一构件的参考轴之间的空间关系。

HISTORY  New Type in IFC4.

索引 1...9 指的是型材边界框上的点。索引 10...19 指的是由几何质心（通常是重力中心）和剪切中心定义的点，以及它们与边界框坐标的组合。特别是，本规范中指定了以下索引值：

  1. 左下
  2. 中下
  3. 右下
  4. 左中
  5. 中中
  6. 右中
  7. 左上
  8. 中上
  9. 右上
  10. 几何质心
  11. 与几何质心对齐的底部
  12. 与几何质心对齐的左侧
  13. 与几何质心对齐的右侧
  14. 与几何质心对齐的顶部
  15. 剪切中心
  16. 与剪切中心对齐的底部
  17. 与剪切中心对齐的左侧
  18. 与剪切中心对齐的右侧
  19. 与剪切中心对齐的顶部
  20. 管道或通道型材内部的最低点（平底中心）

其他索引值是可能的，但超出了本规范的范围。

图 8.10.2.1.A 说明了基数点值。图 8.10.2.1.B 说明了一个具有任意型材 (IfcArbitraryClosedProfileDef) 的挤出形状，该形状与构件轴对齐为“中右”。视线遵循挤出方向 Z，该方向指向上述图的绘图平面。“左”在 IfcProfileDef 的正 X 方向。“上”在 IfcProfileDef 的正 Y 方向。

8.10.2.1.2 形式化命题（Formal propositions）

8.10.2.1.3 形式化表示（Formal representation） 

TYPE IfcCardinalPointReference = INTEGER;
 WHERE
	 GreaterThanZero : SELF > 0;
END_TYPE;
