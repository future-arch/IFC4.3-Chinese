8.8.3.43 IfcSweptDiskSolid

RV ⓘ
AbV ⓘ

8.8.3.43.1 语义定义（Semantic definition）

IfcSweptDiskSolid 通过一种扫掠表示方案表示三维形状，该方案允许一个二维圆形有界平面沿着三维 Directrix 在空间中扫掠。

StartParam 和 EndParam 参数是可选的，如果未提供，则默认为 Directrix 的起点和终点。只有当 Directrix 由有界曲线或闭合曲线给出时，才允许省略 StartParam 和 EndParam 的值。

如果 Directrix 的连续段之间的过渡不是切线连续的，则生成的实体是通过两个段之间的半角斜接创建的。非正式命题限制了两个非切线连续段之间的允许角度。

图 8.8.3.43.A 说明了一个示例。

  • Directrix 由 IfcIndexedPolyCurve 提供，包含线段和圆弧段，这些线段和圆弧段在各段之间是切线连续的
  • Directrix 是一条有界且开放的曲线
  • 未提供 StartParam 和 EndParam，起点和终点默认为 Directrix 的有界曲线的起点和终点
注释 尽管示例显示 Directrix 是平面参考面上的多段线，但 IfcSweptDiskSolid 的定义不限于基于平面曲线。然而，视图定义或实施者协议可能提供限制。
注释 几何项 IfcIndexedPolyCurve 与 IfcCompositeCurve 相比提供了更紧凑的表示，因此是 Directrix 的首选曲线表示。
注释 根据 ISO/CD 10303-42:1992 定义 扫掠圆盘实体是通过沿着三维曲线扫掠圆盘而产生的实体。在扫掠操作过程中，圆盘平面的法线方向与导向曲线的切线方向一致，并且圆盘的中心位于导向曲线上。圆盘可以选择性地具有中心孔，在这种情况下，生成的实体将有一个贯通孔，或者当导向曲线形成闭合曲线时，有一个内部空腔。
注释 实体改编自 ISO 10303-42 中定义的 swept_disk_solid。
沿革 IFC2x2 中的新实体。
IFC4 变更 属性 StartParam 和 EndParam 已设为可选。

非正式命题

  1. 如果 Directrix 曲线定义不是切线连续的，则段之间的过渡必须在切线不连续的可接受范围内。非常尖锐的边缘可能导致几乎不可能进行斜接。实施者协议可以定义切线不连续的可接受范围。
  2. Directrix 的段必须足够长才能应用 Radius。如果圆弧段构成 Directrix 的一部分，则其半径必须大于圆盘 Radius。
  3. Directrix 不得基于相交曲线。

8.8.3.43.5 形式化表示（Formal representation） 

ENTITY IfcSweptDiskSolid
 SUPERTYPE OF (ONEOF
	(IfcSweptDiskSolidPolygonal))
 SUBTYPE OF (IfcSolidModel);
	Directrix : IfcCurve;
	Radius : IfcPositiveLengthMeasure;
	InnerRadius : OPTIONAL IfcPositiveLengthMeasure;
	StartParam : OPTIONAL IfcParameterValue;
	EndParam : OPTIONAL IfcParameterValue;
 WHERE
	DirectrixBounded : (EXISTS(StartParam) AND EXISTS(EndParam)) OR 
(SIZEOF(['IFC4X3_DEV_62887e3f.IFCCONIC', 'IFC4X3_DEV_62887e3f.IFCBOUNDEDCURVE'] * TYPEOF(Directrix)) = 1);
	DirectrixDim : Directrix.Dim = 3;
	InnerRadiusSize : (NOT EXISTS(InnerRadius)) OR (Radius > InnerRadius);
END_ENTITY;
