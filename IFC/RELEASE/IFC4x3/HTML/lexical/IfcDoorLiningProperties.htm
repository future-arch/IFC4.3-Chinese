7.1.3.1 IfcDoorLiningProperties（门框属性）

注释 该实体不是标准化模式子集或实现级别的一部分。

弃用 该定义将在标准的未来主版本中移除。

7.1.3.1.1 语义定义（Semantic definition）

门框是用于固定门扇位置的框架。门框用于悬挂门扇。门框参数定义了门框的几何相关参数。

注释 只有当属性 IfcDoorType.ParameterTakesPrecedence 设置为 TRUE 时，接收应用程序才应使用 IfcDoorLiningProperties 来参数化定义门的 3D 形状。

IfcDoorLiningProperties 包含在 IfcDoorType.HasPropertySets 的属性列表中。有关门框的更多信息可以通过使用另一个 IfcPropertySet 进行动态扩展，包含在 IfcDoorType 的同一列表中。

IfcDoorLiningProperties 不持有自身的几何表示。然而，它定义了可用于创建门类型形状的参数（该形状由 IfcDoor 插入到项目的空间上下文中），如下所述。IfcDoorLiningProperties 的参数定义了一个标准门框，包括（如果提供）门槛和横梁。门框的外部边界由分配给 IfcDoor 的 'Profile' 形状表示确定，该 IfcDoor 插入了 IfcDoorType。

门框应用于洞口侧壁的左侧、右侧和上侧。参数包括：

  • LiningDepth，如果省略，则等于墙厚 - 这仅在提供了 LiningThickness 值时生效。如果两个参数都未提供，则没有门框。
  • LiningThickness
  • LiningToPanelOffsetX
  • LiningToPanelOffsetY
注释 参数 LiningToPanelOffsetX 和 LiningToPanelOffsetY 在 IFC4 中添加。

门框可以仅覆盖洞口侧壁的一部分。

  • LiningOffset，如果门框边缘相对于局部放置的 x 轴有偏移，则提供此参数。
注释 除了 LiningOffset 之外，IfcDoor 的局部放置可能已经相对于墙边有偏移，从而沿 y 轴移动门框。门框的实际位置是从局部放置的原点沿正 y 轴方向，通过 LiningOffset 给出的距离计算得出。

门框可以包括一个门套，该门套覆盖洞口周围墙面的部分区域。门套覆盖墙两侧门框的左侧、右侧和上侧。参数包括：

  • CasingDepth
  • CasingThickness

门框可以包括一个门槛，该门槛覆盖洞口的底部。参数包括：

  • ThresholdDepth，如果省略，则等于墙厚 - 这仅在提供了 ThresholdThickness 值时生效。如果两个参数都未提供，则没有门槛。
  • ThresholdThickness
  • ThresholdOffset（图中未显示），如果门槛边缘相对于局部放置的 x 轴有偏移，则提供此参数。

门框可以有一个横梁，将门扇与窗扇分开。如果提供横梁，则通过以下参数定义：

  • TransomOffset，一条平行于局部放置 x 轴的边
  • TransomThickness

横梁的深度与门框的深度相同，不作为单独参数提供。

注释 LiningDepth 描述了门框沿洞口侧壁的长度。如果门框根据门样式具有特定深度，则可以给出绝对值。然而，它通常等于墙厚。如果使用相同的门样式（如同类型的单开门），但插入到不同厚度的墙中，则需要为每种墙厚创建特殊的门样式。因此，一些 CAD 系统允许将该值设置为"自动对齐"墙厚。这应通过不分配可选属性 LiningDepth 来实现。相同的约定适用于 ThresholdDepth。
沿革 IFC2.0 中的新实体。在 IFC2x 中从 IfcDoorLining 重命名。

IFC4 变更：添加了以下属性 LiningToPanelOffsetX、LiningToPanelOffsetY。属性 ShapeAspectStyle 已弃用，不应再使用。超类型更改为新的 IfcPreDefinedPropertySet。

IFC4.3.2.0 弃用：此实体以及 IfcPreDefinedPropertySet 的大多数其他子类型现已弃用。请改用 Pset_DoorLiningProperties。

7.1.3.1.6 形式化表示（Formal representation） 

ENTITY IfcDoorLiningProperties
 SUBTYPE OF (IfcPreDefinedPropertySet);
	LiningDepth : OPTIONAL IfcPositiveLengthMeasure;
	LiningThickness : OPTIONAL IfcNonNegativeLengthMeasure;
	ThresholdDepth : OPTIONAL IfcPositiveLengthMeasure;
	ThresholdThickness : OPTIONAL IfcNonNegativeLengthMeasure;
	TransomThickness : OPTIONAL IfcNonNegativeLengthMeasure;
	TransomOffset : OPTIONAL IfcLengthMeasure;
	LiningOffset : OPTIONAL IfcLengthMeasure;
	ThresholdOffset : OPTIONAL IfcLengthMeasure;
	CasingThickness : OPTIONAL IfcPositiveLengthMeasure;
	CasingDepth : OPTIONAL IfcPositiveLengthMeasure;
	ShapeAspectStyle : OPTIONAL IfcShapeAspect;
	LiningToPanelOffsetX : OPTIONAL IfcLengthMeasure;
	LiningToPanelOffsetY : OPTIONAL IfcLengthMeasure;
 WHERE
	WR31 : NOT(EXISTS(LiningDepth) AND NOT(EXISTS(LiningThickness)));
	WR32 : NOT(EXISTS(ThresholdDepth) AND NOT(EXISTS(ThresholdThickness)));
	WR33 : (EXISTS(TransomOffset) AND EXISTS(TransomThickness)) XOR
(NOT(EXISTS(TransomOffset)) AND NOT(EXISTS(TransomThickness)));
	WR34 : (EXISTS(CasingDepth) AND EXISTS(CasingThickness)) XOR
(NOT(EXISTS(CasingDepth)) AND NOT(EXISTS(CasingThickness)));
	WR35 : (EXISTS(SELF\IfcPropertySetDefinition.DefinesType[1])) 
AND 
('IFC4X3_DEV_62887e3f.IFCDOORTYPE' IN TYPEOF(SELF\IfcPropertySetDefinition.DefinesType[1]));
END_ENTITY;
