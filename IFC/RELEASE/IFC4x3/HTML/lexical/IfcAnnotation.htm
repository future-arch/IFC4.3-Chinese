5.4.3.6 IfcAnnotation（像片调绘）

注释 该实体是 IfcProduct 或 IfcTypeProduct 的子类型，因此是每个标准化模式子集和实现级别的一部分。

5.4.3.6.1 语义定义（Semantic definition）

像片调绘是在项目几何（和空间）上下文中的信息元素，它为构成项目模型的对象添加注释或含义。像片调绘包括附加点、曲线、文本、尺寸标注、填充图案和其他形式的图形注释。它还包括附加模型组件的虚拟或符号表示，这些表示不表示产品或空间结构，例如测量点和线、等高线或类似内容。

注释 与特定产品表示直接相关的附加表示信息（通常是二维的），例如标签编号或填充图案，包含在具有各种 IfcElement（及其子类型）的 IfcShapeRepresentation 的 IfcProductDefinitionShape 中。只有那些无法直接与单个产品关联的表示信息，才需要包装在 IfcAnnotation 中。

如果可用，像片调绘应通过将像片调绘包含在建筑结构的适当级别（场地、设施、设施部分或建筑、楼层或空间）中，与项目的空间上下文相关联。这是通过 IfcRelContainedInSpatialStructure 关系处理的。

IfcAnnotationTypeEnum 中的预定义类型值建议了一些可以使用的像片调绘类型。

沿革 IFC2x2 中的新实体。

5.4.3.6.6 形式化表示（Formal representation） 

ENTITY IfcAnnotation
 SUBTYPE OF (IfcProduct);
	PredefinedType : OPTIONAL IfcAnnotationTypeEnum;
 INVERSE
	ContainedInStructure : SET [0:1] OF IfcRelContainedInSpatialStructure FOR RelatedElements;
END_ENTITY;
