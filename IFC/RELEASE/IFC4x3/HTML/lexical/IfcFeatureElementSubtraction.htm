5.4.3.30 IfcFeatureElementSubtraction（特征构件减除）

注释 该实体是 IfcProduct 或 IfcTypeProduct 的子类型，因此是每个标准化模式子集和实现级别的一部分。

抽象 该定义可能不会被直接实例化。

5.4.3.30.1 语义定义（Semantic definition）

IfcFeatureElementSubtraction 是通用特征构件的特化，表示一个存在依赖的构件，用于修改关联主构件的形状和外观。IfcFeatureElementSubtraction 提供了将形状修改器作为语义对象在IFC对象模型中处理的能⼒，这些修改器会从主构件的形状中减去。

注释 单个减除特征（如子类型 IfcOpeningElement）通过单个减除关系 IfcRelVoidsElement 分配给 IfcElement 的一个实例。它在开口和构件之间建立了1:1的关系。一个构件可以具有多个 IfcRelVoidsElement 关系，从而允许存在多个空腔。

主构件和减除特征之间的空腔关系通过布尔差运算在几何上解决。

IfcFeatureElementSubtraction 的局部放置在其超类型 IfcProduct 中定义。它由 IfcLocalPlacement 定义，该定义引用了所有几何表示所参照的局部坐标系。局部放置始终相对于特征构件减除所从属的建筑构件的局部放置来定义：

  • IfcObjectPlacement 的 PlacementRelTo 属性应指向（如果提供）IfcElement 的对象放置，该 IfcElement 用于 VoidsElements.RelatingElement 逆向属性（该特征的父构件）。
沿革 IFC2x2 中的新实体。
IFC2x2 变更 该实体作为IFC2x方案向上兼容的扩展引入。它是一个中间抽象超类型，没有定义自身的显式属性。

5.4.3.30.6 概念用法（Concept usage）

5.4.3.30.7 形式化表示（Formal representation） 

ENTITY IfcFeatureElementSubtraction
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcEarthworksCut
	,IfcOpeningElement
	,IfcVoidingFeature))
 SUBTYPE OF (IfcFeatureElement);
 INVERSE
	VoidsElements : IfcRelVoidsElement FOR RelatedOpeningElement;
 WHERE
	HasNoSubtraction : SIZEOF(SELF\IfcElement.HasOpenings) = 0;
	IsNotFilling : SIZEOF(SELF\IfcElement.FillsVoids) = 0;
END_ENTITY;
