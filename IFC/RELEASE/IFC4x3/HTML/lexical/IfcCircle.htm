8.9.3.18 IfcCircle（度盘）

AbV ⓘ

8.9.3.18.1 语义定义（Semantic definition）

IfcCircle 是由一组与中心点距离相等的点构成的曲线。

注释 圆弧段可通过使用 IfcTrimmedCurve 定义，其中 BasisCurve 为 IfcCircle。

图1展示了在构件对象坐标系中三维位置坐标系内 IfcCircle 的定义。

参考依据 ISO/CD 10303-42:1992

度盘通过半径及其位置和方向定义。数据应按下述方式解释：

C = SELF\IfcConic.Position.Location
x = SELF\IfcConic.Position.P[1]
y = SELF\IfcConic.Position.P[2]
z = SELF\IfcConic.Position.P[3]
R = Radius

度盘参数化表示为：

$$ \lambda(u) = C + R(\cos(u)x + \sin(u)y) $$

参数化范围为 0 ≤ u ≤ 2π（0 ≤ u ≤ 360 度）。 在上述定义的定位坐标系中，度盘满足方程 C = 0，其中：

$$ C(x,y,z) = x^2 + y^2 - R^2 $$

度盘上任意点的正方向为曲线在该点处的切线方向 T，其中：

$$ T = (-C_y,C_x,0) $$

沿革 IFC1.0 中新增实体
