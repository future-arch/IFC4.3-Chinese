8.8.3.41 IfcSurfaceCurveSweptAreaSolid（曲面曲线扫掠面积实体）

注释 该实体不是标准化模式子集或实现级别的一部分。

8.8.3.41.1 语义定义（Semantic definition）

IfcSurfaceCurveSweptAreaSolid 是通过沿着位于参考曲面上的准线扫掠一个面积而得到的结果。扫掠面积由 IfcProfileDef 的子类型提供。轮廓通过隐式笛卡尔变换运算符放置在扫掠起点处，在该点处轮廓法线与准线切线一致，轮廓的 x 轴与曲面法线一致。沿着准线的任意点，扫掠轮廓的原点位于准线上，轮廓的法线指向准线切线方向，轮廓的 x 轴与该点的曲面法线相同。

注释 轮廓面积的法线必须与准线在任意给定点的切线相同。如果准线在起点处有直线段，则该线段必须垂直于起点处的轮廓。

Directrix 和 ReferenceSurface 位于对象坐标系内。扫掠操作开始于 StartParam，参数值基于曲线参数化提供。如果未提供 StartParam，则默认从准线起点开始。扫掠操作结束于 EndParam，参数值基于曲线参数化提供。如果未提供 EndParam，则默认到准线终点结束。实体的几何形状不依赖于曲线参数化；体积取决于扫掠面积和 Directrix 的长度。

注释 StartParam 和 EndParam 默认不进行归一化，它们依赖于曲线的参数化。然而，在 IfcCompositeCurve 中使用 IfcReparametrisedCompositeCurveSegment 作为准线允许显式地对基础扫掠曲线进行重新参数化。如果准线是闭合曲线，例如 IfcCircle 或 IfcEllipse，则 StartParam 和 EndParam 不应超过参数范围，即不应 > 360°。
示例 参考曲面是位于三维空间中并定位于对象坐标系内的任意曲面（平面、圆柱面、复合曲面）。在大多数情况下，它是一个平面或拉伸曲面。准线位于该曲面上，对于平面，它可能被定义为多段线或复合曲线；对于圆柱面或其他非平面参考曲面，它通常可能被定义为该参考曲面上的 p 曲线。

在准线的任意点处，可以构造一个平面。隐式平面位置坐标系的原点位于准线上。位置坐标系的 Axis3（z 轴，或法线）与该点处准线的切线相同，位置坐标系的 Axis1（x 轴，或 u）与该点处参考曲面的法线相同。Axis2（y 轴，或 v）由此构造而成。

如果省略了继承的 Position 属性，则扫掠实体的结果体不会重新定位。否则，将使用 Position 属性建立的坐标系来相对于对象坐标系重新定位该体。

示例 图 8.8.3.41.A 展示了一个使用圆柱参考曲面和 p 曲线扫掠矩形的例子。未提供 Position，因此它不会重新定位结果扫掠实体。图 8.8.3.41.B 显示了预期结果。
注释 如图所示，准线的起点位于对象坐标系的原点，
这仅是巧合。准线的起点以及由此扫掠操作的起点可能位于对象坐标系内的任意点，并且仅取决于准线的位置。
注释 根据 ISO/CD 10303-42:1992 的定义 曲面曲线扫掠面积实体是一种扫掠面积实体类型，它是沿着位于 ReferenceSurface 上的 Directrix 扫掠一个面的结果。SweptArea 的方向与曲面法线的方向相关。
注释 实体改编自 ISO 10303-42 中定义的 **surface_curve_swept_area_solid**。
沿革 IFC2x2 中的新实体。

SweptArea 必须是一个位于平面 z = 0 内的曲线边界曲面，并且它沿着 Directrix 扫掠，使得用于定义 SweptArea 的局部坐标系的原点位于 Directrix 上，并且局部 x 轴指向当前点处 ReferenceSurface 的法线方向。结果实体具有这样的属性：在任意点处垂直于 Directrix 的法平面与曲面的交线是 SweptArea 的副本。

SweptArea 沿着 Directrix 扫掠时的方向由具有以下属性的笛卡尔变换运算符 3D 精确定义： > * 局部原点作为点 (0., 0., 0)， * 轴 1 作为在参数为 u 的准线点处参考曲面的法线 N。 * 轴 3 作为在参数为 u 的准线点处切向量 t 的方向。其余属性默认为定义相应的变换矩阵 T(u)，该矩阵随准线参数 u 变化。

非正式命题

  1. SweptArea 必须位于隐式平面 z = 0 内。
  2. Directrix 必须位于 ReferenceSurface 上。
