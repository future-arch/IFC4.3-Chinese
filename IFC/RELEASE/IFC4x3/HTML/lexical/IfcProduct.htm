5.1.3.10 IfcProduct

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.1.3.10.1 语义定义（Semantic definition）

IfcProduct 是与几何或空间上下文相关的任何对象的抽象表示。如果分配了几何表示，则_IfcProduct_出现在空间的特定位置。它可以相对于其他产品放置，但最终相对于项目坐标系。_ObjectPlacement_属性建立了坐标系，在该坐标系中，_Representation_下的几何表示项使用的所有点和方向都基于此。_Representation_由_IfcProductDefinitionShape_提供，它可以是几何形状表示，也可以是拓扑表示（具有或不具有拓扑项的基础几何）。

产品包括制造、供应或创建的对象（称为元素），用于纳入AEC/FM项目。这还包括由其他产品间接创建的对象，如由边界元素定义的空间。产品可以指定为永久使用或临时使用，后者的一个例子是模板。产品由其属性和表示定义。

除了物理产品（由子类型_IfcElement_涵盖）和空间项（由子类型_IfcSpatialElement_涵盖）之外，_IfcProduct_还包括与几何或空间上下文相关的非物理项，如网格、端口、注释、结构作用等。

IfcProduct_的任何实例定义了产品的特定出现。与许多类似（或相同）的_IfcProduct_出现相关的通用类型信息由_IfcTypeProduct（及其子类型）处理，通过使用对象化关系_IfcRelDefinesByType_分配给一个或多个_IfcProduct_出现。除了通用属性外，_IfcTypeProduct_还可以为所有出现提供通用几何表示。

注意 参见_IfcTypeProduct_了解如何使用通用几何表示，以及_IfcRelDefinesByType_了解如何使用和覆盖通用属性。

在通用级别上，可以使用引用相应对象的关系对象将产品分配给流程、控制、资源、项目：

  • 应用控制：使用_IfcRelAssignsToControl_将_IfcProduct_链接到_IfcControl_进行分配
  • 分配给流程：使用_IfcRelAssignsToProcess_将_IfcProduct_链接到_IfcProcess_进行分配
  • 分配给资源：使用_IfcRelAssignsToResource_将_IfcProduct_链接到_IfcResource_进行分配
示例 控制关系的一个例子是将性能历史分配给分布元素。流程分配关系的一个例子是将墙、楼板和柱等产品分配给施工任务以进行施工规划。资源分配关系的一个例子是将产品分配给消耗产品的施工资源。

5.1.3.10.6 概念用法（Concept usage）

5.1.3.10.7 形式化表示（Formal representation） 

ENTITY IfcProduct
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcAnnotation
	,IfcElement
	,IfcLinearElement
	,IfcPort
	,IfcPositioningElement
	,IfcSpatialElement
	,IfcStructuralActivity
	,IfcStructuralItem))
 SUBTYPE OF (IfcObject);
	ObjectPlacement : OPTIONAL IfcObjectPlacement;
	Representation : OPTIONAL IfcProductRepresentation;
 INVERSE
	ReferencedBy : SET [0:?] OF IfcRelAssignsToProduct FOR RelatingProduct;
	PositionedRelativeTo : SET [0:?] OF IfcRelPositions FOR RelatedProducts;
	ReferencedInStructures : SET [0:?] OF IfcRelReferencedInSpatialStructure FOR RelatedElements;
 WHERE
	PlacementForShapeRepresentation : (EXISTS(Representation) AND EXISTS(ObjectPlacement))
            OR (EXISTS(Representation) AND 
      (SIZEOF(QUERY(temp <* Representation.Representations | 'IFC4X3_DEV_62887e3f.IFCSHAPEREPRESENTATION' IN TYPEOF(temp))) = 0))
            OR (NOT(EXISTS(Representation)));
END_ENTITY;
