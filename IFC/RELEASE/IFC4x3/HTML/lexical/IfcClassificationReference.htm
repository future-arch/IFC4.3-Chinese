8.6.3.2 IfcClassificationReference

RV ⓘ
AbV ⓘ

8.6.3.2.1 语义定义（Semantic definition）

IfcClassificationReference 是对分类系统或源（参见 IfcClassification）中特定分类键（或符号）的引用。

继承的属性具有以下含义：

  • Identification：包含为特定分类项（或表）的引用提供的键。
  • Name：允许人类可解释地指定分类符号。
  • Location：可选地包含指向分类系统（或源）的直接 URI 链接，用于对分类键进行超链接。

IfcClassificationReference 可以直接分配给 IfcClassification，例如，如果没有要包含的分类层次结构；或者，如果完整分类层次结构包含在数据集中，则可以引用父分类符号。前者称为“轻量级分类”，后者称为“完整分类”。ReferencedSource 属性然后包含以下信息（通过 IfcClassificationReferenceSelect 选择）：

  1. 类型为 IfcClassification：直接引用分类系统（提供元信息）；
  2. 类型为 IfcClassificationReference：在分类层次结构中引用父分类符号。
示例 The IfcClassificationReference can be used as a form of 'lightweight' classification through the 'Identification' attribute inherited from the abstract IfcExternalReference class. In this case, the 'Identification' could take (for instance) the Uniclass notation "L6814" which, if the classification was well understood by all parties and was known to be taken from a particular classification source, would be sufficient. The Name attribute could be the title "Tanking". This would remove the need for the overhead of the more complete classification structure of the model.
沿革 New entity in IFC2x.
IFC4 变更 The attribute Description and inverse attribute HasReferences are added. The attribute Identification has been renamed from ItemReference.

8.6.3.2.3 特性（Attributes）

8.6.3.2.4 形式化表示（Formal representation） 

ENTITY IfcClassificationReference
 SUBTYPE OF (IfcExternalReference);
	ReferencedSource : OPTIONAL IfcClassificationReferenceSelect;
	Description : OPTIONAL IfcText;
	Sort : OPTIONAL IfcIdentifier;
 INVERSE
	ClassificationRefForObjects : SET [0:?] OF IfcRelAssociatesClassification FOR RelatingClassification;
	HasReferences : SET [0:?] OF IfcClassificationReference FOR ReferencedSource;
END_ENTITY;
