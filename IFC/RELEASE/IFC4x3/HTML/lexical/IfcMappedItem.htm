8.9.3.38 IfcMappedItem (映射项)

RV ⓘ
AbV ⓘ

8.9.3.38.1 语义定义（Semantic definition）

IfcMappedItem 是源定义（用于与块/共享单元/宏定义进行比较）的插入实例。该实例通过应用笛卡尔变换算子作为 MappingTarget 来插入。

注释 A mapped item is a subtype of representation item. It enables a representation to be used as a representation item in one or more other representations. The mapped item allows for the definition of a representation using other representations.
示例 An IfcMappedItem can reuse other mapped items (ako nested blocks), doing so the IfcRepresentationMap is based on an IfcShapeRepresentation including one or more IfcMappedItem's.
注释 Definition according to ISO/CD 10303-43:1992 A mapped item is a type of representation item that specifies the mapping of a representation as an element of the items of a second representation.
注释 Entity adapted from mapped_item defined in ISO 10303-43.
沿革 New entity in IFC2x.

非形式命题

  1. 映射项不得通过参与被映射表示的定义而自包含。
  2. 如果映射源是几何表示项，则映射源和映射目标的维度必须相同。

8.9.3.38.4 形式化表示（Formal representation） 

ENTITY IfcMappedItem
 SUBTYPE OF (IfcRepresentationItem);
	MappingSource : IfcRepresentationMap;
	MappingTarget : IfcCartesianTransformationOperator;
END_ENTITY;
