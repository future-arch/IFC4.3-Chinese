6.1.3.16 IfcDoor

注释 该实体是 IfcProduct 或 IfcTypeProduct 的子类型，因此是每个标准化模式子集和实现级别的一部分。

6.1.3.16.1 语义定义（Semantic definition）

门是一种建筑构件，主要用于为人员、货物、动物和车辆提供受控通道。它包括铰链式、枢轴式、滑动式以及此外的旋转式和折叠式操作的构造。门可以：

  • 是“独立式”门，包含在 IfcSpatialElement（如 IfcBuildingStorey）中。
  • 填充一个开口，通常在墙体中。然后门将具有 FillsVoids 属性，该属性使用 IfcRelFillsElement 关系将 IfcDoor 与 IfcOpeningElement 相关联；
  • 是构件组件的一部分，通常是 IfcCurtainWall。然后门将具有 Decomposes 属性，该属性使用 IfcRelAggregates 关系将门与构件组件相关联；

门实例主要有两种表示方式：

  • 具有三维矩形“Profile”形状表示的 IfcDoor 实体。然后可以使用此截面参数化生成门的几何形状。如果未提供，则门填充开口时可以使用 IfcOpeningElement 的截面。参数在相关的 IfcDoorType 上指定，该类型为每个门扇引用 IfcDoorLiningProperties 和 IfcDoorPanelProperties；
  • 不是参数化生成的，并且只有“Brep”或“SurfaceModel”几何形状的 IfcDoor 实体。

此外，IfcDoor 通常可以包含定义门二维形状及其摆动的“FootPrint”表示。

  • 门的宽度和高度
  • 门的开启方向（通过 ObjectPlacement 的正 y 轴确定）

IfcDoorType 指定了所有 IfcDoor 实例共有的参数：

  • 操作类型（单扇、双扇、旋转式等）
  • 门合页侧（通过右开门和左开门的两种不同样式）
  • 通过 IfcDoorLiningProperties 定义的门框的特定属性
  • 通过 IfcDoorPanelProperties 定义的门扇的特定属性
参考 定义根据 ISO 6707-1：用于关闭开口的构造，主要用于铰链式、枢轴式或滑动式操作。
参考 Definition according to ISO 6707-1: construction for closing an opening, intended primarily for access with hinged, pivoted or sliding operation.
注释 The entity IfcDoorStandardCase has been deleted. Use an IfcDoor with a 'Profile' representation instead. The IfcDoor should also have an IfcDoorType with ParameterTakesPrecedence set to 'TRUE'.
IFC4 变更 添加了属性 PredefinedType 和 OperationType，适用的类型对象已更改为 IfcDoorType。

6.1.3.16.7 形式化表示（Formal representation） 

ENTITY IfcDoor
 SUBTYPE OF (IfcBuiltElement);
	OverallHeight : OPTIONAL IfcPositiveLengthMeasure;
	OverallWidth : OPTIONAL IfcPositiveLengthMeasure;
	PredefinedType : OPTIONAL IfcDoorTypeEnum;
	OperationType : OPTIONAL IfcDoorTypeOperationEnum;
	UserDefinedOperationType : OPTIONAL IfcLabel;
 WHERE
	CorrectPredefinedType : NOT(EXISTS(PredefinedType)) OR
 (PredefinedType <> IfcDoorTypeEnum.USERDEFINED) OR
 ((PredefinedType = IfcDoorTypeEnum.USERDEFINED) AND EXISTS (SELF\IfcObject.ObjectType));
	CorrectTypeAssigned : (SIZEOF(IsTypedBy) = 0) OR
  ('IFC4X3_DEV_62887e3f.IFCDOORTYPE' IN TYPEOF(SELF\IfcObject.IsTypedBy[1].RelatingType));
END_ENTITY;
