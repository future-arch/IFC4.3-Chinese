8.7.3.12 IfcGridPlacement

注释 该实体不是标准化模式子集或实现级别的一部分。

8.7.3.12.1 语义定义（Semantic definition）

IFC4.3.0.0 更改：在 IFC 4.3 中，PlacementRelTo 属性已从 IfcLocalPlacement 移至其超类型 IfcObjectPlacement，后者也是此实体的超类型。这意味着，为了正确的全局定位，IfcGridPlacement 将通过 IfcObjectPlacement.PlacementRelTo 参考 (a) IfcGrid 的 ObjectPlacement，并通过 IfcVirtualGridIntersection 参考 (b) 包含在同一网格中的 IfcGridAxis 对。

IfcGridPlacement 提供了 IfcObjectPlacement 的特化，其中对象坐标系的放置和轴方向由对 IfcGrid 中定义的设计网格的引用定义。

对象坐标系的位置由 PlacementLocation 属性给出。它被定义为 IfcVirtualGridIntersection，即两个网格轴的交点，带有可选的偏移量。

x 轴的轴方向给出如下：

  • PlacementRefDirection = NIL：由虚拟交点处第一条网格轴的切线（如果提供了 PlacementLocation.OffsetDistances，则可能使用偏移曲线）给出；
  • PlacementRefDirection = IfcDirection：由明确提供的方向信息给出；
  • PlacementRefDirection = IfcVirtualGridIntersection：由 PlacementLocation 的虚拟网格交点和 PlacementRefDirection 的虚拟网格交点之间的切线给出。必须考虑 PlacementLocation 和 PlacementRefDirection 的 IfcVirtualGridIntersection 中可能提供的偏移量。

IfcGridPlacement 的 y 轴方向是 x 轴的正交补。由 x 轴和 y 轴定义的平面应与 IfcGrid 的局部放置的 xy 平面共面。

z 轴的方向是 x 轴和 y 轴叉积的方向，即 IfcGridPlacement 的 z 轴应与 IfcGrid 的局部放置的 z 轴共线。

注释 PlacementLocation.OffsetDistances[3] 和 PlacementRefDirection.OffsetDistances[3] 应不赋值或具有相同的 z 偏移值。

以下图/显示了 IfcGridPlacement 的放置位置和方向的使用。

图 8.7.3.12.A 说明了未提供 PlacementRefDirection 的情况 - 对象坐标系由以下项定义：

  • 其位置：由 PlacementLocation 的虚拟网格交点给出
  • 其 x 轴方向：由虚拟网格交点处第一条相交轴的切线给出

图 8.7.3.12.B 说明了 PlacementRefDirection 被提供为 IfcDirection 的情况 - 对象坐标系由以下项定义：

  • 其位置：由 PlacementLocation 的虚拟网格交点给出
  • 其 x 轴方向：由 IfcDirection 的 DirectionRatios 给出，仅考虑 x 和 y 的比例

图 8.7.3.12.C 说明了 PlacementRefDirection 被提供为 IfcVirtualGridIntersection 的情况 - 对象坐标系由以下项定义：

  • 其位置：由 PlacementLocation 的虚拟网格交点给出
  • 其 x 轴方向：由 PlacementLocation 的虚拟网格交点和 PlacementRefDirection 的虚拟网格交点之间的直线的切线给出。
沿革 IFC1.5 中的新实体。实体名称在 IFC2x 中从 IfcConstrainedPlacement 更改。

IFC4 变更：PlacementRefDirection 的属性数据类型已更改为 IfcGridPlacementDirectionSelect。

8.7.3.12.3 特性（Attributes）

8.7.3.12.4 形式化表示（Formal representation） 

ENTITY IfcGridPlacement
 SUBTYPE OF (IfcObjectPlacement);
	PlacementLocation : IfcVirtualGridIntersection;
	PlacementRefDirection : OPTIONAL IfcGridPlacementDirectionSelect;
END_ENTITY;
