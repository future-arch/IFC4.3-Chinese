8.11.2.83 IfcSIUnitName

8.11.2.83.1 语义定义（Semantic definition）

注释 Definition according to ISO/CD 10303-41:1992 An SI unit name is the name of an SI unit. The definitions of the names of SI units are specified in ISO 1000 (clause 2).
注释 Type adapted from si_unit_name defined in ISO 10303-41.
沿革 New entity in IFC1.5.1.
