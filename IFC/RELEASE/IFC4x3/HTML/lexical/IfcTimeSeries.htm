8.5.3.12 IfcTimeSeries

注释 该实体不是标准化模式子集或实现级别的一部分。

抽象 该定义可能不会被直接实例化。

8.5.3.12.1 语义定义（Semantic definition）

时间序列是一组带时间戳的数据条目。它允许自然地关联在时间间隔内收集的数据。时间序列可以是规律的或不规律的。在规律时间序列中，数据以可预测的方式在预定义的间隔到达。在不规律时间序列中，一些或所有时间戳不遵循重复模式，并且不可预测的数据突发可能在未指定的时间点到达。

建筑物及其性能的建模涉及在一段时间内生成和记录的数据。这些数据涵盖了广泛的范围，从天气数据到各种时间表，再到状态测量和报告，以及所有其他与时间相关的方面。它们在时间上的正确放置对于它们的正确理解和使用至关重要，而 IfcTimeSeries 子类型提供了适当的数据结构来适应这些类型的数据。

沿革 新实体在 IFC2x2 中引入。

8.5.3.12.3 特性（Attributes）

8.5.3.12.4 形式化表示（Formal representation） 

ENTITY IfcTimeSeries
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcIrregularTimeSeries
	,IfcRegularTimeSeries));
	Name : IfcLabel;
	Description : OPTIONAL IfcText;
	StartTime : IfcDateTime;
	EndTime : IfcDateTime;
	TimeSeriesDataType : IfcTimeSeriesDataTypeEnum;
	DataOrigin : IfcDataOriginEnum;
	UserDefinedDataOrigin : OPTIONAL IfcLabel;
	Unit : OPTIONAL IfcUnit;
 INVERSE
	HasExternalReference : SET [1:?] OF IfcExternalReferenceRelationship FOR RelatedResourceObjects;
END_ENTITY;
