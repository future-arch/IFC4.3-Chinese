5.4.3.21 IfcElementQuantity（构件数量）

RV ⓘ
AbV ⓘ

5.4.3.21.1 语义定义（Semantic definition）

IfcElementQuantity 定义了构件物理属性的派生度量集合。构件可以是空间结构元素（如建筑、楼层或空间）或建筑元素（如墙体、楼板、面层）。通过使用 IfcRelDefinesByProperties 关系将 IfcElementQuantity 分配给构件。

可选的 MethodOfMeasurement 属性定义了用于计算构件数量的测量方法代码（例如来自标准测量方法）。

注释 名称和测量方法属性的可识别值必须在进一步协议文件（如实施者协议）中达成一致。部分协议可能仅适用于特定区域的测量方法。

在单个 Quantities 中给出的名称属性提供了构件数量的可识别语义含义。需要这两类信息才能为度量值建立精确含义。可为每个 Quantities 分配可选描述。通过单个 IfcElementQuantity 实例分配的所有数量均被视为根据相同测量方法生成。但允许将多个 IfcElementQuantity 实例分配给一个构件，从而使构件能够具有根据不同测量方法生成的数量。

示例 为在德国地区交换空间的净地板面积（作为 IfcSpace），名称可为"Netto-Grundfläche"（净地板面积），测量方法相应为"DIN277-2"（德国工业标准第277号第2版）。
示例 同一 IfcSpace 实例在德国地区根据住房法规可能具有不同的面积度量，名称为"Wohnfläche"，测量方法为"2.BV"。它将通过单独的 IfcRelDefinesByProperties 关系附加到 IfcSpace。

IfcElementQuantity 在其 Quantities 集合中可以包含以下 IfcPhysicalQuantity 子类型，这些子类型构成使用的基本度量类型：

  • 数量度量
  • 重量度量
  • 长度度量
  • 面积度量
  • 体积度量
  • 时间度量

基本数量是独立于特定测量方法的数量定义，因此具有国际适用性。基本数量定义为毛值和净值，并通过测量构件的正确几何形状表示来提供。本规范包含一组基本数量定义。有关适用的基本数量，请参见 IfcElement 的每个子类型。

以下通用协议适用于每个基本数量集：

  • IfcElementQuantity.Name = <数量定义模板的名称>
  • IfcElementQuantity.MethodOfMeasurement = 'BaseQuantities'
  • IfcElementQuantity.Quantities = IfcPhysicalSimpleQuantity 子类型的集合，其 Name 属性值按本规范发布。

5.4.3.21.6 形式化表示（Formal representation） 

ENTITY IfcElementQuantity
 SUBTYPE OF (IfcQuantitySet);
	MethodOfMeasurement : OPTIONAL IfcLabel;
	Quantities : SET [1:?] OF IfcPhysicalQuantity;
 WHERE
	UniqueQuantityNames : IfcUniqueQuantityNames(Quantities);
END_ENTITY;
