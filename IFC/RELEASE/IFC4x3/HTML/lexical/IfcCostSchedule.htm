6.5.3.3 IfcCostSchedule（成本计划）

注释 该实体不是标准化模式子集或实现级别的一部分。

6.5.3.3.1 语义定义（Semantic definition）

IfcCostSchedule 汇集了 IfcCostItem 的实例，其目的可以是识别纯粹的成本信息（如建筑工程成本估算），也可以是将成本信息包含在另一种呈现形式中（如工作订单）。

沿革 IFC2.0 中的新实体。
IFC4 变更 属性 ID 重命名为 Identification 并提升至超类型 IfcControl，PredefinedType 变为可选属性，属性 PreparedBy、SubmittedBy、TargetUsers 已被移除。
