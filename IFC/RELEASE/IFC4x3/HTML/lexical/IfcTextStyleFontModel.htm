8.12.3.37 IfcTextStyleFontModel

RV ⓘ
AbV ⓘ

8.12.3.37.1 语义定义（Semantic definition）

Definition according to W3C for Cascading Style Sheets: Setting font properties will be among the most common uses of style sheets. Unfortunately, there exists no well-defined and universally accepted taxonomy for classifying fonts, and terms that apply to one font family may not be appropriate for others. For example, 'italic' is commonly used to label slanted text, but slanted text may also be labeled as being Oblique, Slanted, Incline, Cursive or Kursiv. Therefore it is not a simple problem to map typical font selection properties to a specific font.

Font matching Because there is no accepted, universal taxonomy of font properties, matching of properties to font faces must be done carefully. The properties are matched in a well-defined order to ensure that the results of this matching process are as consistent as possible across user agents (assuming that the same library of font faces is presented to each of them).

  1. 用户代理（UA）创建（或访问）一个数据库，其中包含UA所知的ョ所有字体的相关CSS1属性。UA可能因为字体已本地安装或已通过网络下载而知道某个字体。如果存在两个具有完全相同属性的字体，则其中一个将被忽略。
  2. 在给定的元素和该元素中的每个字符处，UA会组合适用于该元素的字体属性。使用完整的属性集，UA使用'font-family'属性选择一个暂定的字体系列。其余属性根据每个属性描述的匹配标准进行测试。如果所有剩余属性都匹配，则该字体字面就是给定元素的匹配字体字面。
  3. 如果在步骤2处理的字体系列中没有匹配的字体字面，并且字体集中有下一个备选的'font-family'，则使用下一个备选的'font-family'重复步骤2。
  4. 如果存在匹配的字体字面，但它不包含当前字符的字形，并且字体集中有下一个备选的'font-family'，则使用下一个备选的'font-family'重复步骤2。
  5. 如果在步骤2选择的字体系列中没有字体，则使用UA依赖的默认'font-family'并重复步骤2，使用默认字体中能获得的最佳匹配。

（上述算法可以优化，以避免为每个字符重复检查CSS1属性。）

上述（2）中的每个属性的匹配规则如下：

  1. 首先尝试'font-style'。'italic'将被满足，如果UA的字体数据库中有标记为CSS关键字'italic'（首选）或'oblique'的字面。否则，值必须精确匹配，否则font-style将失败。
  2. 接下来尝试'font-variant'。'normal'匹配未标记为'small-caps'的字体；'small-caps'匹配（1）标记为'small-caps'的字体，（2）合成小写字体的字体，或（3）将所有小写字母替换为大写字母的字体。可以通过电子缩放正常字体的字母来合成小写字体。
  3. 接下来匹配'font-weight'，它永远不会失败。（参见下面的'font-weight'。）
  4. 'font-size'必须在UA依赖的容差范围内进行匹配。（通常，可伸缩字体的尺寸会四舍五入到最接近的整数像素，而点阵字体的容差可能高达20%。）进一步的计算基于使用的'font-size'值，而不是指定的'font-size'值。

继承的 Name 属性用于定义字体名称，特别是在未提供（列表）字体系列的情况下。

注释 Corresponding CSS1 definitions are Font properties ('font-family', 'font-style', 'font-variant', 'font-weight').

8.12.3.37.5 形式化表示（Formal representation） 

ENTITY IfcTextStyleFontModel
 SUBTYPE OF (IfcPreDefinedTextFont);
	FontFamily : LIST [1:?] OF IfcTextFontName;
	FontStyle : OPTIONAL IfcFontStyle;
	FontVariant : OPTIONAL IfcFontVariant;
	FontWeight : OPTIONAL IfcFontWeight;
	FontSize : IfcSizeSelect;
 WHERE
	MeasureOfFontSize : ('IFC4X3_DEV_62887e3f.IFCLENGTHMEASURE' IN TYPEOF(SELF.FontSize)) AND
(SELF.FontSize > 0.);
END_ENTITY;
