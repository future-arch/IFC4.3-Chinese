5.1.3.44 IfcRoot（根）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.1.3.44.1 语义定义（Semantic definition）

IfcRoot 是内核或IFC规范后续层中所有实体定义的最抽象根类。因此，它是除IFC资源模式中定义的实体外所有IFC实体的共同超类型。所有作为_IfcRoot_子类型的实体均可独立使用，而非_IfcRoot_子类型的资源模式实体则不应作为独立实体存在。

注释 视图定义和实施协议可能对使用_OwnerHistory_处理对象版本管理施加额外限制。
沿革 IFC1.0中新增实体

IFC4变更 _OwnerHistory_属性已改为可选项。

ENTITY IfcRoot
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcObjectDefinition
	,IfcPropertyDefinition
	,IfcRelationship));
	GlobalId : IfcGloballyUniqueId;
	OwnerHistory : OPTIONAL IfcOwnerHistory;
	Name : OPTIONAL IfcLabel;
	Description : OPTIONAL IfcText;
 UNIQUE
	UR1 : GlobalId;
END_ENTITY;
