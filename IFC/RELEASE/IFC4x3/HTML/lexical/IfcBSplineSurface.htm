8.9.3.7 IfcBSplineSurface

注释 该实体不是标准化模式子集或实现级别的一部分。

抽象 该定义可能不会被直接实例化。

8.9.3.7.1 语义定义（Semantic definition）

IfcBSplineSurface 是有理或多项式参数曲面的通用形式。

注释 Definition according to ISO/CD 10303-42:1992 A b_spline_surface is a general form of rational or polynomial parametric surface which is represented by control points, basis functions, and possibly, weights. As with the corresponding curve entity it has some special subtypes where some of the data can be derived.

此处使用的符号是：

控制点按以下顺序排列：P00, P01, P02, ..., PK1(K2-1), PK1K2

在有理子类型的情况下，权重按类似顺序排列。

对于每个参数 s = u 或 v，如果 k 是控制点上的上索引，d 是 s 的次数，则节点数组是 (k + d + 2) 个实数的数组 [s-d, ...., sk+1]，使得对于 [-d, k] 中的所有索引 j；sj ≤ sj+1。此数组从相应的 u_knots 或 v_knots 列表中获得，通过根据重数重复每个多重节点。

Nid，度为 d 的第 i 个归一化 B 样条基函数，定义在该数组的子集 [si-d, ...., si+1] 上。

曲面形式用于识别特定的二次曲面类型（次数应为 2）、规则曲面和旋转曲面。与 B 样条曲线一样，曲面形式仅供参考，样条数据优先。

8.9.3.7.4 形式化表示（Formal representation） 

ENTITY IfcBSplineSurface
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcBSplineSurfaceWithKnots))
 SUBTYPE OF (IfcBoundedSurface);
	UDegree : IfcInteger;
	VDegree : IfcInteger;
	ControlPointsList : LIST [2:?] OF LIST [2:?] OF IfcCartesianPoint;
	SurfaceForm : IfcBSplineSurfaceForm;
	UClosed : IfcLogical;
	VClosed : IfcLogical;
	SelfIntersect : IfcLogical;
 DERIVE
	 UUpper : IfcInteger := SIZEOF(ControlPointsList) - 1;
	 VUpper : IfcInteger := SIZEOF(ControlPointsList[1]) - 1;
	 ControlPoints : ARRAY [0:UUpper] OF ARRAY [0:VUpper] OF IfcCartesianPoint := IfcMakeArrayOfArray(ControlPointsList,
0,UUpper,0,VUpper);
END_ENTITY;
