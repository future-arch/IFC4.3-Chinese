5.4.3.66 IfcSpatialElement（空间构件）

注释 该实体是 IfcProduct 或 IfcTypeProduct 的子类型，因此是每个标准化模式子集和实现级别的一部分。

抽象 该定义可能不会被直接实例化。

5.4.3.66.1 语义定义（Semantic definition）

空间构件是所有可能用于定义空间结构或定义空间区域的空间构件的泛化。

  • 作为 IfcSpatialStructureElement 的层次化空间结构元素
  • 空间结构是项目的层次化分解。该空间结构通常用于提供项目结构来组织项目。
  • 空间项目结构可以根据项目需要定义任意多级分解。空间分解中的一些元素示例包括：场地、建筑、楼层、空间。对于基础设施项目还包括：桥梁、铁路、道路、港口、隧道及其部分。
  • 作为 IfcSpatialZone 的空间区域
  • 空间区域是在某些功能考虑下对项目进行的非层次化且可能重叠的分解。
  • 空间区域可用于表示热工区域、照明区域、可用面积区域。
  • 空间区域可用于表示土木工程中使用的水平空间结构。
  • 空间区域可以有其独立的定位和形状表示。
沿革 IFC4 中的新实体。

5.4.3.66.5 概念用法（Concept usage）

5.4.3.66.6 形式化表示（Formal representation） 

ENTITY IfcSpatialElement
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcExternalSpatialStructureElement
	,IfcSpatialStructureElement
	,IfcSpatialZone))
 SUBTYPE OF (IfcProduct);
	LongName : OPTIONAL IfcLabel;
 INVERSE
	ContainsElements : SET [0:?] OF IfcRelContainedInSpatialStructure FOR RelatingStructure;
	ServicedBySystems : SET [0:?] OF IfcRelServicesBuildings FOR RelatedBuildings;
	ReferencesElements : SET [0:?] OF IfcRelReferencedInSpatialStructure FOR RelatingStructure;
	IsInterferedByElements : SET [0:?] OF IfcRelInterferesElements FOR RelatedElement;
	InterferesElements : SET [0:?] OF IfcRelInterferesElements FOR RelatingElement;
END_ENTITY;
