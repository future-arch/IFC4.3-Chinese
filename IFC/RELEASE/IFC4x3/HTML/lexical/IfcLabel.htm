8.11.2.38 IfcLabel

8.11.2.38.1 语义定义（Semantic definition）

标签是可用于引用某物的术语。它是一个字符串，代表某物的可人工解释的名称，并且应具有自然语言含义。

类型：最多255个字符的STRING

注释 类型改编自 ISO 10303-41 中定义的 label。
沿革 IFC2x 中的新类型。
IFC4 变更 之前推荐的255个字符大小限制现在是强制性的。

值限制

作为用于命名的可读字符串，标签通常是人工指定且与区域设置相关的（与纯粹机器可读的标识符 IfcIdentifier 相对）。

注释 ISO 10303.21 中定义的交换结构中可能出现的 STRINGs 的字符集在 ISO 10646 中提供。文件交换时的字符编码在 ISO 10303-21 和 ISO 10303-28 中定义。这些规范定义了来自 ISO 8859-1...-16 的8位字符以及来自 ISO 10646 的2字节和4字节 Unicode 字符的编码。
注释 虽然 IfcIdentifier 限制为255个字符，但根据具体的编码和标识符的内容，编码后在交换文件中的大小可能远大于255个字节。

8.11.2.38.2 形式化表示（Formal representation） 

TYPE IfcLabel = STRING(255);
END_TYPE;
