6.4.4.15 Pset_ManufacturerTypeInformation（制造商类型信息）

6.4.4.15.1 语义定义（Semantic definition）

定义制造商可能提供的制造产品类型（系列）的特征。请注意，“制造”一词也可用于指代由供应商提供和标识的产品，或由第三方供应商在场外组装的产品。

HISTORY: This property set replaces the entity IfcManufacturerInformation from previous IFC releases. IFC 2x4: AssemblyPlace property added.

6.4.4.15.2 适用实体

PSET_TYPEDRIVENOVERRIDE None
  • IfcElement
  • IfcElementType
