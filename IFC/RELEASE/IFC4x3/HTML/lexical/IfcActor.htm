5.1.3.1 IfcActor

RV ⓘ
AbV ⓘ

5.1.3.1.1 语义定义（Semantic definition）

IfcActor 定义了在项目全生命周期中涉及的所有参与者或人员代理。它促进了在IFC对象模型资源部分使用人员和组织的定义。这包括名称、地址、通信地址和角色。

沿革 IFC2.0中的新实体

5.1.3.1.6 形式化表示（Formal representation） 

ENTITY IfcActor
 SUPERTYPE OF (ONEOF
	(IfcOccupant))
 SUBTYPE OF (IfcObject);
	TheActor : IfcActorSelect;
 INVERSE
	IsActingUpon : SET [0:?] OF IfcRelAssignsToActor FOR RelatingActor;
END_ENTITY;
