8.9.3.35 IfcIndexedPolyCurve

RV ⓘ
AbV ⓘ

8.9.3.35.1 语义定义（Semantic definition）

IfcIndexedPolyCurve 是一个有界曲线，仅包含线性段和圆弧段，由笛卡尔点列表和可选的段列表定义，段列表提供了指向笛卡尔点列表的索引。如果未提供 Segments 列表，则 IfcCartesianPointList 中的所有点都按其在 IfcCartesianPointList 中出现的顺序通过直线段连接。

如果提供了 Segments 列表，则其解释如下：

  • 类型为 IfcLineIndex 的段：第一个索引是折线段的起点，最后一个索引是折线段的终点。如果包含超过两个索引，则所有中间索引定义了折线的中点，按索引出现的顺序连接；
  • 类型为 IfcArcIndex 的段：第一个索引是圆弧的起点，第二个索引是圆弧上的一个点，第三个索引是圆弧的终点。这三个点不应共线。如果此非正式命题未得到满足，则该圆弧段应被视为折线段。
示例 图 8.9.3.35.B 说明了一个具有直线段和圆弧段的有界开放 IfcIndexedPolyCurve。在此示例中，直线段仅有两个点和一个边，但可以包含指向笛卡尔点列表的多个索引，从而定义了一个多边折线段。
示例 图 8.9.3.35.B 说明了一个仅具有直线段的有界开放 IfcIndexedPolyCurve。在此示例中，未提供 Segments 列表，因此点按其在 IfcCartesianPointList 中出现的顺序绘制。

IfcIndexedPolyCurve 表示开放或闭合曲线，具体取决于以下条件：

  • 如果提供了 Segments 列表：如果最后一个 Segment 的最后一个索引与第一个 Segment 的第一个索引相同，则该多曲线是闭合曲线，否则它是开放曲线。
  • 如果未提供 Segments 列表：如果笛卡尔点列表的第一个和最后一个点相同，则该多曲线是闭合曲线，否则它是开放曲线。
沿革 IFC4 ADD1 中的新实体。

Informal Propositions

  1. 考虑到适用的 IfcGeometricRepresentationContext 提供的 Precision 因子后，IfcIndexedPolyCurve 的任意两个连续点不应重合。
  2. 考虑到适用的 IfcGeometricRepresentationContext 提供的 Precision 因子后，IfcIndexedPolyCurve 的任意 IfcArcIndex 段的三个点不应共线。

8.9.3.35.5 形式化表示（Formal representation） 

ENTITY IfcIndexedPolyCurve
 SUBTYPE OF (IfcBoundedCurve);
	Points : IfcCartesianPointList;
	Segments : OPTIONAL LIST [1:?] OF IfcSegmentIndexSelect;
	SelfIntersect : OPTIONAL IfcBoolean;
 WHERE
	Consecutive : NOT(EXISTS(Segments)) OR IfcConsecutiveSegments(Segments);
END_ENTITY;
