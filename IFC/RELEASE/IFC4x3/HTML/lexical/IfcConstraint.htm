8.3.3.1 IfcConstraint

注释 该实体不是标准化模式子集或实现级别的一部分。

抽象 该定义可能不会被直接实例化。

8.3.3.1.1 语义定义（Semantic definition）

IfcConstraint 用于定义可应用于对象或属性值的约束、限制值或边界条件。

约束可细分为用户定义约束和系统定义约束。用户定义约束由用户应用，并限于高级定义，如对象属性。系统定义约束可应用于任何对象属性，通常由应用程序定义以强制执行数据验证或参数行为，旨在由应用程序强制执行，但不能由最终用户直接编辑。

IfcConstraint 可通过 IfcRelAssociatesConstraint 关系与 IfcObjectDefinition 或 IfcPropertyDefinition 的任何子类型相关联，以指示系统定义约束，或可通过 IfcResourceConstraintRelationship 与 IfcResourceObjectSelect（如 IfcPropertySingleValue）相关联，以指示用户定义约束。

约束必须通过 IfcConstraint.Name 属性具有名称，并可选择通过 IfcConstraint.Description 提供描述。约束的等级（硬性、软性、咨询性）必须通过 IfcConstraint.ConstraintGrade 或 IfcConstraint.UserDefinedGrade 指定，而约束的来源、创建参与者和创建时间可以通过 IfcConstraint.ConstraintSource、IfcConstraint.CreatingActor 和 IfcConstraint.CreationTime 可选地断言。

约束还可以通过 IfcExternalReferenceRelationship 与附加的外部信息（如分类或文档信息）相关联，可通过逆向属性 IfcConstraint.HasExternalReferences 访问。

沿革 New entity in IFC2.0
IFC4 变更 CreationTime changed to IfcDateTime for ISO 8601 representation, HasExternalReferences new inverse attribute.

8.3.3.1.5 形式化表示（Formal representation） 

ENTITY IfcConstraint
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcMetric
	,IfcObjective));
	Name : IfcLabel;
	Description : OPTIONAL IfcText;
	ConstraintGrade : IfcConstraintEnum;
	ConstraintSource : OPTIONAL IfcLabel;
	CreatingActor : OPTIONAL IfcActorSelect;
	CreationTime : OPTIONAL IfcDateTime;
	UserDefinedGrade : OPTIONAL IfcLabel;
 INVERSE
	HasExternalReferences : SET [0:?] OF IfcExternalReferenceRelationship FOR RelatedResourceObjects;
	PropertiesForConstraint : SET [0:?] OF IfcResourceConstraintRelationship FOR RelatingConstraint;
 WHERE
	WR11 : (ConstraintGrade <> IfcConstraintEnum.USERDEFINED) OR
((ConstraintGrade = IfcConstraintEnum.USERDEFINED) AND EXISTS(SELF\IfcConstraint.UserDefinedGrade));
END_ENTITY;
