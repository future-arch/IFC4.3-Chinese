8.10.3.7 IfcMaterialLayerSet

RV ⓘ
AbV ⓘ

8.10.3.7.1 语义定义（Semantic definition）

IfcMaterialLayerSet 是一个名称，用于标识由多层材料构成的元素的材料，并通过它表达各个层相对位置的可能性。

材料层集基线（MlsBase）描述了材料层定位的假想轴。

  • 如果 IfcMaterialLayerSet 直接分配给元素或元素类型，则各个层根据其在 MaterialLayers 列表中的位置堆叠，而不提供如何将材料层信息在空间上与元素或元素类型的形状表示相关联的信息。
  • 如果通过 IfcMaterialLayerSetUsage 将 IfcMaterialLayerSet 分配给元素，则 MlsBase 沿元素的参考轴或参考平面定位。IfcMaterialLayerSetUsage 支持从参考轴或平面到 MlsBase 的偏移，该偏移结合了层和偏移。IfcMaterialLayerWithOffsets 的子类型支持从元素边缘的偏移。正 LayerSetDirection（MlsDirection）描述了各个材料层堆叠的方向。IfcMaterialLayer 是无间隙堆叠的。材料层集内的间隙本身表示为层。
示例 A cavity brick wall would be modeled as IfcMaterialLayerSet consisting of three IfcMaterialLayer's: brick, air cavity and brick. The air gap is identified by the IsVentilated flag at IfcMaterialLayer.
沿革 New entity in IFC1.0
IFC4 变更 Subtyped from IfcMaterialDefinition, the attribute Description has been added at the end of attribute list.

属性使用定义

如图 8.10.3.7.A 所示，每个 IfcMaterialLayerSet 都隐式定义了一个材料层集基线（MlsBase），第一个 IfcMaterialLayer 的开始与之对齐。层集的总厚度从各个层厚计算得出，第一层从 MlsBase 开始，后续层堆叠在前一层之上（无间隙或重叠）。

8.10.3.7.3 特性（Attributes）

8.10.3.7.4 形式化表示（Formal representation） 

ENTITY IfcMaterialLayerSet
 SUBTYPE OF (IfcMaterialDefinition);
	MaterialLayers : LIST [1:?] OF IfcMaterialLayer;
	LayerSetName : OPTIONAL IfcLabel;
	Description : OPTIONAL IfcText;
 DERIVE
	 TotalThickness : IfcLengthMeasure := IfcMlsTotalThickness(SELF);
END_ENTITY;
