8.8.3.27 IfcPolygonalBoundedHalfSpace

AbV ⓘ

8.8.3.27.1 语义定义（Semantic definition）

多边形边界半空间是半空间实体的一个特殊子类型，其中布尔表达式中使用的半空间材料由二维边界限定。半空间的基准曲面相对于对象坐标系（在超类型 IfcHalfSpaceSolid 中定义）由其法线定位，其边界（包含直线段和圆弧段）在由 Position 特性建立的位置坐标系的 XY 平面上定义，减去体垂直于位置坐标系的 XY 平面（即沿 Position 特性定义的 Z 轴正向）进行挤出。

边界由位置坐标系 XY 平面内的二维折线（或由直线段和圆弧段组成的二维复合曲线，或索引折线）定义。半空间所在的曲面一侧由曲面法线和约定标志确定。如果约定标志为 TRUE，则子集是法线指向外部的子集。如果约定标志为 FALSE，则子集是法线指向内部的子集。

注释 A polygonal bounded half space is not a subtype of IfcSolidModel, half space solids are only useful as operands in Boolean expressions. The term IfcPolygonalBoundedHalfSpace was coined when only curves of type IfcPolyline were allowed as boundary definitions.

图 8.8.3.27.A 说明了一个多边形边界半空间。

  • 黑色坐标表示对象坐标系（通常由 IfcLocalPlacement 提供）。
  • 绿色坐标表示位置坐标系；PolygonalBoundary 在此坐标系内给出。它由 IfcPolygonalBoundedHalfSpace.Position 提供。此坐标系相对于对象坐标系。减去体的挤出方向是正 Z 轴。
  • 红色坐标表示平面的法线。它由 BaseSurface (IfcSurface.Position) 定义。此法线也相对于对象坐标系。

多边形边界半空间用于限制布尔差集表达式中的半空间体积。仅使用半空间实体和通过挤出多边形边界定义的挤出区域实体之间的理论交集所定义的区域用于布尔表达式。

PolygonalBoundary 定义了限定半空间在布尔表达式中有效性的二维折线。BaseSurface 由一个平面定义，平面的法线与 AgreementFlag 一起定义了半空间材料的侧面。

沿革 New entity in IFC2x.

非正式命题

  1. 提供 PolygonalBoundary 的 IfcPolyline 或 IfcCompositeCurve 应为闭合的。
  2. 如果 PolygonalBoundary 由 IfcCompositeCurve 提供，则它只能包含 IfcPolyline 类型的 IfcCompositeCurveSegment，或者 IfcTrimmedCurve（其 BasisCurve 类型为 IfcLine 或 IfcCircle）。
  3. IfcHalfSpaceSolid 超类型中定义的 BaseSurface 应为 IfcPlane 类型。
  4. 作为 BaseSurface 的平面的法线不应垂直于位置坐标系的 z 轴。

8.8.3.27.5 形式化表示（Formal representation） 

ENTITY IfcPolygonalBoundedHalfSpace
 SUBTYPE OF (IfcHalfSpaceSolid);
	Position : IfcAxis2Placement3D;
	PolygonalBoundary : IfcBoundedCurve;
 WHERE
	BoundaryDim : PolygonalBoundary.Dim = 2;
	BoundaryType : SIZEOF(TYPEOF(PolygonalBoundary) * [
    'IFC4X3_DEV_62887e3f.IFCPOLYLINE',
    'IFC4X3_DEV_62887e3f.IFCCOMPOSITECURVE',
    'IFC4X3_DEV_62887e3f.IFCINDEXEDPOLYCURVE'
  ]) = 1;
END_ENTITY;
