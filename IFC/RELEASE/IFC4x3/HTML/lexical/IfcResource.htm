5.1.3.43 IfcResource（资源）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.1.3.43.1 语义定义（Semantic definition）

IfcResource 包含表示在流程中使用某事物所产生的成本、进度及其他影响所需的信息。IfcResource 并非用于对事物本身的通用属性进行建模，但可指定从 IfcResource 到所使用事物的可选链接（具体而言，通过 IfcRelAssignsToResource 关系建立从 IfcResource 子类型到 IfcProduct 的关系）。

IfcResource 有两种基本用途。首先，若使用 IfcResource 时不需要事物的属性，或该事物类型尚未在 IFC 中显式建模，则无需在系统中实例化资源与事物之间的链接。也就是说，仅凭 IfcResource（或其子类型）的属性就足以表示在项目中将某事物作为资源使用的情况。

示例 诸如土方运输车辆或工具等施工设备目前未在 IFC 中建模。出于估算和调度的目的，可以仅使用 IfcResource 的子类型来表示这些设备。

其次，若使用 IfcResource 对象时需要事物的属性，且这些事物已显式建模为对象，则可将 IfcResource 实例链接到所引用事物类型的实例。可能用作资源且已在 IFC 中建模的事物包括物理产品、人员和组织以及材料。为此方法提供了关系对象 IfcRelAssignsToResource。

继承的属性 ObjectType 用作标识资源类型的文本代码。

沿革 IFC1.0 中的新实体
IFC2x 变更 抽象实体中的 BaseUnit 和 ResourceConsumption 属性已被移除；它们在层次结构的较低级别重新引入。

5.1.3.43.5 形式化表示（Formal representation） 

ENTITY IfcResource
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcConstructionResource))
 SUBTYPE OF (IfcObject);
	Identification : OPTIONAL IfcIdentifier;
	LongDescription : OPTIONAL IfcText;
 INVERSE
	ResourceOf : SET [0:?] OF IfcRelAssignsToResource FOR RelatingResource;
END_ENTITY;
