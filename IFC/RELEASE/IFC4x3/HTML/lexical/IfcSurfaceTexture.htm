8.12.3.35 IfcSurfaceTexture

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

8.12.3.35.1 语义定义（Semantic definition）

IfcSurfaceTexture 提供了一个二维图像纹理贴图。它可以引用外部图像文件（通过URL引用，IfcImageTexture），将图像文件作为二进制大对象（长二进制）包含在数据集中（IfcBlobTexture），或者通过显式包含像素数组（IfcPixelTexture）来提供。

以下来自ISO/IEC 19775-1 X3D Architecture and base components (X3D Specification) 的定义适用：

  • Texture: 用于纹理贴图的图像，在应用于几何体时创建视觉外观效果。
  • Texture map: 纹理加上将纹理映射到几何体所需的通用参数。
  • 纹理由二维图像定义，其中包含描述纹理的颜色值数组。纹理值的解释取决于纹理的组件数量和图像格式的具体细节。通常，纹理可以使用以下形式之一进行描述：
  • 强度纹理（单分量）
  • 强度加Alpha不透明度纹理（二分量）
  • 全RGB纹理（三分量）
  • 全RGB加Alpha不透明度纹理（四分量）
注释 Image formats specify an alpha opacity, not transparency (where alpha = 1 - transparency).

8.12.3.35.1.1 Texture coordinate systems

图1说明了纹理坐标系。

以下来自ISO/IEC 19775-1 X3D Architecture and base components (X3D Specification) 关于纹理坐标的定义适用：

  • 纹理贴图在二维坐标系（s, t）中定义，在两个方向上都从[0.0, 1.0]开始。图像的底边对应于纹理贴图的S轴，图像的左边对应于纹理贴图的T轴。图像的左下像素对应于s=0, t=0，图像的右上像素对应于s=1, t=1。纹理贴图可以被视为二维颜色函数，给定一个（s, t）坐标，返回一个颜色值colour(s, t)。
  • RepeatS 和 RepeatT 布尔标志控制当纹理贴图应用于几何曲面时，是否在[0.0, 1.0]纹理坐标范围之外重复，或者是否被限制在[0.0, 1.0]范围内。TextureTransform 在纹理应用于几何曲面之前，对纹理应用二维非均匀变换。
  • 这些参数支持纹理在形状上的大小、方向和位置的变化。请注意，这些操作在几何体表面上看起来是相反的。例如，scale值为(2 2)将缩放纹理坐标，并导致纹理大小缩小一半（纹理坐标是原来的两倍，因此导致纹理重复）。(0.5 0.0)的平移将纹理坐标沿S轴正向移动0.5个单位，并在几何体表面上将纹理沿S轴负向平移0.5个单位。纹理坐标绕π/2旋转会导致纹理在几何体上绕-π/2旋转。
  • center 字段指定了纹理坐标空间中的一个平移偏移量，缩放和旋转字段将在此偏移量周围应用。scale 字段指定了在_center_点周围纹理坐标的S和T方向上的缩放因子。scale值应在(-∞,∞)范围内。rotation 字段指定了在应用缩放后，纹理坐标绕_center_点的旋转（以弧度为单位）。正的旋转值使纹理坐标绕中心逆时针旋转，从而使纹理本身的外观顺时针旋转。translation 字段指定了纹理坐标的平移。

以下约定适用：

  • center = TextureTransform.LocalOrigin;
  • rotation = TextureTransform.Axis1
  • scale S = TextureTransform.Scale
  • scale T = TextureTransform.Scale2

纹理在曲面或实体上的映射由以下优先级确定：

  1. 如果几何体是定义纹理映射的特殊IFC类型，例如IfcSphere、IfcBlock或IfcExtrudedAreaSolid，则该定义是默认的纹理映射。
  2. 如果为曲面纹理分配了IfcTextureCoordinate纹理坐标，则使用这些纹理坐标。
  3. 如果几何体没有纹理映射定义且没有纹理坐标，则纹理s坐标来自局部顶点X坐标，t坐标来自局部顶点Y坐标。沿每个轴，s和t纹理坐标从0到1，而几何体从边界框最小值到最大值。

8.12.3.35.3 特性（Attributes）

8.12.3.35.4 形式化表示（Formal representation） 

ENTITY IfcSurfaceTexture
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcBlobTexture
	,IfcImageTexture
	,IfcPixelTexture))
 SUBTYPE OF (IfcPresentationItem);
	RepeatS : IfcBoolean;
	RepeatT : IfcBoolean;
	Mode : OPTIONAL IfcIdentifier;
	TextureTransform : OPTIONAL IfcCartesianTransformationOperator2D;
	Parameter : OPTIONAL LIST [1:?] OF IfcIdentifier;
 INVERSE
	IsMappedBy : SET [0:?] OF IfcTextureCoordinate FOR Maps;
	UsedInStyles : SET [0:?] OF IfcSurfaceStyleWithTextures FOR Textures;
END_ENTITY;
