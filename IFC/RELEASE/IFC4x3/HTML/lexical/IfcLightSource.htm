8.14.3.3 IfcLightSource

注释 该实体不是标准化模式子集或实现级别的一部分。

抽象 该定义可能不会被直接实例化。

8.14.3.3.1 语义定义（Semantic definition）

注释 定义参考 ISO 10303-46： 光源实体由表面样式渲染中指定的反射率决定。光照是逐表面应用的：未定义表面之间的交互，如阴影或反射。
注释 实体采自 ISO 10303-46 中定义的 light_source。
注释 除了 ISO 10303-46 中定义的属性外，还添加了 ISO/IEC 14772-1:1997（VRML）中的附加属性：ambientIntensity 和 Intensity。还添加了 Name 属性（因为它不是通过 representation_item 继承的）。
沿革 IFC2x 中新增实体，IFC2x2 中重命名并增强。

8.14.3.3.3 特性（Attributes）

8.14.3.3.4 形式化表示（Formal representation） 

ENTITY IfcLightSource
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcLightSourceAmbient
	,IfcLightSourceDirectional
	,IfcLightSourceGoniometric
	,IfcLightSourcePositional))
 SUBTYPE OF (IfcGeometricRepresentationItem);
	Name : OPTIONAL IfcLabel;
	LightColour : IfcColourRgb;
	AmbientIntensity : OPTIONAL IfcNormalisedRatioMeasure;
	Intensity : OPTIONAL IfcNormalisedRatioMeasure;
END_ENTITY;
