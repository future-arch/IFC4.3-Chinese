8.8.2.4 IfcGeometricSetSelect

8.8.2.4.1 语义定义（Semantic definition）

IfcGeometricSetSelect 包括适用于构成几何集合的几何表示项。

注释 Definition from ISO10303-42: This select type identifies the types of entities which can occur in a geometric set.
注释 Corresponding STEP type geometric_set_select defined in ISO 10303-42.
沿革 New type in IFC2x.

8.8.2.4.2 类型值

8.8.2.4.3 形式化表示（Formal representation） 

TYPE IfcGeometricSetSelect = SELECT
	(IfcCurve
	,IfcPoint
	,IfcSurface);
END_TYPE;
