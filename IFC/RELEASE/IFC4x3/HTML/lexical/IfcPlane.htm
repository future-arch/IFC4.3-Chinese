8.9.3.46 IfcPlane（平面）

AbV ⓘ

8.9.3.46.1 语义定义（Semantic definition）

平面是在 x 和 y 方向上无界的表面。有界平面通过使用 IfcBoundedSurface 的子类型来定义，其中 BasisSurface 为平面。

注释 根据 ISO/CD 10303-42:1992 的定义 平面是具有恒定法向量的无界表面。平面由平面上的一点和该平面的法线方向定义。数据应按如下方式解释：
注释 矩形有界平面可通过 IfcRectangularTrimmedSurface 定义，其中 BasisSurface 为平面，U1 = x 方向的左边界，U2 = x 方向的右边界，V1 = y 方向的下边界，V2 = y 方向的上边界（假设 Usense 和 Vsense 与基础表面的方向一致）。
  C = Position.Location
  x = Position.P[1]
  y = Position.P[2]
  z = Position.P[3] => 平面法向量

继承属性解释如下：

  • SELF\IfcElementarySurface.Position 定义平面的位置和方向。
  • SELF\IfcElementarySurface.Position.Location 定义平面上的一个点。
  • SELF\IfcElementarySurface.Position.P[3] 定义平面的法向量。
注释 实体改编自 ISO 10303-42 中的 plane。
沿革 IFC1.5 中新增实体
