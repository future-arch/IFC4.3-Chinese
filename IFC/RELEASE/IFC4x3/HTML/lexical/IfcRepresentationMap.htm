8.9.3.58 IfcRepresentationMap

RV ⓘ
AbV ⓘ

8.9.3.58.1 语义定义（Semantic definition）

IfcRepresentationMap 定义了 MappingOrigin 内的基定义（也称为块、单元或宏），称为 MappedRepresentation。MappingOrigin 定义了定义 MappedRepresentation 的坐标系。

注释 Definition according to ISO/CD 10303-43:1992 A representation map is the identification of a representation and a representation item in that representation for the purpose of mapping. The representation item defines the origin of the mapping. The representation map is used as the source of a mapping by a mapped item.

RepresentationMap 通过 IfcMappedItem 在一个或多个 IfcShapeRepresentation 中使用。笛卡尔变换算子可用于将 MappedRepresentation 变换到形状表示的放置坐标系。表示映射的变换被限制为笛卡尔变换映射（平移、旋转、镜像和缩放）。

注释 The definition of a mapping which is used to specify a new representation item comprises a representation map and a mapped item entity. Without both entities, the mapping is not fully defined. Two entities are specified to allow the same source representation to be mapped into multiple new representations.
注释 Entity adapted from representation_map defined in ISO 10303-43.
沿革 New entity in IFC2x.

8.9.3.58.5 形式化表示（Formal representation） 

ENTITY IfcRepresentationMap;
	MappingOrigin : IfcAxis2Placement;
	MappedRepresentation : IfcRepresentation;
 INVERSE
	HasShapeAspects : SET [0:?] OF IfcShapeAspect FOR PartOfProductDefinitionShape;
	MapUsage : SET [0:?] OF IfcMappedItem FOR MappingSource;
 WHERE
	ApplicableMappedRepr : 'IFC4X3_DEV_62887e3f.IFCSHAPEMODEL' IN TYPEOF(MappedRepresentation);
END_ENTITY;
