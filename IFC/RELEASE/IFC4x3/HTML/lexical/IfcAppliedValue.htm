8.4.3.1 IfcAppliedValue

RV ⓘ
AbV ⓘ

8.4.3.1.1 语义定义（Semantic definition）

此实体捕获由公式驱动的值，并附加了单位基础、有效日期范围和分类等附加限定。

IfcAppliedValue 的范围由 AppliedValue 特性确定，该特性可以定义为 IfcMeasureWithUnit、IfcMonetaryMeasure 或通过 IfcAppliedValueSelect 类型定义的 IfcRatioMeasure。

可选地，IfcAppliedValue 可以具有适用日期。这旨在固定值开始生效的日期。它可以是模型中设置值的日期，也可以是值生效的之前或之后的日期。同样，IfcAppliedValue 可以具有“固定至”日期。这旨在固定值不再相关的日期。

IfcAppliedValue 的实例可以断言单位基础。这被定义为 IfcMeasureWithUnit，它确定了应用目的的单位值的范围。假定当断言此特性时，IfcAppliedValue 的值是单位数量的值。这在模式中并未强制执行，因此需要在应用程序中进行控制。

应用的值可以从文档（如价格表）中引用。一个或多个 IfcAppliedValue（或其子类型）的实例之间的关系是通过使用 IfcExternalReferenceRelationship 实现的，其中文档提供了 IfcExternalReferenceRelationship.RelatingExtReference，值实例是 IfcExternalReferenceRelationship.RelatedResourceObjects。

沿革 IFC2x2 中新增实体。
IFC4 变更 实体变为非抽象，以支持约束的通用公式表达式，日期相关属性的数据类型更改为 IfcDate，ValueType 和 Condition 从 IfcCostValue 提升，添加了 Components 和 ArithmeticOperator 属性以替换 IfcAppliedValueRelationship，实现更高效的编码和引用跟踪。

非正式命题

  1. 当 ArithmeticOperator 等于 MODULO 时，Components 列表应包含正好两个值，这两个值应为正整数。

8.4.3.1.3 特性（Attributes）

8.4.3.1.4 形式化表示（Formal representation） 

ENTITY IfcAppliedValue
 SUPERTYPE OF (ONEOF
	(IfcCostValue));
	Name : OPTIONAL IfcLabel;
	Description : OPTIONAL IfcText;
	AppliedValue : OPTIONAL IfcAppliedValueSelect;
	UnitBasis : OPTIONAL IfcMeasureWithUnit;
	ApplicableDate : OPTIONAL IfcDate;
	FixedUntilDate : OPTIONAL IfcDate;
	Category : OPTIONAL IfcLabel;
	Condition : OPTIONAL IfcLabel;
	ArithmeticOperator : OPTIONAL IfcArithmeticOperatorEnum;
	Components : OPTIONAL LIST [1:?] OF IfcAppliedValue;
 INVERSE
	HasExternalReference : SET [0:?] OF IfcExternalReferenceRelationship FOR RelatedResourceObjects;
END_ENTITY;
