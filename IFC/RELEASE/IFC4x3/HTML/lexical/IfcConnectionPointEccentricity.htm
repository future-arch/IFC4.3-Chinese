8.7.3.7 IfcConnectionPointEccentricity（连接点偏心）

注释 该实体不是标准化模式子集或实现级别的一部分。

8.7.3.7.1 语义定义（Semantic definition）

IfcConnectionPointEccentricity 用于描述在两个对象的点或顶点处进行物理连接的几何约束，这些点具有关联的点坐标。两个对象的连接点之间存在物理距离或偏心量。偏心量可以通过以下方式给出：

  • 提供 PointOnRelatingElement 和 PointOnRelatedElement，其中两个点坐标在公共父坐标系（最终在世界坐标系内）内不相同；
  • 提供 PointOnRelatingElement 以及三个距离度量 EccentricityInX、EccentricityInY 和 EccentricityInZ（或者如果底层坐标系是二维的，则仅提供 EccentricityInX 和 EccentricityInY）；或
  • 同时提供两者。
注释 如果同时提供了 PointOnRelatedElement 和 EccentricityInX、EccentricityInY（EccentricityInZ），这些值应保持一致。如果存在任何不一致，则以 PointOnRelatingElement 和 PointOnRelatedElement 之间的计算距离为准。

PointOnRelatingElement 属性处的 IfcPoint（或具有关联 IfcPoint 的 IfcVertexPoint）定义了连接元素的基本几何项连接的点。点坐标在 RelatingElement 的局部坐标系内提供，如使用 IfcConnectionPointGeometry 的 IfcRelConnects 子类型中所指定。可选地，相同的点坐标也可以通过使用 PointOnRelatedElement 属性在 RelatedElement 的局部坐标系内提供，否则必须通过三个偏心值给出到 RelatedElement 处的点的距离。

EccentricityInX、EccentricityInY 和 EccentricityInZ 的显式值始终在以下方向和坐标系中测量（定义值为正或负的情况）：

  • 在 RelatingElement 的坐标系内，从 PointOnRelatedElement 到 PointOnRelatingElement。
  • 此外：当用于指定结构分析模型中的连接时，IfcStructuralMember 应作为使用 IfcConnectionPointEccentricity 的关系对象的 RelatingElement，而 IfcStructuralConnection 是 RelatedElement。
沿革 IFC2x3 中的新实体。

8.7.3.7.4 形式化表示（Formal representation） 

ENTITY IfcConnectionPointEccentricity
 SUBTYPE OF (IfcConnectionPointGeometry);
	EccentricityInX : OPTIONAL IfcLengthMeasure;
	EccentricityInY : OPTIONAL IfcLengthMeasure;
	EccentricityInZ : OPTIONAL IfcLengthMeasure;
END_ENTITY;
