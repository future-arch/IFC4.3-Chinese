8.9.3.74 IfcTrimmedCurve

注释 该实体不是标准化模式子集或实现级别的一部分。

8.9.3.74.1 语义定义（Semantic definition）

IfcTrimmedCurve 是一条在两端都被裁剪的有界曲线。裁剪点可以由笛卡尔点或参数值提供，具体取决于 BasisCurve 的参数化。 SenseAgreement 特性指示 IfcTrimmedCurve 的方向是否与 BasisCurve 的方向一致或相反。

注释 In case of the BasisCurve being a closed curve, such as an IfcCircle or IfcEllipse, the SenseAgreement affects the geometric shape of the IfcTrimmedCurve.
注释 Since the BasisCurve is closed (type IfcCircle), the exception of the informal proposition IP3 applies, i.e. the sense flag is not required to be consistent with the parameter values of Trim1 and Trim1, so the rule (sense = parameter 1 < parameter 2) may not be fulfilled.
注释 Definition according to ISO/CD 10303-42:1992 A trimmed curve is a bounded curve which is created by taking a selected portion, between two identified points, of the associated basis curve. The basis curve itself is unaltered and more than one trimmed curve may reference the same basis curve. Trimming points for the curve may be identified by:
注释 对于闭合曲线，可能需要增加 t1 或 t2 的参数长度以与方向标志保持一致。
注释 实体改编自 ISO 10303-42 中定义的 trimmed_curve
注释 Entity adapted from trimmed_curve defined in ISO 10303-42
  • parametric value
  • geometric position
  • both of the above

曲线的每一端至少应指定其中一个。 SenseAgreement 可以明确定义闭合曲线（如圆）的任何段。方向和有序端点的组合使得可以定义连接圆或其他闭合曲线上的两个不同点的四个不同的定向段。为此，假定参数范围的循环属性；例如，370 度等同于 10 度。

The IfcTrimmedCurve has a parameterization which is inherited from the particular basis curve reference. More precisely the parameter s of the trimmed curve is derived from the parameter of the basis curve as follows:

  • if SenseAgreement is TRUE: s = t - t~1~
  • if SenseAgreement is FALSE: s = t~2~ - t

在上述方程中，t~1~ 是由 Trim1 给出的值或对应于点 1 的参数值，t~2~ 是由 Trim2 给出的值或对应于点 2 的参数值。结果 IfcTrimmedCurve 的参数范围从第一个裁剪点的 0 到第二个裁剪点的 |t~2~ - t~1~|。

非正式命题

  1. 当 Trim1 和 Trim2 同时存在参数值和笛卡尔点时，它们必须一致。（即，在参数值处评估的 BasisCurve 必须与指定的点重合）。
  2. 当 Trim1 或 Trim2 指定笛卡尔点时，该点必须位于 BasisCurve 上。
  3. 除了闭合 BasisCurve 且参数 1 和参数 2 都存在的情况外，它们必须与方向标志一致，即 (方向 = 参数 1 < 参数 2)。或者，对于所有存在的参数 1 和参数 2 的开放曲线，它们必须与 SenseAgreement 一致，即 SenseAgreement = (参数 1 < 参数 2)。
  4. 如果参数 1 和参数 2 都存在，则参数 1 <> 参数 2。对于闭合基准曲线，例如 IfcCircle 或 IfcEllipse，这也适用于循环属性，因为 360' 等同于 0'，参数 1 = 360' 和参数 2 = 0' 被视为相等，因此违反此命题。
  5. 当 Trim1 或 Trim2 指定参数值时，该值必须位于 BasisCurve 的参数范围内。

8.9.3.74.5 形式化表示（Formal representation） 

ENTITY IfcTrimmedCurve
 SUBTYPE OF (IfcBoundedCurve);
	BasisCurve : IfcCurve;
	Trim1 : SET [1:2] OF IfcTrimmingSelect;
	Trim2 : SET [1:2] OF IfcTrimmingSelect;
	SenseAgreement : IfcBoolean;
	MasterRepresentation : IfcTrimmingPreference;
 WHERE
	NoTrimOfBoundedCurves : NOT('IFC4X3_DEV_62887e3f.IFCBOUNDEDCURVE' IN TYPEOF(BasisCurve));
	Trim1ValuesConsistent : (HIINDEX(Trim1) = 1) OR (TYPEOF(Trim1[1]) <> TYPEOF(Trim1[2]));
	Trim2ValuesConsistent : (HIINDEX(Trim2) = 1) OR (TYPEOF(Trim2[1]) <> TYPEOF(Trim2[2]));
END_ENTITY;
