8.5.3.1 IfcEventTime

RV ⓘ
AbV ⓘ

8.5.3.1.1 语义定义（Semantic definition）

IfcEventTime 捕获关于事件的时间信息，包括不同类型的事件日期（即实际、计划、最早和最晚）。

所有给定的值应由应用程序提供，也就是说，IFC schema 不处理过程时间值之间的依赖关系。在此阶段，也没有通过保证有意义的日期值填充的规则进行一致性检查。因此，应用程序负责提供合理的值，并且如果应用程序接收到事件日期，则必须自行进行一致性检查。

IfcEventTime 此外还提供了一个通用机制，用于区分用户给定的时间值与从用户给定的时间值和其他约束（如工作日历和分配的资源（来自过程图））派生的时间值。数据来源标志作为一个单一属性应用于 IfcEventTime 的所有日期时间相关属性。

8.5.3.1.4 形式化表示（Formal representation） 

ENTITY IfcEventTime
 SUBTYPE OF (IfcSchedulingTime);
	ActualDate : OPTIONAL IfcDateTime;
	EarlyDate : OPTIONAL IfcDateTime;
	LateDate : OPTIONAL IfcDateTime;
	ScheduleDate : OPTIONAL IfcDateTime;
END_ENTITY;
