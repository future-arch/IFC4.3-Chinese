5.3.3.9 IfcWorkControl（工作控制）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.3.3.9.1 语义定义（Semantic definition）

IfcWorkControl 是一个抽象超类型，用于捕获 IfcWorkPlan 和 IfcWorkSchedule 共有的信息。

沿革 IFC2x 中新增实体
IFC4 变更 修正了文档中资源与工作控制的分配关系。根据任务时间定义的变更及摘要任务的引入，更新了任务与工作控制的分配关系。标识符已重命名（现为 Identification）并提升至超类型 IfcControl

工作控制可分配资源，这通过 IfcRelAssignsToControl 关系处理。工作控制还应定义提供其使用详情的上下文。若无需特殊上下文信息，则应使用 IfcProject 实例作为全局上下文。此时应通过 IfcRelDeclares 建立工作控制与 IfcProject 的显式链接。

属性 IfcWorkControl.Purpose 用于定义工作计划或工作进度表的目的。对于 IfcWorkPlan，该属性可用于确定工作计划是用于成本估算、任务调度还是其他特定目的。

5.3.3.9.5 概念用法（Concept usage）

5.3.3.9.6 形式化表示（Formal representation） 

ENTITY IfcWorkControl
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcWorkPlan
	,IfcWorkSchedule))
 SUBTYPE OF (IfcControl);
	CreationDate : IfcDateTime;
	Creators : OPTIONAL SET [1:?] OF IfcPerson;
	Purpose : OPTIONAL IfcLabel;
	Duration : OPTIONAL IfcDuration;
	TotalFloat : OPTIONAL IfcDuration;
	StartTime : IfcDateTime;
	FinishTime : OPTIONAL IfcDateTime;
END_ENTITY;
