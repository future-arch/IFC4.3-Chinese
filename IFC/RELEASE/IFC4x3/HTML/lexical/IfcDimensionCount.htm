8.9.2.7 IfcDimensionCount（维度计数）

8.9.2.7.1 语义定义（Semantic definition）

IfcDimensionCount 定义了坐标空间的维度。在本规范中，其维度被限制为1、2或3。

注释 分配给几何表示上下文的形状表示可能包含较低维度的几何表示项，特别是在为3D扫描区域定义平面边界作为2D横截面时。
注释 根据ISO/CD 10303-42:1992的定义 维度计数是一个用于定义坐标空间维度的正整数。
注释 类型改编自ISO 10303-42中定义的 dimension_count。
沿革 IFC1.5中新增类型

8.9.2.7.2 形式化命题（Formal propositions）

8.9.2.7.3 形式化表示（Formal representation） 

TYPE IfcDimensionCount = INTEGER;
 WHERE
	 WR1 : { 0 < SELF <= 3 };
END_TYPE;
