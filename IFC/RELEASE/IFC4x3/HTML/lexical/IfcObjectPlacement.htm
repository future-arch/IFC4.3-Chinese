8.7.3.15 IfcObjectPlacement

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

8.7.3.15.1 语义定义（Semantic definition）

IfcObjectPlacement 是定义对象坐标系的特殊类型的抽象超类型。对于具有形状表示的每个产品，都必须提供 IfcObjectPlacement。

对象放置可以提供如下：

  • 绝对放置：通过轴 2 放置，相对于世界坐标系，
  • 相对放置：通过轴 2 放置，相对于另一个产品的对象放置，
  • 通过网格引用：通过设计网格的两个轴提供的虚拟交点和参考方向，
  • 线性放置：通过沿曲线的距离，带有可选偏移量。

在任何情况下，对象放置都必须明确定义对象坐标系，作为二维轴放置 (IfcAxis2Placement2D) 或三维轴放置 (IfcAxis2Placement3D)。可能需要计算轴放置。

沿革 New entity in IFC2x.

8.7.3.15.3 特性（Attributes）

8.7.3.15.4 形式化表示（Formal representation） 

ENTITY IfcObjectPlacement
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcGridPlacement
	,IfcLinearPlacement
	,IfcLocalPlacement));
	PlacementRelTo : OPTIONAL IfcObjectPlacement;
 INVERSE
	PlacesObject : SET [0:?] OF IfcProduct FOR ObjectPlacement;
	ReferencedByPlacements : SET [0:?] OF IfcObjectPlacement FOR PlacementRelTo;
END_ENTITY;
