8.15.3.26 IfcTrapeziumProfileDef（梯形轮廓定义）

注释 该实体不是标准化模式子集或实现级别的一部分。

弃用 该定义将在标准的未来主版本中移除。

8.15.3.26.1 语义定义（Semantic definition）

IfcTrapeziumProfileDef 定义了一个梯形作为扫掠表面几何体或扫掠面积实体所使用的轮廓定义。它通过其顶部X和底部X范围、Y范围以及顶部X范围的偏移量来给出，并放置于由 Position 属性建立的二维位置坐标系内。它位于位置坐标系的中心，即边界框的中心。

沿革 IFC1.5 中的新实体。在 IFC2x 中，使用定义已更改。
IFC4.3.0.0 弃用 此实体已被弃用。请改用 IfcArbitraryClosedProfileDef，通过显式外边界来定义梯形。

图 8.15.3.26.A 展示了梯形轮廓定义的参数。

位置

参数化轮廓定义了其自身的位置坐标系。基础坐标系由使用该轮廓定义的扫掠表面或扫掠面积实体定义。它是以下任一情况的 xy 平面：

  • IfcSweptSurface.Position
  • IfcSweptAreaSolid.Position

或者，在截面脊线的情况下，是 IfcSectionedSpine.CrossSectionPositions 每个列表成员的 xy 平面。

通过使用位置坐标的偏移量，参数化轮廓可以定位于中心（使用 x, y 偏移量 = 0.），或定位于相对于轮廓的任何位置。显式坐标偏移量用于定义基点（例如，左上边界）。

参数

IfcTrapeziumProfileDef 在位置坐标系内定义，其中 BottomDim 定义了底边的长度度量（沿正 x 轴的一半），YDim 定义了底边和顶边平行距离的长度度量（沿正 y 轴的一半）。顶边从点 -BottomLine/2, YDim 开始，沿正 x 轴偏移 TopXOffset 的距离（该偏移量可以为负、零或正），并具有沿正 x 轴的长度 TopXDim。
