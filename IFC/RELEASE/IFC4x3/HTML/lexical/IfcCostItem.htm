6.5.3.2 IfcCostItem

注释 该实体不是标准化模式子集或实现级别的一部分。

6.5.3.2.1 语义定义（Semantic definition）

IfcCostItem 描述了成本或财务价值以及描述其上下文的描述性信息，以便能够将其用于成本计划。IfcCostItem 可用于表示商品和服务的成本、通过流程执行的工作成本、生命周期成本等。

每个 IfcCostItem 实例都可以有一个名称和描述。根据成本的预期用途，这些值应基于协议确定。例如，Name 属性可用于提供一个通用值，以便将不同的实例组合在嵌套安排中（如下所述），而 Description 属性可用于提供成本计划中项目描述的文本。

IfcCostItem 可以链接一个或多个 IfcCostValue，这些 IfcCostValue 表示单位成本、总成本，或带有用于生成总成本的一个或多个数量的单位成本。数量可以作为单独的数量提供，或者这些数量由一个或多个建筑构件作为构件数量提供。IfcCostValue.Category 属性指示成本的类别，该类别可用于在特定列中显示该值。对于嵌套的成本项（具有 IfcRelNests 关系），IfcCostValue.Category 很重要，这样 IfcCostValue.AppliedValue 将计算为具有相同 IfcCostValue.Category 的所有嵌套成本的总和，或者如果设置为星号 ('*')，则为所有成本类型的嵌套成本的总和。IfcCostValue 可以表示原始值或由公式得出的值。例如，税款可以按总计的百分比计算。

6.5.3.2.5 形式化表示（Formal representation） 

ENTITY IfcCostItem
 SUBTYPE OF (IfcControl);
	PredefinedType : OPTIONAL IfcCostItemTypeEnum;
	CostValues : OPTIONAL LIST [1:?] OF IfcCostValue;
	CostQuantities : OPTIONAL LIST [1:?] OF IfcPhysicalQuantity;
END_ENTITY;
