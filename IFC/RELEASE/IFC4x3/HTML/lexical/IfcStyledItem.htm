8.12.3.28 IfcStyledItem

RV ⓘ
AbV ⓘ

8.12.3.28.1 语义定义（Semantic definition）

IfcStyledItem 保存产品（Products）的表示样式信息，这些信息可以是显式地为分配给产品的 IfcShapeRepresentation 的一部分的 IfcGeometricRepresentationItem 提供的，也可以是通过分配给作为产品其他表示的 IfcMaterial 来提供的。

  • 如果 IfcStyledItem 在 IfcProductDefinitionShape 的引用中使用，则必须提供一个 Item。
  • 如果 IfcStyledItem 在 IfcMaterialDefinitionRepresentation 的引用中使用，则不得提供 Item。
注释 Definition according to ISO/CD 10303-46:1992 The styled item is an assignment of style for presentation to a geometric representation item as it is used in a representation.
注释 Entity adapted from styled_item defined in ISO10303-46.
沿革 New entity in IFC2x2.
IFC2x3 变更 The attribute Item has been made optional, upward compatibility for file based exchange is guaranteed.
IFC4 变更 The subtype IfcAnnotationOccurrence and its subtypes are deleted. Use IfcStyledItem for all instantiations. The data type of Styles has been changed to IfcStyleAssignmentSelect

IFC2x2 Addendum 1 CHANGE The entity IfcStyledItem has been made non abstract and the attribute Name added.

8.12.3.28.5 形式化表示（Formal representation） 

ENTITY IfcStyledItem
 SUBTYPE OF (IfcRepresentationItem);
	Item : OPTIONAL IfcRepresentationItem;
	Styles : SET [1:?] OF IfcPresentationStyle;
	Name : OPTIONAL IfcLabel;
 WHERE
	ApplicableItem : NOT('IFC4X3_DEV_62887e3f.IFCSTYLEDITEM' IN TYPEOF(Item));
END_ENTITY;
