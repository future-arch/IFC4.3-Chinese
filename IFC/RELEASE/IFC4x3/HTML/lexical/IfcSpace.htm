5.4.3.64 IfcSpace（空间）

注释 该实体是 IfcProduct 或 IfcTypeProduct 的子类型，因此是每个标准化模式子集和实现级别的一部分。

5.4.3.64.1 语义定义（Semantic definition）

空间表示实际或理论上界定的区域或体积。空间是建筑物内为特定功能提供服务的区域或体积。

空间与建筑楼层相关联（对于外部空间则与场地相关联）。空间可以跨越多个相连的空间。因此，空间组提供了楼层中包含的空间集合。空间也可以被分解为多个部分，每个部分定义一个局部空间。这由超类型 IfcSpatialStructureElement 的 CompositionType 属性定义，其解释如下：

  • COMPLEX = 空间组
  • ELEMENT = 空间
  • PARTIAL = 局部空间
注释 视图定义和实施协议可能限制具有相同 CompositionType 的空间不能重叠。

IfcSpace 用于构建建筑物的空间结构（作为主要项目分解结构，且必须是层次化的）。空间结构元素通过使用对象化关系 IfcRelAggregates 链接在一起。

图1显示了作为空间结构一部分的 IfcSpace。它还作为空间相关元素的空间容器。

注释 关于强制性元素包含和放置结构关系的详细要求在视图定义和实施者协议中给出。

以下指南适用于使用 Name、Description、LongName 和 ObjectType 属性：

  • Name 保存平面图中的唯一名称（或空间编号）
  • Description 保存用户可能指定的任何附加信息字段，无进一步建议
  • LongName 保存空间的完整名称，通常与 Name 一起使用，如果房间分配了编号，则描述性名称作为 LongName 交换
  • ObjectType 保存空间类型，即通常的空间功能类别
注释 如果 IfcSpace 的几何表示与周围 IfcRelSpaceBoundary 的组合几何表示不一致，空间的几何表示应优先于周围空间边界的几何表示。

图2描述了 IfcSpace 的高度和标高：

  • 空间标高（结构板顶部）等于楼层标高：由 IfcBuildingStorey.Elevation 相对于 IfcBuilding.ElevationOfRefHeight 提供
  • 空间楼面标高（板上楼面层顶部）：由 IfcSpace.ElevationWithFlooring 相对于 IfcBuilding.ElevationOfRefHeight 提供
  • 空间高度（下方板顶部到上方板底部）：由名称为"Height"的BaseQuantity提供
  • 空间楼层高度（下方板顶部到楼面层顶部）：由名称为"FinishFloorHeight"的BaseQuantity提供
  • 空间净高度（楼面层顶部到吊顶底部）：由名称为"FinishCeilingHeight"的BaseQuantity提供

5.4.3.64.6 概念用法（Concept usage）

5.4.3.64.7 形式化表示（Formal representation） 

ENTITY IfcSpace
 SUBTYPE OF (IfcSpatialStructureElement);
	PredefinedType : OPTIONAL IfcSpaceTypeEnum;
	ElevationWithFlooring : OPTIONAL IfcLengthMeasure;
 INVERSE
	HasCoverings : SET [0:?] OF IfcRelCoversSpaces FOR RelatingSpace;
	BoundedBy : SET [0:?] OF IfcRelSpaceBoundary FOR RelatingSpace;
 WHERE
	CorrectPredefinedType : NOT(EXISTS(PredefinedType)) OR
 (PredefinedType <> IfcSpaceTypeEnum.USERDEFINED) OR
 ((PredefinedType = IfcSpaceTypeEnum.USERDEFINED) AND EXISTS (SELF\IfcObject.ObjectType));
	CorrectTypeAssigned : (SIZEOF(IsTypedBy) = 0) OR
  ('IFC4X3_DEV_62887e3f.IFCSPACETYPE' IN TYPEOF(SELF\IfcObject.IsTypedBy[1].RelatingType));
END_ENTITY;
