5.1.3.5 IfcGroup（组）

RV ⓘ
AbV ⓘ

5.1.3.5.1 语义定义（Semantic definition）

IfcGroup 是对任意组的泛化。组是对象的逻辑集合，它不具有自身的位置，也不能持有自身的形状表达。因此，组是在某些非几何/拓扑分组方面的聚合。

示例 组的示例之一是系统，因为它根据元素的角色方面进行分组，而不管它们在建筑中的位置。代表系统的最重要组用途之一是 IfcDistributionSystem，它将分布组件（如空间加热器和阀门）分组到供暖系统中。

组可以容纳任何对象集合，关系 IfcRelAssignsToGroup 用于建立组集合。组内的对象可以是产品、过程、控制、资源、参与者或其他组，因此组可以嵌套。一个对象可以是零个、一个或多个组的一部分。分组关系不需要是分层的，也不意味着依赖关系。

可以使用关系 IfcRelReferencedInSpatialStructure 在空间结构中引用组，例如引用建筑的机械分布系统。

通过引用相应对象的关系对象，将组分配给其他对象（如过程或资源）：

  • 过程：使用 IfcRelAssignsToProcess 分配
  • 资源：使用 IfcRelAssignsToResource 分配
  • 控制：使用 IfcRelAssignsToControl 影响组

组可以在其组集合中尚未包含对象的情况下进行交换。

IfcGroup 不定义自身的对象坐标系，也不具有独立的形状表达。

注释 将 IfcRelAggregates 与 IfcProduct 的适当子类型一起使用，以定义可能具有自身位置和形状表达的产品聚合。此关系应用于创建产品分解结构。
沿革 IFC1.0 中的新实体。
IFC4 变更 反向关系 IsGroupedBy 设置为 0..n

5.1.3.5.5 概念用法（Concept usage）

5.1.3.5.6 形式化表示（Formal representation） 

ENTITY IfcGroup
 SUPERTYPE OF (ONEOF
	(IfcAsset
	,IfcInventory
	,IfcStructuralLoadGroup
	,IfcStructuralResultGroup
	,IfcSystem))
 SUBTYPE OF (IfcObject);
 INVERSE
	IsGroupedBy : SET [0:?] OF IfcRelAssignsToGroup FOR RelatingGroup;
	ReferencedInStructures : SET [0:?] OF IfcRelReferencedInSpatialStructure FOR RelatedElements;
END_ENTITY;
