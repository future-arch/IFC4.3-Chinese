5.1.3.5 IfcGroup

RV ⓘ
AbV ⓘ

5.1.3.5.1 语义定义（Semantic definition）

IfcGroup 是任意组的泛化。组是对象的逻辑集合。它没有自己的位置，也不能保持自己的形状表示。因此，组是在某些非几何/拓扑分组方面的聚合。

示例 组的一个例子是系统，因为它根据元素的角色方面对元素进行分组，而不管它们在建筑中的位置。表示组的系统最重要的用途之一是_IfcDistributionSystem_，它将分布组件（如空间加热器和阀门）分组到供暖系统中。

组可以包含任何对象集合，使用关系_IfcRelAssignsToGroup_来建立组集合。组内的对象可以是产品、流程、控制、资源、参与者或其他组，因此组可以嵌套。一个对象可以是零个、一个或多个组的一部分。分组关系不需要是分层的，也不意味着依赖关系。

可以使用关系_IfcRelReferencedInSpatialStructure_在空间结构中引用组，例如引用建筑物的机械分布系统。

通过引用相应对象的关系对象将组分配给其他对象（如流程或资源）：

  • 流程：使用_IfcRelAssignsToProcess_分配
  • 资源：使用_IfcRelAssignsToResource_分配
  • 控制：使用_IfcRelAssignsToControl_影响组

可以在组集合中还没有对象的情况下交换组。

_IfcGroup_不定义自己的对象坐标系，也没有独立的形状表示。

沿革 IFC1.0中的新实体。

注意 使用_IfcRelAggregates_与_IfcProduct_的适当子类型一起定义可能具有自己位置和形状表示的产品聚合。此关系应用于创建产品分解结构。

IFC4变更 反向_IsGroupedBy_关系设置为0..n

5.1.3.5.5 概念用法（Concept usage）

5.1.3.5.6 形式化表示（Formal representation） 

ENTITY IfcGroup
 SUPERTYPE OF (ONEOF
	(IfcAsset
	,IfcInventory
	,IfcStructuralLoadGroup
	,IfcStructuralResultGroup
	,IfcSystem))
 SUBTYPE OF (IfcObject);
 INVERSE
	IsGroupedBy : SET [0:?] OF IfcRelAssignsToGroup FOR RelatingGroup;
	ReferencedInStructures : SET [0:?] OF IfcRelReferencedInSpatialStructure FOR RelatedElements;
END_ENTITY;
