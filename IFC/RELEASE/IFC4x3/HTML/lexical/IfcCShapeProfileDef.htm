8.15.3.5 IfcCShapeProfileDef

注释 该实体不是标准化模式子集或实现级别的一部分。

8.15.3.5.1 语义定义（Semantic definition）

IfcCShapeProfileDef 定义了一个截面轮廓，它提供了 C 形截面的定义参数，用于扫掠区域实体。这种截面通常由冷弯钢生产。其参数和相对于位置坐标系的定向符合以下图示。位置坐标系的中心位于轮廓边界框的中心。

沿革 IFC2x2 中的新实体。
IFC2x3 变更 所有轮廓原点现在位于边界框的中心。
IFC4 变更 InternalFilletRadius 的类型已放宽，允许零半径。删除了尾部属性 CentreOfGravityInX，请改用 IfcProfileProperties 中的相应属性。

图 8.15.3.5.A 说明了 C 形截面定义的参数。参数化轮廓定义了自己的位置坐标系。底层坐标系由使用轮廓定义的扫掠区域实体定义。它是：

  • IfcSweptAreaSolid.Position

的 xy 平面。通过使用位置偏移，参数化轮廓可以居中放置（使用 x,y 偏移量 = 0.），或相对于轮廓的任何位置。参数化轮廓由一组参数属性定义。在所示示例中，IfcProfileProperties 中的“CentreOfGravityInX”属性（如果提供）为负值。

8.15.3.5.7 形式化表示（Formal representation） 

ENTITY IfcCShapeProfileDef
 SUBTYPE OF (IfcParameterizedProfileDef);
	Depth : IfcPositiveLengthMeasure;
	Width : IfcPositiveLengthMeasure;
	WallThickness : IfcPositiveLengthMeasure;
	Girth : IfcPositiveLengthMeasure;
	InternalFilletRadius : OPTIONAL IfcNonNegativeLengthMeasure;
 WHERE
	ValidGirth : Girth < (Depth / 2.);
	ValidInternalFilletRadius : NOT(EXISTS(InternalFilletRadius)) OR
((InternalFilletRadius <= Width/2. - WallThickness) AND (InternalFilletRadius <= Depth/2. - WallThickness));
	ValidWallThickness : (WallThickness < Width/2.) AND (WallThickness < Depth/2.);
END_ENTITY;
