8.6.3.1 IfcClassification

RV ⓘ
AbV ⓘ

8.6.3.1.1 语义定义（Semantic definition）

IfcClassification 用于根据共同的目的或共同的特征将对象排列到一个类或类别中。IfcClassification 意义上的分类是分类法或分类体系，以分层结构排列。对象的一个类别与其他类别在泛化-特化关系中相关联。因此，分类中的分类项以树状结构组织。

IfcClassification 标识分类引用所指向的分类系统或源。每个分类引用都可以引用 IfcClassification 的一个实例。声明的分类系统可以是正式发布的，也可以是本地定义的分类信息的方法。

注释 Examples for such formally published classifications are Omniclass, Uniclass, Masterformat, or DIN277.

有两种方法可以定义和引用分类系统：

  1. 在数据集中包含分类系统结构：此处包含一个 IfcClassificationReference 的分层树，该树定义了分类系统，包括分类项之间的关系。IfcClassificationReference 的 ReferencedSource 特性将分类项链接到父项，父项最终链接到 IfcClassification。
  2. 通过分类键或 ID 引用分类系统：此处使用 IfcClassificationReference 作为 Identification 特性来分配分类 ID 或键，并且它可以通过 ReferencedSource 特性直接链接到 IfcClassification。

IFC 2x4 CHANGE Attribute Edition made optional. Attributes: Description, Location, and ReferenceTokens and inverse attribute HasReferences added.

8.6.3.1.3 特性（Attributes）

8.6.3.1.4 形式化表示（Formal representation） 

ENTITY IfcClassification
 SUBTYPE OF (IfcExternalInformation);
	Source : OPTIONAL IfcLabel;
	Edition : OPTIONAL IfcLabel;
	EditionDate : OPTIONAL IfcDate;
	Name : IfcLabel;
	Description : OPTIONAL IfcText;
	Specification : OPTIONAL IfcURIReference;
	ReferenceTokens : OPTIONAL LIST [1:?] OF IfcIdentifier;
 INVERSE
	ClassificationForObjects : SET [0:?] OF IfcRelAssociatesClassification FOR RelatingClassification;
	HasReferences : SET [0:?] OF IfcClassificationReference FOR ReferencedSource;
END_ENTITY;
