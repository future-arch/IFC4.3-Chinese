5.1.3.17 IfcPropertyTemplate（属性模板）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.1.3.17.1 语义定义（Semantic definition）

IfcPropertyTemplate 是一个抽象超类型，包含所有动态可扩展属性的模板，可以是 IfcComplexPropertyTemplate，也可以是 IfcSimplePropertyTemplate。这些模板决定了以下结构：

  • 对于 IfcComplexPropertyTemplate 的情况
  • 一个 IfcComplexProperty
  • 一个 IfcPhysicalComplexQuantity
  • 对于 IfcSimplePropertyTemplate 的情况
  • IfcSimpleProperty 的子类型
  • IfcPhysicalSimpleQuantity 的子类型

各个属性模板根据其 Name 属性进行解释，并可能具有预定义的属性类型、属性单位和属性数据类型。但模板不应具有已分配的测量值或数量值。

注释 属性模板可以构成属性库的一部分，作为项目库的一部分使用和附加。通常，包含 IfcPropertyTemplate 子类型的 IfcPropertySetTemplate 将直接链接到 IfcProjectLibrary。
沿革 IFC4 中的新实体。

5.1.3.17.5 形式化表示（Formal representation） 

ENTITY IfcPropertyTemplate
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcComplexPropertyTemplate
	,IfcSimplePropertyTemplate))
 SUBTYPE OF (IfcPropertyTemplateDefinition);
 INVERSE
	PartOfComplexTemplate : SET [0:?] OF IfcComplexPropertyTemplate FOR HasPropertyTemplates;
	PartOfPsetTemplate : SET [0:?] OF IfcPropertySetTemplate FOR HasPropertyTemplates;
END_ENTITY;
