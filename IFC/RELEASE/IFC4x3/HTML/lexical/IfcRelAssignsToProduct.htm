5.1.3.27 IfcRelAssignsToProduct（关联关系-分配给产品）

注释 该实体不是标准化模式子集或实现级别的一部分。

5.1.3.27.1 语义定义（Semantic definition）

对象化关系 IfcRelAssignsToProduct 处理对象（IfcObject 的子类型）到产品（IfcProduct 的子类型）的分配关系。Name 属性应用于分类 IfcRelAssignsToProduct 对象化关系的使用方式。建议采用以下 Name 取值：

  • 'Context'：分配特定上下文的表达形式，例如将结构构件分配到不同上下文表达（可能具有不同的分解结构），如针对特定上下文的建筑构件表达。
  • 'View'：将通过 RelatingProduct 关联的按专业视图分解的产品，分配到通过 RelatedObjects 关联的按不同专业视图分解的另一产品。例如将建筑楼板分配到楼板预制构件（在预制混凝土专业视图下）的不同分解结构中。
沿革 IFC2x 中新增实体
IFC2x3 变更 空间结构内产品的引用现由新的关系对象 IfcRelReferencedInSpatialStructure 处理。从 IFC2x3 开始，IfcRelAssignsToProduct 不得用于表示此关系。

5.1.3.27.6 形式化表示（Formal representation） 

ENTITY IfcRelAssignsToProduct
 SUBTYPE OF (IfcRelAssigns);
	RelatingProduct : IfcProductSelect;
 WHERE
	NoSelfReference : SIZEOF(QUERY(Temp <* SELF\IfcRelAssigns.RelatedObjects | RelatingProduct :=: Temp)) = 0;
END_ENTITY;
