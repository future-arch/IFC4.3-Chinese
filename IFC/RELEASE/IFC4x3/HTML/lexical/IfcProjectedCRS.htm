8.18.3.11 IfcProjectedCRS

RV ⓘ
AbV ⓘ

8.18.3.11.1 语义定义（Semantic definition）

IfcProjectedCRS 是地图的坐标参考系统 (CRS)，局部施工或设施工程项目的地图转换与此 CRS 相关。假定投影坐标参考系统是 2D 或 3D 右手笛卡尔坐标系，可选的 MapUnit 属性可用于确定地图使用的长度单位。

尽管其名称暗示了这一点：

  • 如果引用的 IfcGeometricRepresentationContext 是 3D，则 IfcProjectedCRS 应为复合坐标参考系统，即多个 CRS 的组合，从中可以始终明确无误地识别 GeodeticDatum 和 VerticalDatum。

  • 如果引用的 IfcGeometricRepresentationContext 是 2D，则 IfcProjectedCRS 可以是投影坐标参考系统或复合坐标参考系统。

示例 The code EPSG:9286 (ETRS89 + NAP height) is the combination of a geographic CRS (ETRS89) and a vertical CRS (NAP height), with their respective datums.
示例 The code EPSG:9306 (HS2 Survey Grid + HS2-VRF height) is the combination of a projected CRS (HS2 Survey Grid), and a vertical CRS (HS2-VRF height), with their respective datums.

坐标参考系统所知的明确标识符存储在继承的 Name 属性中。定义良好的标识符包括地图投影、地图区域信息以及所有必需的基准面。在这些情况下，可以省略 VerticalDatum、MapProjection、MapZone 以及继承的属性 GeodeticDatum。

示例 The code 'EPSG:5555' identifies the combination of ETRS89 / UTM zone 32N + DHHN92 height (i.e., a projected CRS + a vertical CRS). The code 'EPSG:6258' identifies the geodetic datum of the projected CRS ETRS89 / UTM zone 32N (itself identified as EPSG:25832). The code 'EPSG:5181' identifies the vertical datum of the vertical CRS DHHN92 height (itself identified as EPSG:5783).
注释 Definition from OpenGIS Abstract Specification, Topic 2: A 2D (or with vertical coordinate axis 3D) coordinate reference system used to approximate the shape of the earth on a planar surface, but in such a way that the distortion that is inherent to the approximation is carefully controlled and known. Distortion correction is commonly applied to calculated bearings and distances to produce values that are a close match to actual field values.

8.18.3.11.5 形式化表示（Formal representation） 

ENTITY IfcProjectedCRS
 SUBTYPE OF (IfcCoordinateReferenceSystem);
	VerticalDatum : OPTIONAL IfcIdentifier;
	MapProjection : OPTIONAL IfcIdentifier;
	MapZone : OPTIONAL IfcIdentifier;
	MapUnit : OPTIONAL IfcNamedUnit;
 WHERE
	MapUnitIsLength : NOT(EXISTS(SELF.MapUnit)) OR (SELF.MapUnit.UnitType = IfcUnitEnum.LENGTHUNIT);
END_ENTITY;
