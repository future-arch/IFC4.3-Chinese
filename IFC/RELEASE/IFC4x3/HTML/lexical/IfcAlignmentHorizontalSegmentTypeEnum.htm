8.7.2.2 IfcAlignmentHorizontalSegmentTypeEnum

8.7.2.2.1 语义定义（Semantic definition）

IfcAlignmentHorizontalSegmentTypeEnum 指示了水平对齐段 (IfcAlignmentHorizontalSegment) 的一个段的类型。从几何角度和运动学角度都可以看待水平段。近年来，运动学角度变得越来越重要。枚举值根据这一发展进行了详细说明，尤其是在现代轨道设计中。

水平对齐段的运动学视角

运动学视角的核心参数是车辆因方向改变而产生的横向加速度。在水平布局中，这由段的曲率表示。根据曲率值，可以进行以下分类：

水平对齐段的几何视角

传统观点在与 IfcAlignment 文档相关的业务术语的背景下被描述。在现代计算机可用之前，对齐设计是使用“传统”绘图技术完成的。在计算机化初期，这种起源导致首先在 x,y 空间中表示，然后第二步检查安全相关属性。这仍然可以在 1980 年或更早生效的规定中看到。当然，基于这些规定设计的图纸反映了文档精度中的“足够好”的态度。

在后期，可以看到运动学视角的重要性日益增加。在此，精确控制横向加速度（水平和纵坡布局）和垂直加速度（垂直布局）变得普遍。设计者开始使用高性能过渡曲线，尤其是在高速场景中。在运动学视角中，连续段之间的精确曲率拟合需要比传统几何视角中的“足够好”方法更好。核心术语例如“加加速度”、“理论纵坡”或“纵坡不足”。

给定曲率的内在 x, y 坐标的通用计算

对于具有已知曲率公式的每个水平对齐段，存在一个计算段内在坐标的通用方法。

$$\displaylines{ \varphi(s) = \int d \varphi = \int \kappa(s) \ d s \\ x(s) = \int cos \ \varphi(s) \ d s = \int cos( \int \kappa(s) \ d s \ ) d s \\ y(s) = \int sin \ \varphi(s) \ d s = \int \sin( \int \kappa(s) \ d s\ ) d s }$$

注释 While it is possible to apply the generic calculation also for trivial cases like LINE or CIRCULARARC it is much more efficient to use available formulas.

警告

在将“足够好”的传统设计纳入高精度 3D 模型之前，必须仔细检查。可能需要进行中间校正。幸运的是，曲线在经典几何视角和更新的运动学视角中都与可比的文档质量配合得非常好。幸运的是，绝大多数水平过渡曲线被设计和实现为曲线。

建议

检查相关网络的相关规定。对齐设计本身在道路或轨道的使用寿命内非常稳定。特别是对于旧设计，必须非常仔细地检查可用文档的质量和精度。在实现高精度 BIM 环境和遗留文档系统之间自动化数据流之前，应建立对局限性的清晰理解。这既适用于遗留的中央数据库，也适用于遗留的单个文档。
