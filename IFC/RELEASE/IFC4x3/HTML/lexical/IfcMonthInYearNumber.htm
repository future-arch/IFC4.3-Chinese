8.5.2.7 IfcMonthInYearNumber（年中月数）

8.5.2.7.1 语义定义（Semantic definition）

IfcMonthInYearNumber 是一个整数，用于定义指定月份在一年中的位置。

日历月份数字与日历月份名称的对应关系如下：

  • 1: 一月
  • 2: 二月
  • 3: 三月
  • 4: 四月
  • 5: 五月
  • 6: 六月
  • 7: 七月
  • 8: 八月
  • 9: 九月
  • 10: 十月
  • 11: 十一月
  • 12: 十二月

类型：INTEGER

注释 参见 ISO 8601 中定义的日历月的定义。
沿革 IFC1.5.1 中新增的类型。

8.5.2.7.2 形式化命题（Formal propositions）

8.5.2.7.3 形式化表示（Formal representation） 

TYPE IfcMonthInYearNumber = INTEGER;
 WHERE
	 ValidRange : {1 <= SELF <= 12};
END_TYPE;
