5.1.3.9 IfcProcess（过程）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.1.3.9.1 语义定义（Semantic definition）

IfcProcess 被定义为在时间上有序的单个活动或事件，它与其他过程具有顺序关系，将输入转化为输出，并可通过输入输出关系与其他过程相连接。一个 IfcProcess 可以是一项活动（或任务），也可以是一个事件。它通常发生在建筑工程中，旨在设计、成本核算、采购、建造或维护产品及其他类似任务或程序。图1展示了过程关系。

注释 根据ISO9000的定义： 过程是一组相互关联或相互作用的活动。过程利用资源将输入转化为输出。过程之间相互连接，因为一个过程的输出成为另一个过程的输入。实际上，过程通过这种输入输出关系"粘合"在一起。

过程信息通过建立以下关系与其他对象相关联：

  • 过程的嵌套：IfcRelNests - 一个过程可以包含子过程，从而实现嵌套。
  • 过程的排序：IfcRelSequence - 过程可以按顺序排列（包括并行任务的重叠），并具有前驱和后继。
  • 将过程分配给进度表：IfcRelAssignsToControl - 活动（如任务，主要是摘要任务）被分配给工作进度表。
  • 将产品作为输入分配给过程：IfcRelAssignsToProcess - 产品可以作为输入分配给过程，例如用于施工过程规划。
  • 将产品作为输出分配给过程：IfcRelAssignsToProduct - 产品可以作为输出分配给过程，例如用于施工过程规划。
  • 将控制项作为过程控制分配给过程：IfcRelAssignsToProcess - 作为过程控制项的项目可以分配给过程，例如用于成本管理（将成本项分配给工作任务）。
  • 将资源作为过程消耗分配给过程：IfcRelAssignsToProcess - 作为过程机制的项目，例如成本计算中的劳动力、材料和设备。
沿革 IFC1.0中的新实体。
IFC4 变更 属性 Identification 已从子类型 IfcTask 及其他类型中提升。

IFC2x变更 属性 Productivity 已被移除。

5.1.3.9.5 概念用法（Concept usage）

5.1.3.9.6 形式化表示（Formal representation） 

ENTITY IfcProcess
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcEvent
	,IfcProcedure
	,IfcTask))
 SUBTYPE OF (IfcObject);
	Identification : OPTIONAL IfcIdentifier;
	LongDescription : OPTIONAL IfcText;
 INVERSE
	IsPredecessorTo : SET [0:?] OF IfcRelSequence FOR RelatingProcess;
	IsSuccessorFrom : SET [0:?] OF IfcRelSequence FOR RelatedProcess;
	OperatesOn : SET [0:?] OF IfcRelAssignsToProcess FOR RelatingProcess;
END_ENTITY;
