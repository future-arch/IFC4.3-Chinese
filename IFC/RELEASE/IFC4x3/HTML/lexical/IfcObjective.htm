8.3.3.3 IfcObjective

注释 该实体不是标准化模式子集或实现级别的一部分。

8.3.3.3.1 语义定义（Semantic definition）

IfcObjective 捕获基于目标约束的定性信息。

IfcObjective 是 IfcConstraint 的子类型，可以通过 IfcControlExtension schema 中的 IfcRelAssociatesConstraint 关系与任何 IfcRoot 的子类型相关联，或者通过 IfcResourceConstraintRelationship 与 IfcProperty 相关联。

IfcObjective 的目的是指定约束应用的用途，并捕获约束的基准指标。

沿革 New entity in IFC2.0.
IFC4 变更 Attribute BenchmarkValues modified to be a LIST of IfcConstraint, attribute ResultValues replaced with IfcLogicalOperatorEnum.

8.3.3.3.5 形式化表示（Formal representation） 

ENTITY IfcObjective
 SUBTYPE OF (IfcConstraint);
	BenchmarkValues : OPTIONAL LIST [1:?] OF IfcConstraint;
	LogicalAggregator : OPTIONAL IfcLogicalOperatorEnum;
	ObjectiveQualifier : IfcObjectiveEnum;
	UserDefinedQualifier : OPTIONAL IfcLabel;
 WHERE
	WR21 : (ObjectiveQualifier <> IfcObjectiveEnum.USERDEFINED) OR
((ObjectiveQualifier = IfcObjectiveEnum.USERDEFINED) AND EXISTS(SELF\IfcObjective.UserDefinedQualifier));
END_ENTITY;
