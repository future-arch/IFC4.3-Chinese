5.1.3.15 IfcPropertySetDefinition（属性集定义）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.1.3.15.1 语义定义（Semantic definition）

IfcPropertySetDefinition 是所有可分配给对象或类型对象的独立属性集的泛化。属性集定义可以是以下两种类型之一：

  • 动态可扩展属性集 - IfcPropertySet，一种属性集合，IFC规范仅为其提供一种"元模型"，需通过协议进一步声明。这意味着IFC规范中不存在这些属性的实体定义。声明是通过为实体 IfcPropertySet 中定义的 Name 属性以及被属性集引用的 IfcProperty 的每个子类型的 Name 属性分配一个有意义的字符串值来完成的。动态定义的属性集可能具有由 IfcPropertySetTemplate 提供的基础模板。
  • 静态定义属性集 - IfcPreDefinedPropertySet，一种在IFC规范中存在的属性集实体。每个静态定义属性集的语义由其实体类型声明，属性的含义由代表它的显式属性的名称和数据类型定义。

属性集定义定义了在对象的多个实例（无论是对象实例还是对象类型）之间共享的信息。IfcPropertySetDefinition（通过其可实例化的子类型）可以参与以下关系：

  • 分配给对象类型 - 与 IfcTypeObject 的 DefinesType 直接关系，将属性集及其所有包含的属性应用于对象类型。这些属性适用于所有具有相同对象类型的对象实例。
  • 分配给对象实例 - 与 IfcRelDefinesByProperties 的 DefinesOccurrence 关系，将属性集及其所有包含的属性应用于对象实例。
注释 分配给对象实例的属性可能会覆盖分配给对象类型的属性。有关更多信息，请参见 IfcRelDefinesByType。
沿革 IFC2x 中的新实体
IFC4 变更 添加了子类型 IfcPreDefinedPropertySet。

5.1.3.15.5 形式化表示（Formal representation） 

ENTITY IfcPropertySetDefinition
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcPreDefinedPropertySet
	,IfcPropertySet
	,IfcQuantitySet))
 SUBTYPE OF (IfcPropertyDefinition);
 INVERSE
	DefinesType : SET [0:?] OF IfcTypeObject FOR HasPropertySets;
	IsDefinedBy : SET [0:?] OF IfcRelDefinesByTemplate FOR RelatedPropertySets;
	DefinesOccurrence : SET [0:?] OF IfcRelDefinesByProperties FOR RelatingPropertyDefinition;
END_ENTITY;
