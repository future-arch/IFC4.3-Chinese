8.8.3.35 IfcSectionedSolidHorizontal

AbV ⓘ

8.8.3.35.1 语义定义（Semantic definition）

IfcSectionedSolidHorizontal 是一个实体模型，通过沿水平曲线扫掠可能变化的横截面来构造。

该实体是通过在 CrossSectionPositions 之间扫掠 CrossSections 生成的，在具有相同标签的轮廓点之间进行线性插值。轮廓法线从 IfcAxis2PlacementLinear 导出，不一定是 Directrix 的切线。轮廓的 X 轴是 IfcAxis2PlacementLinear 的 RefDirection 方向，轮廓的 Y 轴是 Axis 方向。

对于根据单一超高旋转的横截面，可以使用 IfcDerivedProfileDef 来指示这种旋转，其中每个 ParentProfile 都引用相同的底层轮廓。对于根据多个超高变换的横截面，其中点独立变化，每个轮廓可以是不同实例但类型相同（例如 IfcArbitraryClosedProfileDef），并且可以选择通过匹配标签（使用 IfcIndexedPolyCurve 和 IfcCartesianPointList2D.TagList）将横截面点与样条线（“引导曲线”）关联起来。

图 8.8.3.35.A 说明了具有恒定参数化轮廓的桥梁的四个桥墩，具有恒定任意轮廓的两个护栏（其中一个被镜像），以及具有可变任意轮廓的桥面板。

Informal Propositions

  1. 任何两个连续的截面不应相交。
  2. 如果截面的类型不是 IfcParameterizedProfileDef（即任意轮廓），则两个连续轮廓的点和边的数量应相同。
  3. 如果导曲线不是切线连续的，则生成的实体是通过两个段之间的半角斜接形成的。
  4. 非常尖锐的边可能导致几乎不可能的斜接；实施协议可以定义切线不连续的可接受限制，或要求导曲线是切线连续的。
  5. 导曲线不应相交。

8.8.3.35.5 形式化表示（Formal representation） 

ENTITY IfcSectionedSolidHorizontal
 SUBTYPE OF (IfcSectionedSolid);
	CrossSectionPositions : LIST [2:?] OF IfcAxis2PlacementLinear;
 WHERE
	CorrespondingSectionPositions : SIZEOF(CrossSections) = SIZEOF(CrossSectionPositions);
	NoLongitudinalOffsets : SIZEOF(QUERY(temp <* CrossSectionPositions | EXISTS(temp.Location.OffsetLongitudinal))) = 0;
END_ENTITY;
