8.7.3.14 IfcLocalPlacement

RV ⓘ
AbV ⓘ

8.7.3.14.1 语义定义（Semantic definition）

IfcLocalPlacement 定义了产品相对于另一个产品的位置的相对放置，或产品在项目几何表示上下文中的绝对放置。

IfcLocalPlacement 允许 IfcProduct 通过此 IfcLocalPlacement（通过 ObjectPlacement 属性）放置在另一个 IfcProduct 的对象放置的局部坐标系中，该对象放置由 PlacementRelTo 引用。防止循环相对放置的规则必须在应用程序级别引入。

如果未给出 PlacementRelTo，则 IfcProduct 在世界坐标系中绝对放置。

如果使用相对放置，则以下约定应作为默认相对位置适用。约定适用于 IfcProduct 的所有五个直接子类型：IfcSpatialStructureElement、IfcElement、IfcAnnotation、IfcGrid、IfcPort。更详细的放置信息在上述五个类型的子类型级别提供。

  • 对于 IfcSpatialStructureElement 的子类型，适用以下约定
  • IfcSite 应在 IfcProject 的几何表示上下文所建立的世界坐标系中绝对放置
  • IfcFacility 应相对于 IfcSite 的局部放置进行放置
  • IfcFacilityPart 应相对于 IfcFacility 的局部放置进行放置
  • 对于 IfcGrid 和 IfcAnnotation，适用以下约定，即它们应相对于
  • 其容器的局部放置，该容器是 IfcSite、IfcBuilding 或 IfcBuildingStorey
    • 它应该是通过 IfcRelContainedInSpatialStructure 包含关系引用的同一个容器元素，
  • 对于 IfcAlignment，相对于 IfcProject 的世界坐标系进行放置。
  • 对于 IfcPort，适用以下约定，即它应相对于
  • 其所属 IfcElement 的局部放置
    • 它应该是通过 IfcRelConnectsPortToElement 连接关系引用的同一个元素，
  • 对于 IfcElement，适用以下约定，即它应相对于：
  • 其容器的局部放置，该容器是 IfcSite、IfcFacility 或 IfcFacilityPart
    • 它应该是通过 IfcRelContainedInSpatialStructure 包含关系引用的同一个容器元素，
  • 其所属的 IfcElement 的局部放置，该元素通过元素组合关系连接
    • 对于相对于主组件定位的特征（如开口），通过 IfcRelVoidsElement、IfcRelProjectsElement 和 IfcRelAdheresToElement 表示；
    • 对于填充开口的元素（如门或窗），通过 IfcRelFillsElement 表示；
    • 对于覆盖元素的覆盖层，通过 IfcRelCoversBldgElements 表示；
    • 对于聚合到主组件的子组件，通过 IfcRelAggregates 和 IfcRelNests 表示。
沿革 New entity in IFC1.0.

8.7.3.14.5 形式化表示（Formal representation） 

ENTITY IfcLocalPlacement
 SUBTYPE OF (IfcObjectPlacement);
	RelativePlacement : IfcAxis2Placement;
 WHERE
	WR21 : IfcCorrectLocalPlacement(RelativePlacement, PlacementRelTo);
END_ENTITY;
