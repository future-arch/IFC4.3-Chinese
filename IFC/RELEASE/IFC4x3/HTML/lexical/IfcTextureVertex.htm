8.12.3.45 IfcTextureVertex

注释 该实体不是标准化模式子集或实现级别的一部分。

8.12.3.45.1 语义定义（Semantic definition）

IfcTextureVertex 是一个包含 2 个（S, T）纹理坐标的列表。

注释 根据 ISO/IEC 19775-1 的定义：
沿革 IFC2x2 中的新实体。

每个基于顶点的几何节点都使用一组二维纹理坐标，将纹理映射到顶点。纹理贴图值（ImageTexture、PixelTexture）在 S 轴和 T 轴上的范围是 [0.0, 1.0]。但是，纹理坐标值可能在 (-∞,∞) 的范围内。纹理坐标标识纹理贴图中的一个位置（因此是一个颜色值）。水平坐标 S 在前，然后是垂直坐标 T。如果纹理贴图在给定方向（S 轴或 T 轴）上重复，则纹理坐标 C（s 或 t）映射到具有 N 个像素的纹理贴图，如下所示： 

纹理贴图位置 = (C - floor(C)) × N

如果纹理贴图不重复，则纹理坐标将被限制在 0.0 到 1.0 的范围内，如下所示： 

纹理贴图位置 = N,   如果 C > 1.0,

           = 0.0,  如果 C < 0.0,

           = C × N, 如果 0.0 ≤ C ≤ 1.0.

注意：纹理坐标可以通过提供纹理定义中的一个组件 TextureTransform 来进行变换（缩放、旋转、平移）。
