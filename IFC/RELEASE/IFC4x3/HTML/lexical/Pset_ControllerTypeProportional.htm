7.2.4.15 Pset_ControllerTypeProportional（控制器类型-比例）

7.2.4.15.1 语义定义（Semantic definition）

用于比例控制器的信号处理，该控制器接收设定点和反馈输入，并产生单一值的输出。

HISTORY: IFC4 中，SignalFactor1、SignalFactor2 和 SignalFactor3 更改为 ProportionalConstant、IntegralConstant 和 DerivativeConstant。SignalTime1 和 SignalTime2 更改为 SignalTimeIncrease 和 SignalTimeDecrease。

7.2.4.15.2 适用实体

PSET_TYPEDRIVENOVERRIDE None
  • IfcController/PROPORTIONAL
  • IfcControllerType/PROPORTIONAL
