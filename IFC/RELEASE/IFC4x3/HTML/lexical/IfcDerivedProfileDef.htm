8.15.3.10 IfcDerivedProfileDef

注释 该实体不是标准化模式子集或实现级别的一部分。

8.15.3.10.1 语义定义（Semantic definition）

IfcDerivedProfileDef 通过从父剖面进行变换来定义剖面。变换由二维变换算子给出。变换包括平移、旋转、镜像和缩放。后者可以是均匀的或非均匀的。派生剖面可用于定义扫掠曲面、扫掠实体或分段脊线。

变换会影响剖面在底层坐标系中的位置、旋转、镜像或缩放，即由使用剖面定义的扫掠曲面或扫掠实体定义的坐标系。它是以下任一坐标系的 xy 平面：

  • IfcSweptSurface.Position
  • IfcSweptAreaSolid.Position

或者在分段脊线的情况下，是 IfcSectionedSpine.CrossSectionPositions 的每个列表成员的 xy 平面。在应用笛卡尔变换算子之前，会考虑父剖面在底层坐标系中的位置和潜在旋转。

注释，如果只需要镜像，则应使用 IfcMirroredProfileDef 而不是 IfcDerivedProfileDef。

沿革 IFC2x 中的新实体。

参数
IfcDerivedProfileDef 使用 IfcCartesianTransformationOperator2D (CTO) 定义，该算子应用于父剖面定义。

示例
该示例显示了一个均匀缩放和对 IfcRectangleProfileDef 的变换，以匹配左下角基点。CTO 的属性为：

Axis1 = NIL (默认为 1.,0.) Axis2 = NIL (默认为 0.,1.) LocalOrigin = IfcCartesianPoint(<1/2 XDim>,<1/2 YDim>) Scale = 2.

注释 ParentProfile 已具有 Position = IfcCartesianPoint(<1/2 XDim>,<1/2 YDim>)。

参数
IfcDerivedProfileDef 通过应用 IfcCartesianTransformationOperator2DnonUniform 作为 2D CTO 的子类型来定义非均匀变换。

示例
该示例显示了一个非均匀缩放和对 IfcRectangleProfileDef 的平移，以匹配左下角基点。CTO 的属性为：

Axis1 = NIL (默认为 1.,0.) Axis2 = NIL (默认为 0.,1.) LocalOrigin = IfcCartesianPoint(0.,<1/2 YDim>) Scale = 1. Scale2 = 2.

注释 ParentProfile 已具有 Position = IfcCartesianPoint(<1/2 XDim>,<1/2 YDim>)。

参数
IfcDerivedProfileDef 通过将 IfcCartesianTransformationOperator2D (CTO) 应用于父剖面来定义镜像。

示例
该示例显示了一个对 IfcLShapeProfileDef 的镜像，以匹配中心基点。CTO 的属性为：

Axis1 = (-1.,0.) Axis2 = NIL (默认为 0.,1.) LocalOrigin = IfcCartesianPoint(0.,0.) Scale = NIL (默认为 1.)

注释 ParentProfile 已具有 Position = IfcCartesianPoint(0.,0.)。

此示例仅用于说明。 如果变换仅导致镜像，如示例所示，则应使用 IfcMirroredProfileDef 而不是 IfcDerivedProfileDef。

注意：以下颜色映射适用：

  • 黑色坐标轴显示扫掠曲面、扫掠实体或分段脊线的底层坐标系
  • 红色坐标轴显示父剖面的位置坐标系
  • 棕色坐标轴显示派生剖面的位置坐标系

8.15.3.10.7 形式化表示（Formal representation） 

ENTITY IfcDerivedProfileDef
 SUPERTYPE OF (ONEOF
	(IfcMirroredProfileDef))
 SUBTYPE OF (IfcProfileDef);
	ParentProfile : IfcProfileDef;
	Operator : IfcCartesianTransformationOperator2D;
	Label : OPTIONAL IfcLabel;
 WHERE
	InvariantProfileType : SELF\IfcProfileDef.ProfileType = ParentProfile.ProfileType;
END_ENTITY;
