5.1.3.49 IfcTypeResource（资源类型）

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

5.1.3.49.1 语义定义（Semantic definition）

IfcTypeResource 定义了资源的特定（或类型）定义。它用于定义资源规范（该资源所有实例共有的特定资源），并可作为资源模板使用。

IfcTypeResource 可附加一组属性集。这些属性的值对该资源类型的所有实例均通用。类型实例关系通过对象化关系 IfcRelDefinesByType 实现。

IfcTypeResource 的子类型可在未分配给 IfcResource 子类型的情况下进行交换。

沿革 IFC4 新增实体。

5.1.3.49.5 形式化表示（Formal representation） 

ENTITY IfcTypeResource
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcConstructionResourceType))
 SUBTYPE OF (IfcTypeObject);
	Identification : OPTIONAL IfcIdentifier;
	LongDescription : OPTIONAL IfcText;
	ResourceType : OPTIONAL IfcLabel;
 INVERSE
	ResourceOf : SET [0:?] OF IfcRelAssignsToResource FOR RelatingResource;
END_ENTITY;
