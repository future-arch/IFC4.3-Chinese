6.5.3.1 IfcActionRequest（行动请求）

RV ⓘ
AbV ⓘ

6.5.3.1.1 语义定义（Semantic definition）

行动请求是指请求某事物的行为或实例，例如信息请求、投标提交或工作执行请求。

根据需求的不同，行动请求可能采取多种形式，包括维护故障报告、小型工程请求和采购请求（当这些需要通过服务台或采购功能进行时）。

沿革 IFC2x2 中的新增实体。
IFC4 变更 属性 RequestID 重命名为 Identification 并提升至超类型 IfcControl，新增属性 PredefinedType、Status 和 LongDescription。

	Status : OPTIONAL IfcLabel;
	LongDescription : OPTIONAL IfcText;
END_ENTITY;
