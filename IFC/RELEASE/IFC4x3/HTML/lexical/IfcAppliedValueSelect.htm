8.4.2.1 IfcAppliedValueSelect

8.4.2.1.1 语义定义（Semantic definition）

IfcAppliedValueSelect 定义了一个将在公式中计算的值。

类型的使用如下：

  • IfcValue：使用项目默认单位的常量值。
  • IfcMeasureWithUnit：使用指定单位的常量值。
  • IfcReference：在对象特性上引用的值。

对于成本值，适用以下指导：

  • IfcMeasureWithUnit 允许指定值本身以及值表示的货币。
  • 选择 IfcMonetaryMeasure 只允许指定值，货币由全局上下文设定。
  • 选择 IfcRatioMeasure 假定金额是百分比或其他实数。请注意，如果金额通常表示为 -20%，则需要将其转换为 0.8 的乘数。
沿革 New select type in IFC2x2.

8.4.2.1.2 类型值

8.4.2.1.3 形式化表示（Formal representation） 

TYPE IfcAppliedValueSelect = SELECT
	(IfcMeasureWithUnit
	,IfcReference
	,IfcValue);
END_TYPE;
