8.20.3.11 IfcLoop（环）

注释 该实体不是标准化模式子集或实现级别的一部分。

8.20.3.11.1 语义定义（Semantic definition）

注释 根据 ISO/CD 10303-42:1992 的定义 环是一种拓扑实体，由单个顶点构造，或通过将连接的（有向）边或线性段首尾相连于同一顶点而构成。它通常用于界定位于曲面上的面。环的维度为 0 或 1。0 维环的定义域是一个单点。1 维环的定义域是一条连通的、有向的曲线，但不一定是流形。由于环是一个循环，其起点/终点的位置是任意的。环的定义域包括其边界，即 0 ≤ Ξ < ∞。环由单个顶点、或由有向边的有序集合、或由点的有序集合表示。环是一个图，因此 M 和图亏格 G^l^ 可以通过图遍历算法确定。由于 M = 1，欧拉方程 (1) 在这种情况下简化为
注释 实体改编自 ISO 10303-42 中定义的 loop。
沿革 IFC2x 中的新实体。

非形式化命题

  1. 环具有有限范围。
  2. 环描述了一条闭合的（拓扑）曲线，其起点和终点顶点重合。
