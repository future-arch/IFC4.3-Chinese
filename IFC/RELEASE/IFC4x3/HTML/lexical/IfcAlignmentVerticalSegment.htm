8.7.3.4 IfcAlignmentVerticalSegment

AbV ⓘ

8.7.3.4.1 语义定义（Semantic definition）

IfcAlignmentVerticalSegment 是 IfcAlignmentVertical 的单个段，在距离/z 坐标空间中定义。

垂直对齐由连接端点的段定义。垂直对齐曲线的几何形状在平面中定义，其中 x = 水平距离，y = 高度（或标高）。段连接处的过渡不强制为切向，IfcSegment Transition 枚举明确指定了过渡的类型。

定义了以下垂直段类型：

  • 直线段 - IfcAlignmentVerticalSegmentTypeEnum .LINE.
  • 圆弧段 - IfcAlignmentVerticalSegmentTypeEnum .ARC.
  • 抛物线弧段 - IfcAlignmentVerticalSegmentTypeEnum .PARABOLICARC. 可以描述对称抛物线，不对称抛物线可以通过定义两个对称抛物线段来创建
  • 曲率变化线性过渡段 - IfcAlignmentVerticalSegmentTypeEnum .CLOTHOID.

对于每个垂直段，提供以下非冗余信息：

  • 起点（以距离/高度坐标表示）
  • 起始坡度（以比率表示，水平为 0，上坡为正，下坡为负）通常在 1 < n < -1 之间（等于 100% < n < -100% 的百分比，或 45° < n < -45° 的角度，但可能存在更高值）
  • 长度（作为沿水平对齐的距离长度，而不是曲线段长度）
  • 圆弧和抛物线弧段所需的曲线参数

以下信息可以计算得出（因此不明确交换以避免冗余和不一致）：

  • 结束距离（从距离和段长度计算）
  • 结束高度（从起始距离、坡度、长度和曲线参数计算）
  • 结束方向（从起始方向、段长度和曲线参数计算）
  • 竖曲线交点（从起始方向和结束方向计算）

可以进行以下检查以验证正确的交换：

  • 连续性 – 前一个段的计算结束距离是否与本段提供的起始距离匹配
  • 切向连续性 – 前一个段的计算结束坡度是否与本段提供的起始坡度匹配
注释 Specific subtypes of the IfcAlignmentVerticalSegment add specific geometric curve parameters. Connectivity between vertical segments is not necessarily tangential, but this can be enforced as a requirement through the attribute TangentialContinuity.

8.7.3.4.3 特性（Attributes）

8.7.3.4.4 形式化表示（Formal representation） 

ENTITY IfcAlignmentVerticalSegment
 SUBTYPE OF (IfcAlignmentParameterSegment);
	StartDistAlong : IfcLengthMeasure;
	HorizontalLength : IfcNonNegativeLengthMeasure;
	StartHeight : IfcLengthMeasure;
	StartGradient : IfcRatioMeasure;
	EndGradient : IfcRatioMeasure;
	RadiusOfCurvature : OPTIONAL IfcLengthMeasure;
	PredefinedType : IfcAlignmentVerticalSegmentTypeEnum;
END_ENTITY;
