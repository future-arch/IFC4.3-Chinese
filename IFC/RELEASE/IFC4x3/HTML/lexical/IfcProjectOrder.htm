6.5.3.5 IfcProjectOrder

RV ⓘ
AbV ⓘ

6.5.3.5.1 语义定义（Semantic definition）

项目订单是购买产品和/或执行工作的指令，例如用于施工或设施管理。

项目订单通常是两个组织之间的正式合同，成本和时间信息可能根据包含的计划类型和约束而固定或灵活。

沿革 新实体于 IFC2.0 中引入
IFC4 变更 属性 ID 重命名为 Identification 并提升至超类型 IfcControl。添加了属性 'LongDescription'。
