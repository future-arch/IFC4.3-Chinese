8.8.3.19 IfcFacetedBrepWithVoids

注释 该实体不是标准化模式子集或实现级别的一部分。

8.8.3.19.1 语义定义（Semantic definition）

IfcFacetedBrepWithVoids 是一个分面边界表示的特化，其中包含一个或多个内部空腔。空腔表示为封闭的壳，其定义使得壳法线指向空腔内部。

注释 Entity adapted from brep_with_voids AND faceted_brep defined in ISO 10303-42.
沿革 New entity in IFC1.0
IFC4 变更 Subtyping changed from IfcManifoldSolidBrep to IfcFacetedBrep with upward compatibility for file based exchange.

非正式命题

  1. 每个空腔壳应与外壳以及其他所有空腔壳分离。
  2. 每个空腔壳应包含在外壳内，但不得包含在任何其他空腔壳内。特别是，外壳不应包含在空腔壳集中。
  3. IfcManifoldSolidBrep 中的每个壳只能被引用一次。
  4. IfcFacetedBrep 中所有壳的所有面的所有边界环都必须是 IfcPolyLoop 类型。
