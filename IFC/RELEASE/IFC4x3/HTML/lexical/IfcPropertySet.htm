5.1.3.14 IfcPropertySet（属性集）

RV ⓘ
AbV ⓘ

5.1.3.14.1 语义定义（Semantic definition）

IfcPropertySet 是属性树中容纳属性的容器。这些属性根据其名称属性进行解释。每个独立属性都具有一个有意义的名称字符串。本规范中包含一些预定义的属性集，通过分配特定名称来指示其预设属性组。这些属性集在本规范的"属性集"章节中列出。适用于特定对象的属性集在对象规范中列出。命名约定"Pset_Xxx"适用于所有作为本规范组成部分定义的属性集，并应作为 Name 属性的值使用。

此外，可以捕获任何用户定义的属性集。未声明为IFC规范组成部分的属性集，其 Name 属性值不得包含"Pset_"前缀。

IfcPropertySet 可分配给对象实例和对象类型。分配给对象类型的 IfcPropertySet 在同一对象类型的所有实例间共享。

注释 关于如何在对象实例中覆盖分配给对象类型的属性集，请参见 IfcRelDefinesByType。

IfcPropertySetTemplate 可定义底层结构，即可附加属性集的适用对象或对象类型、可包含的独立属性等必需名称。属性集通过使用引用相应对象的关系对象与其他对象建立关联：

  • 实例对象：使用反向属性 DefinesOccurrence 的 IfcRelDefinesByProperties
  • 类型对象：通过反向属性 DefinesType 直接链接
  • 底层模板：使用反向属性 IsDefinedBy 的 IfcRelDefinesByTemplate
  • 外部引用：IfcRelAssociates 的子类型用于链接到分类系统或提供属性集进一步参考的外部库。可通过反向属性 HasAssociations 访问
沿革 IFC1.0 中新实体
IFC4 变更 所有静态定义的属性集实体不再是 IfcPropertySet 的子类型。

5.1.3.14.6 形式化表示（Formal representation） 

ENTITY IfcPropertySet
 SUBTYPE OF (IfcPropertySetDefinition);
	HasProperties : SET [1:?] OF IfcProperty;
 WHERE
	ExistsName : EXISTS(SELF\IfcRoot.Name);
	UniquePropertyNames : IfcUniquePropertyName(HasProperties);
END_ENTITY;
