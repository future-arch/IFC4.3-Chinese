8.20.3.7 IfcFace（面）

注释 该实体不是标准化模式子集或实现级别的一部分。

8.20.3.7.1 语义定义（Semantic definition）

IfcFace 是一种用于定义壳体表面（由环界定）的拓扑实体。

注释 在正确构造的边界表示模型中，面法线将指向实体外部，每条边被使用两次：一次正向，一次反向。面法线完全由界定该面的 IfcEdge 和 IfcOrientedEdge 的连续方向定义。如果面的所有边按照边方向以逆时针方式连接，则面法线将指向外侧。IfcFaceSurface 的方向或 IfcFaceSurface.SameSense 属性的值对面的方向没有影响。
注释 根据 ISO/CD 10303-42:1992 的定义 面是一个二维拓扑实体，对应于由环界定的表面片段的直观概念。如果存在，其域是 R^m^ 中的一个有向、连通、有限的二维流形。面域不得有柄，但可以有孔，每个孔由一个环界定。如果存在，面基础几何的域不包含其边界，且 0 < Ξ < ∞。
注释 实体改编自 ISO 10303-42 中定义的 face。
沿革 IFC1.0 中的新实体

面由其边界环表示，这些环被定义为面边界。面具有拓扑法线 n，环的切线为 t。对于具有已定义几何的面的边界环，叉积 n x t 指向面的内部。也就是说，如果我们认为法线 n 指向上方，则每个环在从上往下看时围绕面逆时针运行。每个环关联一个布尔标志，以表示环方向是相对于面法线定向（TRUE）还是应反转（FALSE）。

面应至少有一个边界，且环不得相交。一个环可选地区分为面的外环。如果是这样，它建立了一种将面域嵌入平面的首选方式，其中面的其他边界环位于外边界之内。因为面域是弧连通的，所以任何内环都不会包含任何其他环。无论选择哪种平面嵌入方式，这一点都成立。

非正式命题

  1. 任何边不应被该面引用超过两次。
  2. 面的不同面边界应没有公共顶点。
  3. 如果存在几何，同一面的不同环不应相交。
  4. 面应满足欧拉方程：（顶点数）-（边数）-（环数）+（环的亏格之和）= 0。

8.20.3.7.5 形式化表示（Formal representation） 

ENTITY IfcFace
 SUPERTYPE OF (ONEOF
	(IfcFaceSurface))
 SUBTYPE OF (IfcTopologicalRepresentationItem);
	Bounds : SET [1:?] OF IfcFaceBound;
 INVERSE
	HasTextureMaps : SET [0:?] OF IfcTextureMap FOR MappedTo;
 WHERE
	HasOuterBound : SIZEOF(QUERY(temp <* Bounds | 'IFC4X3_DEV_62887e3f.IFCFACEOUTERBOUND' IN TYPEOF(temp))) <= 1;
END_ENTITY;
