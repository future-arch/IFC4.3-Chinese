8.8.3.20 IfcFixedReferenceSweptAreaSolid（固定引用扫掠面积实体）

注释 该实体不是标准化模式子集或实现级别的一部分。

8.8.3.20.1 语义定义（Semantic definition）

IfcFixedReferenceSweptAreaSolid 是一种扫掠面积实体类型，是通过沿着_导线_扫掠一个面积得到的结果。扫掠面积由 IfcProfileDef 的子类型提供。轮廓通过隐式笛卡尔变换算子放置在扫掠起点处，在该点处轮廓法线与导线在该点的起始切线一致，轮廓的 x 轴与_固定引用_方向在起始切线和_固定引用_平面内的正交投影一致。扫掠操作过程中曲线的方向由_固定引用_方向控制。

SweptArea 沿着_导线_扫掠的方式是：用于定义 SweptArea 的局部坐标系的原点位于_导线_上，并且局部 x 轴位于_固定引用_在该点处导线法平面上的投影方向。生成的实体具有这样的特性：在任何点处，通过垂直于_导线_的法平面与实体表面相交的横截面是 SweptArea 的副本。生成的扫掠实体通过 Position 坐标系放置。

导线_和_参考曲面_位于对象坐标系内。扫掠操作开始于 _StartParam，它可以是基于曲线参数化的参数值，也可以是从导线起点开始沿导线测量的长度。如果未提供 StartParam，则默认从导线起点开始。扫掠操作结束于 EndParam，它可以是基于曲线参数化的参数值，也可以是从导线起点开始沿导线测量的长度。如果未提供 EndParam，则默认到导线终点结束。

注释 如果 StartParam 和 EndParam 作为参数值提供，默认情况下它们不会被归一化，它们取决于曲线的参数化。然而，在 IfcCompositeCurve 中使用 IfcReparametrisedCompositeCurveSegment 作为导线允许显式地对基础扫掠曲线进行重新参数化。
注释 对于由 IfcCurveSegment 组成的曲线，IfcSweptAreaSolid 的 Position 取决于 IfcCurveSegment 的 StartPlacement 以及定义段起点和终点（明确定义为下一个段的起点）之间_名称-函数_关系的 BasisCurve 类型。
示例 参考曲面是位于 3D 空间中并定位于对象坐标系内的任何曲面（平面、圆柱面、复合曲面）。在许多情况下，它是一个拉伸曲面。导线位于该曲面上，通常定义为该参考曲面上的 p 曲线。在导线的任何点处，都可以构造一个平面。位置坐标系的原点位于导线上。位置坐标系的 Axis3（z 轴，或法线）与该点处导线的切线相同，位置坐标系的 Axis1（x 轴，或 u）与_固定引用_方向相同。Axis2（y 轴，或 v）被构造出来。在这种情况下，生成的扫掠实体不会被重新定位。

SweptArea 沿着_导线_扫掠时的方向通过具有以下属性的 CartesianTransformationOperator3d 精确定义：

  • LocalOrigin 作为点 (0; 0; 0)，
  • Axis1 作为 FixedReference 在由 Axis3 定义的法平面上的正交投影。
  • Axis3 作为参数为 u 的 Directrix 点处的切向量 t 的方向。

其余属性默认为定义相应的变换矩阵 T(u)，该矩阵随 Directrix 参数 u 变化。

注释 实体的几何形状不依赖于曲线参数化；体积取决于扫掠的面积和_导线_的长度。
注释 实体改编自 ISO 10303-42 中定义的 fixed_reference_swept_surface。
沿革 IFC4 中的新实体。

非正式命题

  1. SweptArea 应位于平面 z = 0 内。
  2. FixedReference 不应与此曲线上任意点处的导线切向量平行。
  3. Directrix 曲线应是切线连续的。
