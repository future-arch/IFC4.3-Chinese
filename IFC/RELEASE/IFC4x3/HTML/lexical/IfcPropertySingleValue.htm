8.16.3.12 IfcPropertySingleValue

RV ⓘ
AbV ⓘ

8.16.3.12.1 语义定义（Semantic definition）

具有单个值的属性 IfcPropertySingleValue 定义了一个属性对象，该对象具有一个（数值或描述性）值。它定义了一个属性-单个值组合，其中提供了属性 Name、可选的 Description 和可选的 NominalValue（带度量类型）。此外，可以通过分配 Unit 来覆盖项目单位上下文指定的默认单位。

单位由 Unit 特性处理，表 8.16.3.12.A 显示了各种单值属性的示例：

  • 如果未提供 Unit 特性，则单位已由 IfcMeasureValue 或 IfcDerivedMeasureValue 的类型隐含。关联单位可在项目级别全局定义的 IfcUnitAssignment (IfcProject.UnitsInContext) 中找到。
  • 如果提供了 Unit 特性，则 Unit 特性分配的单位将覆盖全局分配的单位。
沿革 IFC1.0 中的新实体。

IFC2x 变更：实体已从 IfcSimpleProperty 重命名。

IFC2x3 变更：NominalValue 特性已设为可选，并保持文件交换的向上兼容性。

8.16.3.12.3 特性（Attributes）

8.16.3.12.4 形式化表示（Formal representation） 

ENTITY IfcPropertySingleValue
 SUBTYPE OF (IfcSimpleProperty);
	NominalValue : OPTIONAL IfcValue;
	Unit : OPTIONAL IfcUnit;
END_ENTITY;
