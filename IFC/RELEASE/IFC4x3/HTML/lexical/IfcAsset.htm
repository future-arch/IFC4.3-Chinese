6.4.3.1 IfcAsset（资产）

RV ⓘ
AbV ⓘ

6.4.3.1.1 语义定义（Semantic definition）

资产是指作为单一实体运作、具有财务价值或可作为单一单元进行操作的一组唯一可识别元素的集合。

资产通常是执行维护操作的粒度级别。资产是一个可以包含一个或多个元素的组。虽然可以定义组件或元素的财务价值，但出于会计目的，财务价值也在资产级别进行定义。

有多种参与者可以与资产关联，每个参与者都有相应的角色。项目范围内的参与者通过 IfcRelAssignsToActor 关系进行标识，这种情况下角色应通过 IfcActorRole 类定义；否则主要参与者将作为类的属性进行标识。如果两者同时存在，直接属性优先。

有多种成本可以与资产关联，每项成本都有相应的角色。这些通过 OriginalValue、CurrentValue、TotalReplacementCost 和 DepreciatedValue 属性进行指定。

沿革 IFC2x 中新增实体。
IFC4 变更 所有属性改为可选，日期值改为使用 IfcDate。

6.4.3.1.5 概念用法（Concept usage）

6.4.3.1.6 形式化表示（Formal representation） 

ENTITY IfcAsset
 SUBTYPE OF (IfcGroup);
	Identification : OPTIONAL IfcIdentifier;
	OriginalValue : OPTIONAL IfcCostValue;
	CurrentValue : OPTIONAL IfcCostValue;
	TotalReplacementCost : OPTIONAL IfcCostValue;
	Owner : OPTIONAL IfcActorSelect;
	User : OPTIONAL IfcActorSelect;
	ResponsiblePerson : OPTIONAL IfcPerson;
	IncorporationDate : OPTIONAL IfcDate;
	DepreciatedValue : OPTIONAL IfcCostValue;
END_ENTITY;
