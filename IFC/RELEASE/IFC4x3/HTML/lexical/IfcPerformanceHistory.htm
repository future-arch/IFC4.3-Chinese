5.2.3.1 IfcPerformanceHistory（性能历史）

注释 该实体不是标准化模式子集或实现级别的一部分。

5.2.3.1.1 语义定义（Semantic definition）

IfcPerformanceHistory 用于记录实例对象随时间的实际性能表现。它包括来自楼宇自动化系统的机器测量数据以及人工指定的数据（如任务和资源使用情况）。这些数据可以代表实际状况、预测或模拟。

性能历史跟踪的实时数据采用属性集的形式，其中所有属性都基于时间序列。与实例和类型中基于设计的数据不同，性能驱动数据具有时间敏感性，可能通过某些测量设备实时变化。数据可以在不规则间隔捕获（例如当值变化超出既定阈值时），或在指定持续时间的规则间隔捕获。

5.2.3.1.1.1 Declaration use definition

IfcPerformanceHistory 可在项目中使用 IfcRelDeclares 进行声明，其中 RelatingContext 引用 IfcProject，RelatedDefinitions 包含 IfcPerformanceHistory。默认单位（用于属性集）由声明项目指示。仅声明顶级对象；嵌套的性能历史对象（通过 IfcRelNests）不参与此类关系。

沿革 IFC2x2 中的新实体。
