8.10.3.1 IfcMaterial

RV ⓘ
AbV ⓘ

8.10.3.1.1 语义定义（Semantic definition）

IfcMaterial 是可用于构成元素（物理产品或其组件）的均质或非均质物质。

IfcMaterial 是材料命名和定义的基体实体；这包括通过名称和分类（通过引用外部分类）进行标识，以及通过（IfcMaterialProperties 的子类型）定义的材料属性（各向同性或各向异性）的关联。IfcMaterial 的实例可以通过 IfcRelAssociatesMaterial 关系关联到元素或元素类型。分配可以是直接的单一材料信息，或通过

  • 材料层集
  • 材料型材集
  • 材料构成集

IfcMaterial 也可以关联表示信息。此类表示信息由 IfcMaterialDefinitionRepresentation 提供，将曲线样式、填充定义或表面着色/渲染信息关联到材料。

IFC4 变更 The attributes Description and Category have been added.

8.10.3.1.5 概念用法（Concept usage）

8.10.3.1.6 形式化表示（Formal representation） 

ENTITY IfcMaterial
 SUBTYPE OF (IfcMaterialDefinition);
	Name : IfcLabel;
	Description : OPTIONAL IfcText;
	Category : OPTIONAL IfcLabel;
 INVERSE
	HasRepresentation : SET [0:1] OF IfcMaterialDefinitionRepresentation FOR RepresentedMaterial;
	IsRelatedWith : SET [0:?] OF IfcMaterialRelationship FOR RelatedMaterials;
	RelatesTo : SET [0:1] OF IfcMaterialRelationship FOR RelatingMaterial;
END_ENTITY;
