8.18.3.12 IfcRepresentation

RV ⓘ
AbV ⓘ

抽象 该定义可能不会被直接实例化。

8.18.3.12.1 语义定义（Semantic definition）

IfcRepresentation 定义了表示产品属性（特别是产品形状）的通用概念。

注释 来自 ISO 10303-43 的定义： 表示是由一个或多个表示项组成的集合，这些表示项在指定的表示上下文中相关联。表示项与表示上下文的关系是区分哪些表示项实体相关的基础。表示项可以直接作为表示中的一个元素出现，或者通过任意数量的中间实体（每个实体都为表示项类型）间接引用，从而与表示上下文相关联。
注释 实体改编自 ISO 10303-42 中定义的 representation。
沿革 新实体在 IFC2.0 中引入。
IFC2x3 变更 添加了逆向属性 LayerAssignments 和 RepresentationMap，并保持了向上兼容性。
IFC4 变更 实体 IfcRepresentation 已更改为抽象超类型。

表示使用定义

每个表示，无论是 IfcShapeRepresentation 还是 IfcTopologyRepresentation，都必须具有明确定义的：

  • ContextOfItems：指向为该表示约定的 IfcGeometricRepresentationContext 的引用。
  • RepresentationIdentifier：表示的名称，例如，3D 形状的“Body”，2D 地面视图的“FootPrint”，参考轴的“Axis”。
  • RepresentationType：几何或拓扑表示类型的名称，例如，扫描实体（SweptSolid）的“SweptSolid”，边界表示（Brep）的“Brep”。
注释 在 IfcElement 的每个子类型的几何使用定义部分提供了有关将正确值应用于这些属性的指南。这些指南可以在视图定义或实施协议中进一步细化。

8.18.3.12.3 特性（Attributes）

8.18.3.12.4 形式化表示（Formal representation） 

ENTITY IfcRepresentation
 ABSTRACT SUPERTYPE OF (ONEOF
	(IfcShapeModel
	,IfcStyleModel));
	ContextOfItems : IfcRepresentationContext;
	RepresentationIdentifier : OPTIONAL IfcLabel;
	RepresentationType : OPTIONAL IfcLabel;
	Items : SET [1:?] OF IfcRepresentationItem;
 INVERSE
	RepresentationMap : SET [0:1] OF IfcRepresentationMap FOR MappedRepresentation;
	LayerAssignments : SET [0:?] OF IfcPresentationLayerAssignment FOR AssignedItems;
	OfProductRepresentation : SET [0:?] OF IfcProductRepresentation FOR Representations;
END_ENTITY;
